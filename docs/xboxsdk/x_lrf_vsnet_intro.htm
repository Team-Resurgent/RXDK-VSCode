Using Visual Studio .NET

The Xbox Development Kit for Microsoft® Visual Studio® .NET provides many features that make development easier and faster for the Xbox® video game system from Microsoft. Visual Studio .NET includes the Microsoft® Visual C++® 7.0 compiler. Visual Studio .NET 2003 includes the Visual C++ 7.1 compiler. If you are upgrading from Visual C++ 6.0, there are a few changes that could potentially cause problems in existing game code. For more information, see Moving to the Visual C++ 7.1 Compiler.

Note    The Xbox support for Visual Studio .NET and Visual Studio .NET 2003 does not currently support the VSIP (Visual Studio Integration Program) version of Visual Studio .NET. The VSIP version of Visual Studio .NET is intended for developers interested in extending the Visual Studio .NET integrated development environment.

This section and the topics listed below provide guidelines for using Visual Studio .NET to create Xbox games. In addition, this section summarizes the differences between developing under Visual C++ 6.0 and under the new Visual Studio .NET or Visual Studio .NET 2003 environments, and highlights the new features in Visual Studio .NET and Visual Studio .NET 2003 that make developing a game easier. This section also discusses converting Visual C++ 6.0 projects into Visual Studio .NET or Visual Studio .NET 2003 projects.
