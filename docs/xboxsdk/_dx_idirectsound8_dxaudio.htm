IDirectSound8

Used to create buffer objects and set up the environment.

The IDirectSound8 interface supersedes IDirectSound and adds new methods. See Remarks.

Obtain this interface by using the DirectSoundCreate function.

The methods of the IDirectSound8 interface can be organized into the following groups:

The IDirectSound8 interface also supports the AddRef and Release methods. The QueryInterface method is not supported on the Xbox game system.

The LPDIRECTSOUND8 type is defined as a pointer to the IDirectSound8 interface:

typedef struct IDirectSound8    *LPDIRECTSOUND8;

Remarks

  • In CreateSoundBuffer, DSBCAPS_CTRL3D cannot be set on a buffer with a non-mono format, and DSBCAPS_CTRL3D cannot be combined with DSBCAPS_CTRLPAN.
  • New buffer creation flags are supported.
  • DirectSound does not initialize the contents of the buffer, so the application cannot assume that it contains silence.
