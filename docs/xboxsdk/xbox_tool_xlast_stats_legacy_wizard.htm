The New Stats Configuration Wizard for Legacy Versions

To create a new stats configuration submission for versions of Stats before version 3, click New on the File menu. Click Legacy Stats Configuration and then click OK. This starts the New Stats Configuration Wizard for legacy Stats versions.

After displaying the welcome dialog box, the New Stats Configuration Wizard presents a dialog box asking for the following information.

Title ID
Contains the 8-digit hexadecimal title identifier that you receive from Microsoft.
Project Directory
Specifies the path to the directory containing the XML project file (with an extension of .xlast) and a build file (which has an extension of .xsc). Both of these use the Title ID as their file names.
Number of Leaderboards
Specifies the number of leaderboards to display on Xbox Live for this stats configuration.
Reset Type
Indicates how often the leaderboards should be reset to zero. This can be set to Never, Weekly, Monthly, Bi-Monthly (every other month), Quarterly, Semiannually (every 6 months), and Annually.
Maximum Number of Attachments
Specifies the maximum number of attachments any given leaderboard may have.
Maximum Attachments Size
Contains the maximum size, in bytes, of any single attachment.
Decay
Specifies the decay rate, in days, of the statistics in the leaderboard. This information is used in conjunction with the Reset Type to determine when statistical information is cleared from the leaderboard.
Top 100
Enables or disables the display of Top 100 rankings.
Stats Available on the Web
Displays the statistical information on the Web if checked. Otherwise, the information is displayed only on Xbox Live.

The New Stats Configuration Wizard next displays a dialog box stating that you have finished inputting data for your project. Click Finish to create the XML project file.
