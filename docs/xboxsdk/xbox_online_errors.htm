Xbox Live Error Codes

This section provides general information on the error codes returned by the Xbox Live Functions. For error code information specific to a particular function, see the reference page for that function.

  • General Errors
  • Unexpected Errors

General Errors

The following errors can be returned from any of the Xbox Live Functions.

Value Description
XONLINE_E_OUT_OF_MEMORY Insufficient memory. Free some memory.
E_OUTOFMEMORY Insufficient memory. Free some memory.
XONLINE_E_NO_SESSION This function was called without first calling logon. Call logon before calling any other functions.
XONLINE_E_NO_USER The player is not logged on. Log the player on before calling functions that specify that player.
XONLINE_E_USER_NOT_LOGGED_ON The player is not logged on. Log the player on before calling functions that specify that player.
XONLINE_E_NO_GUEST_ACCESS The function called does not allow guests.
XONLINE_E_NOT_INITIALIZED XOnlineStartup has not been called yet. Call XOnlineStartup first.

Unexpected Errors

These errors should never occur. If a title encounters one of these errors in a game session, it should display "Try again later" or a fail message. If you encounter this during development, contact your TGM or developer support to ensure that our operations center is configured correctly for your title.

Matchmaking

Value Description
XONLINE_E_MATCH_INVALID_DATA_TYPE An attribute ID or parameter type specifies an invalid data type.
XONLINE_E_MATCH_REQUEST_TOO_SMALL The request did not meet the minimum length for a valid request.
XONLINE_E_MATCH_REQUEST_TRUNCATED The self-described length is greater than the buffer size.
XONLINE_E_MATCH_INVALID_VERSION The request contained a bad protocol version.
XONLINE_E_MATCH_OVERFLOW An attribute or parameter flowed past the end of the request.
XONLINE_E_MATCH_INVALID_RESULT_COL The referenced stored procedure returned a column with an unsupported data type.
XONLINE_E_MATCH_INVALID_STRING A string was passed with a length-prefix of zero or without a terminating null character.
XONLINE_E_MATCH_SESSION_ALREADY_EXISTS The session ID already exists in the database.
XONLINE_E_MATCH_CRITICAL_DB_ERR A critical error occurred in the database.
XONLINE_E_MATCH_NOT_ENOUGH_COLUMNS The search result set had too few columns.
XONLINE_E_MATCH_PERMISSION_DENIED The stored procedure has incorrect permissions.
XONLINE_E_MATCH_INVALID_PART_SCHEME The title specified an invalid partitioning scheme.
XONLINE_E_MATCH_DATA_TYPE_MISMATCH The data type specified in an attribute ID did not match the type of attribute being set.
XONLINE_E_MATCH_NO_USERS The search did not contain any authenticated users.
XONLINE_E_MATCH_INVALID_BLOB A binary block of data (BLOB) attribute was invalid.

Statistics

Value Description
XONLINE_E_STAT_NOT_ENOUGH_PARAMETERS Error for features that are currently not implemented. If you get this error, please contact your TGM or developer support to report the problem.
XONLINE_E_STAT_INVALID_PROCEDURE Error for features that are currently not implemented. If you get this error, please contact your TGM or developer support to report the problem.
XONLINE_E_STAT_EXCEEDED_WRITE_READ_LIMIT Encountered when your game exceeds the read-write limit while using the XOnlineStatWrite, XOnlineStatWriteEx, and XOnlineStatRead functions. By default, every title on Xbox Live is configured to have a max of five writes and five reads per player per request. A read request is defined as anything being read in a leaderboard for a player. The number of stats does not matter. In the same way, a write means anything being written to a leaderboard for a player. Again, the number of stats does not matter.
XONLINE_E_STAT_LEADERBOARD_READONLY The leaderboard is in a read-only state. This occurs only when the server is undergoing maintenance. While in this state, statistics cannot be written to the leaderboard.
