IDirect3DFixup8::GetSize

Returns the size, in bytes, of the current fix-up as created by the last IDirect3DPushBuffer8::BeginFixup/IDirect3DPushBuffer8::EndFixup bracket.

HRESULT GetSize(
  DWORD *pSize
);

Parameters

pSize
[out, retval] Pointer to a DWORD value containing the size in bytes of the current fix-up as crreated by the last IDirect3DPushBuffer8::BeginFixup/IDirect3DPushBuffer8::EndFixup bracket. This value will be the correct amount needed even if IDirect3DPushBuffer8::EndFixup failed because of lack of space. This enables a zero-size fix-up object to be used to efficiently query the amount of space required by a set of fix-ups.

Return Values

If the method succeeds, the return value is S_OK.
