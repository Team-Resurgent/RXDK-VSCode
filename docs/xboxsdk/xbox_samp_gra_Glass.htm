Glass Sample

The Glass sample shows how to render 3D objects so they look like glass that both refracts and reflects light.

Overview

Reflection is nothing new, so this sample is really showing off an approximation of refraction. Note that this is a gross approximation. Real refraction would require knowledge of each light ray's entry point and exit point into and out of the model. Here, we approximate refraction by using the surface normal to lookup a perturbed normal in a pre-computed cubemap, and using that perturbed normal to look up a value in our environment map.
