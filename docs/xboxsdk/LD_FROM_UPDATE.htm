LD_FROM_UPDATE

Returned by the XGetLaunchInfo function when retrieving the launch information for a title launched by the Xbox Updater.

typedef struct{
    DWORD dwContext;
    HRESULT hr;
    DWORD dwFlags;
    DWORD dwReserved[4];
    BYTE Data[MAX_LAUNCH_DATA_SIZE - 28];
} LD_FROM_UPDATE, *PLD_FROM_UPDATE;

Members

dwContext
Title-specific context value that was passed as the dwContext parameter to XOnlineTitleUpdate, or as the dwContext member of the LD_UPDATE structure passed to XOnlineTitleUpdateEx.
hr
The result code of the update operation.
dwFlags
Flags indicating information about the returned structure. Can be a combination of the following values:
Value Description
LD_UPDATE_FLAG_DATA_PRESENT The Data member contains valid launch data.
dwReserved
Reserved; do not use.
Data
Title-specific launch data specified in the Data member of the LD_UPDATE structure passed to XOnlineTitleUpdateEx. This member contains valid data only if the LD_UPDATE_FLAG_DATA_PRESENT flag is present in dwFlags.

Remarks

If the update was performed using the XOnlineTitleUpdateEx function, then dwContext will contain the context value specified in the dwContext member of LD_UPDATE, hr will contain the result code of the update operation, dwFlags will contain LD_UPDATE_FLAG_DATA_PRESENT, and the Data member will contain the launch data passed in the Data member of the LD_UPDATE structure.

If the update was performed using the XOnlineTitleUpdate function, then dwContext will contain the context value specified in the dwContext parameter and hr will contain the result code of the update operation, but dwFlags will not contain LD_UPDATE_FLAG_DATA_PRESENT, and the Data member will not contain valid data.

Requirements
