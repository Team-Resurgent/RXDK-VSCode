D3DPOOL

Defines the memory class that holds a resource's buffers.

typedef DWORD D3DPOOL;

#define D3DPOOL_DEFAULT             0
#define D3DPOOL_MANAGED             0
#define D3DPOOL_SYSTEMMEM           0

Remarks

All parameters typed as D3DPOOL will be ignored.

Xbox Optimizations

This enumeration has been redefined to a DWORD so that zero value may be used. The DirectX 8.0 implementation was defined as follows:

typedef enum _D3DPOOL {
    D3DPOOL_DEFAULT                 = 0,
    D3DPOOL_MANAGED                 = 1,
    D3DPOOL_SYSTEMMEM               = 2,

    D3DPOOL_FORCE_DWORD             = 0x7fffffff
} D3DPOOL;
