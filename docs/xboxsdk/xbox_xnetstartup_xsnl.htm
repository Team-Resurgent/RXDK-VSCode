XNetStartup

Initializes the Xbox Secure Network Library (SNL).

INT XNetStartup(
    const XNetStartupParams *pxnsp
);

Parameters

pxnsp
Address of a XNetStartupParams structure that contains all of the initial settings. Should be set to NULL if the title is using the default settings.

Return Values

Returns zero if successful, an error code otherwise.

Remarks

XNetStartup must be called before any network calls are made, with the exception of XNetGetEthernetLinkStatus. After the network session is complete, the title must call the XNetCleanup function.

By default, the Xbox SNL operates in secure mode, which does not allow communication with untrusted hosts, such as personal computers. However, the developer kit version of the Xbox SNL (Xnet.lib and Xnetd.lib) allows you to bypass this security by specifying the XNET_STARTUP_BYPASS_SECURITY flag in the cfgFlags member of the XNetStartupParams structure. The following example demonstrates an easy way of doing this:

XNetStartupParams xnsp;
memset(&xnsp, 0, sizeof(xnsp));
xnsp.cfgSizeOfStruct = sizeof(XNetStartupParams);
xnsp.cfgFlags = XNET_STARTUP_BYPASS_SECURITY;
INT err = XNetStartup(&xnsp);

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
