XONLINE_REQUEST_ANSWER_TYPE

Defines the values used to specify responses to a request.

typedef enum {
    XONLINE_REQUEST_NO = 0,
    XONLINE_REQUEST_YES = 1,
    XONLINE_REQUEST_BLOCK = 2,
} XONLINE_REQUEST_ANSWER_TYPE;

Constants

XONLINE_REQUEST_NO
Declines the request.
XONLINE_REQUEST_YES
Accepts the request.
XONLINE_REQUEST_BLOCK

Blocks the player that created the request. Adds the requesting player to the current player's Never list. The current player no longer receives friend requests from the requesting player.

If the current player makes a subsequent friend request of the blocked player, the blocked player becomes unblocked—that is, is removed from the Never list. The blocked player is removed from the Never list regardless of their response to the current player's friend request.

Blocking a player from making friend requests does not automatically add that player to the mutelist.
