IDirect3DDevice8::EndState

EndState must be called after IDirect3DDevice8::BeginState, once the data is completely written to the push-buffer.

HRESULT EndState(
    DWORD *pPush
);

Parameters

pPush
[in] pPush should point to the first DWORD following the data written to the push-buffer.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

For an example of how to use EndState, see the remarks section of BeginState.

After calling EndState, the data written to the push-buffer may not be modified.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
