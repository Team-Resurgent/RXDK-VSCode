RenderState (XbView)

The RenderState structure stores a Direct3D render state constant and the value assigned to that state.

struct RenderState 
{
	DWORD State;
	DWORD Value;
};

These are set in BaseFrameNode::Draw just before the Direct3D draw calls. Render states do not persist across passes. Each pass resets the render states to the defaults for XbView by calling XBR::SetDefaultState. Any render states that are not set explicitly here are set to the normal Direct3D defaults.
