XFONT::GetTextStyle

Gets the text style.

unsigned __stdcall GetTextStyle();

Parameters

None.

Return Values

On success, the function returns one of the following values:

Value Meaning
XFONT_NORMAL Text will appear without bold or italics.
XFONT_BOLD Text will appear bold.
XFONT_ITALICS Text will appear italicized.
XFONT_BOLDITALICS Text will appear both bold and italicized.

Requirements
