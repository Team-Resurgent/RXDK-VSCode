z-buffer
A buffer that stores a depth value for each pixel in a scene. Pixels with a small z-value overwrite pixels with a large z-value.

zero-latency streaming
A method for minimizing both memory footprint and streaming latency for wave playback, whereby the beginning of a wave is read into memory ahead of time, so the programmer can begin playback of the wave instantaneously, while the rest of the wave is streamed in from disk as its data is needed. Also known as "primed" streaming. See also streaming.
