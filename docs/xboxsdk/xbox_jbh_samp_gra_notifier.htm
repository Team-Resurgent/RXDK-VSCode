Notifier Sample

The Notifier sample shows how to use the following APIs: SetVerticalBlankCallback, BlockUntilVerticalBlank, InsertFence, BlockOnFence, and InsertCallback.

Overview

The Fence and InsertCallback APIs can be used to determine if a resource is in use by the GPU. They can also be used to determine when geometry has been completely processed by the pipeline.
