ELO Sample

The ELO sample demonstrates the ELO ranking system on Xbox Live.

Overview

The ELO service is a way for a title to rate and rank players in a meaningful way. ELO scoring is more accurate at reflecting true skill than other rating systems. Since the ELO service is an extension of the arbitration service, cheating exploits are avoided.

Path

Source: (XDK Root)\Samples\Xbox\XboxLive3\ELO

Remarks

This sample uses a leaderboard posted to PartnerNet. A new leaderboard must be created for your title ID. Leaderboards are created and submitted to PartnerNet using the Xbox Live Authoring and Submission Tool (XLAST). If you have questions about the leaderboard creation process, please contact your Xbox account manager.

Programming Notes

The code for this sample is found in three files source files.

File Name Description
match.cpp Used to find and join a match for which a host already exists. Handles network messaging.
GameMsg.cpp Handles network blobs (aka messages) and hooks into callbacks provided by the Elo sample class.
EloDemo.cpp Contains the main logic to demonstrate ELO rankings.

Running the Sample

To run the ELO sample, you must have at least two Xbox consoles and one PC with the Matchmaking Simulation tool (MatchSim) installed.

Use the following steps to run the sample.

  1. Copy the ELO Sample XBE to each Xbox.
  2. Using the Xbox Neighborhood tool, set one Xbox as the default.
  3. Open the MatchSim file provided with the sample. Its name is MatchMaking.xms. In MatchSim, select Tools from the main menu, and then click Connect Xbox. This action routes the Xbox QuickSearch through the PC.
  4. Repeat step 3 for the other Xbox console(s).
  5. From the main menu of MatchSim, select Match and then click Start Server to start the MatchMaking simulation.
  6. Start the Elo sample on both Xbox consoles and log into the Xbox Live service using different accounts.
  7. Create a match using one Xbox and join the match with the other (using QuickMatch)
