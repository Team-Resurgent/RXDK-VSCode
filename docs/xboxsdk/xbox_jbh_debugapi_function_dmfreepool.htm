DmFreePool

Frees memory that was allocated using DmAllocatePool or DmAllocatePoolWithTag.

DMAPI VOID __stdcall DmFreePool(
  PVOID p
  );

Parameters

p
Pointer to the memory that is to be released.

Return Values

This function does not return a value.

Requirements

Header: Declared in XbDm.h
