DmUseSharedConnection

Enables you to share a single (socket) connection for all API commands, or to have the debugging subsystem open and close a new socket for each API call.

DMHRAPI DmUseSharedConnection(
  BOOL fShare
  );

Parameters

fShare
If TRUE, debugger API calls will share a single connection. If FALSE, each API call will open a connection, send a command, and close the connection.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

Normally, each API call causes a connection to be opened, the command to be sent, and the connection to be closed. If you pass TRUE to DmUseSharedConnection, the connection is opened only once and will be shared by all subsequent API calls. If you pass FALSE, then any existing idle connection will be closed.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
