FileStream Sample

The FileStream sample demonstrates how to use the wave file media object to stream an ADPCM .wav file from disk and loop it. Since the audio processor can decompress ADPCM in hardware, all you need to do to play an ADPCM .wav file is pass the packets on to DirectSoundStream.

This sample has changed substantially from earlier releases; AdpcmCreateDecoder no longer exists because there's no need for it. ADPCM data can be passed straight to the stream.

Since the wave file media object operates synchronously, the sample runs the media object on a separate worker thread to avoid blocking the main thread. By processing on a separate thread, only that thread is blocked during I/O. Alternatively, the application could have its own loading mechanism instead of using the wave file media object, and pass the ADPCM data straight to DirectSoundStream.
