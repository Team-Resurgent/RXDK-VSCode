XOnlineTeamEnumerate

Retrieves team details for a list of known team XUID structures.

HRESULT XOnlineTeamEnumerate(
    DWORD dwUserIndex,
    DWORD dwTeamCount,
    const XUID *pxuidTeams,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player requesting the enumeration.
dwTeamCount
[in] Number of teams to enumerate. This value is should be less than or equal to XONLINE_MAX_TEAM_COUNT.
pxuidTeams
[in] Pointer to an array of XUID structures containing the team information.
hWorkEvent
[in, optional] Handle to an event.
phTask
[out] Pointer to an XONLINETASK_HANDLE handle returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

When the task is complete, the title can call the XOnlineTeamGetDetails function to access the retrieved team information for individual teams.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
