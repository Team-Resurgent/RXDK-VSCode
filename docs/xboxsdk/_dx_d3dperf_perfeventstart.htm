D3DPERF_PerfEventStart

Inserts a user performance event.

D3DPERFEvent * WINAPI D3DPERF_PerfEventStart(
    WORD Type,
    BOOL RecordGpuTime
);

Parameters

Type
[in] Member of the D3DPERFEventsType enumerated type, describing which user event is being inserted. Type must be between D3DPERFEvent0 and D3DPERFEvent10.
RecordGpuTime
[in] TRUE to insert a fence for counting GPU times.

Return Values

Returns a pointer to the D3DPERFEvent structure that will record information about the event that was inserted. The function return NULL if

  • The CountD3DPerfEvents member of D3DPERF is zero. This member is set by calling D3DPERF_StartCountingPerfEvent.
  • The event type specified by Type does not match up to one that is being recorded as set in the RecordD3DPerfEvent member of D3DPERF.

Remarks

Titles should call D3DPERF_StartCountingPerfEvent before calling D3DPERF_PerfEventStart.

To stop the recording of a given performance event, titles should call D3DPERF_PerfEventEnd.

Requirements

Header: Declared in D3d8perf.h.

Import Library: Use D3d8i.lib or d3d8d.lib.
