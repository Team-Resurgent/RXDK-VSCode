Xbox Live Structures

The following Xbox Live structures are used:

Structure Description
LD_DOWNLOADER Passed to Downloader.xbe as the pLaunchData parameter of XLaunchNewImage. When Downloader.xbe reboots to the title, it passes this structure back to the title as the pLaunchData parameter of XLaunchNewImage.
LD_UPDATE Used in the XOnlineTitleUpdateEx function to pass launch data to a title across an update.
XONLINE_ACCEPTED_GAMEINVITE Contains details about an accepted game invitation.
XONLINE_ARB_ID Contains information required for arbitrating the results of a round.
XONLINE_ARB_REGISTRANT Contains data about a registrant in an arbitrated session.
XONLINE_ARB_REPORT_DATA Contains data about lost connections or suspicious activity during an arbitrated session.
XONLINE_ARB_SUSPICIOUS_INFO Contains data about suspicious activity during an arbitrated session.
XONLINE_ATTRIBUTE Contains a single attribute for a matchmaking search or session.
XONLINE_ATTRIBUTE_SPEC Contains the layout of an attribute.
XONLINE_COMP_CREATE_RESULTS Contains results information about a competition creation.
XONLINE_COMP_INTERVAL_UNIT Specifies the units for the intervals between competition events.
XONLINE_COMP_SINGLE_ELIMINATION_ATTRIBUTES Contains information describing a single-elimination competition.
XONLINE_COMP_TOPOLOGY_SE_RESULTS Used to hold the results of a topology search of a single-elimination competition.
XONLINE_COMP_UNITS_OR_MASK Contains information about competition units.
XONLINE_FEEDBACK_PARAMS Provides additional information about feedback sent using the XOnlineFeedbackSend function.
XONLINE_FRIEND Stores information about a player on the friends list.
XONLINE_FRIENDS_RESULTS_ERROR Contains information about a player list change.
XONLINE_GAME_JOIN_INFO Contains the information required for a player to join a game.
XONLINE_GAMEINVITE_ANSWER_INFO Contains game invitation information.
XONLINE_LATEST_ACCEPTED_GAMEINVITE Contains information about the latest accepted game invitation.
XONLINE_LOGON_STATE Contains a logon state returned by a call to XOnlineSaveLogonState.
XONLINE_MATCH_SEARCHRESULT Contains the base session information returned to the client after a successful search.
XONLINE_MSG_SEND_RESULT Contains the results of a message transmission.
XONLINE_MSG_SUMMARY Contains message summary information.
XONLINE_MUTELISTUSER Contains information about a player that has been added to a mutelist.
XONLINE_NOTIFICATION_EX_INFO Contains additional notification information.
XONLINE_NOTIFICATION_MSG Contains the notification message information.
XONLINE_PRESENCE Contains presence information about the specified player.
XONLINE_PRICE Contains the price of an online offering.
XONLINE_REMOTE_FILE_REFERENCE Storage service.
XONLINE_SERVICE_INFO Contains information about a requested online service.
XONLINE_SIGNATURE_TO_VERIFY Contains a signature/digest pair to be verified with XOnlineSignatureVerify.
XONLINE_STARTUP_PARAMS Contains the startup parameters for Xbox Live.
XONLINE_STAT Represents a statistic that can be stored or retrieved using the statistics service.
XONLINE_STAT_ATTACHMENT_REFERENCE Contains information about a stats attachment.
XONLINE_STAT_CONDITIONAL Contains information about a stats comparison.
XONLINE_STAT_CONDITIONAL_UNIT Units are no longer supported by Xbox Live
XONLINE_STAT_CUSTOM Contains the parameters for a title-specific statistics update procedure.
XONLINE_STAT_CUSTOM_PARAM Contains parameter information for a title-specific statistics update procedure.
XONLINE_STAT_ELO Contains information for calculating the ELO rating for a pair of competitors.
XONLINE_STAT_ELO_UNIT Not supported.
XONLINE_STAT_NAME Specifies the name of a team and player.
XONLINE_STAT_PROC Contains statistical information about a player, Xbox Live team.
XONLINE_STAT_SPEC Specifies a player and a set of statistics associated with that player on a particular leaderboard.
XONLINE_STAT_SPEC_UNIT Units are no longer supported by Xbox Live.
XONLINE_STAT_UNIT Not supported.
XONLINE_STAT_UPDATE Contains statistical information about a player or Xbox Live team in a competition.
XONLINE_STAT_UPDATE_UNIT Not supported.
XONLINE_STAT_USER Specifies a particular player of the statistics service.
XONLINE_TEAM Contains information about a team.
XONLINE_TEAM_MEMBER Contains information about a team member.
XONLINE_TEAM_MEMBER_PROPERTIES Contains property information about the specified team member.
XONLINE_TEAM_PROPERTIES Contains property information about a team.
XONLINE_USER Contains information about an online player.
XONLINEOFFERING_DETAILS Contains details about an online offering.
XONLINEOFFERING_ENUM_PARAMS Specifies parameters and options for an offering enumeration initiated with XOnlineOfferingEnumerate.
XONLINEOFFERING_INFO Contains details about an online offering.
XONLINESTORAGE_FILE_INFO Contains information about downloadable content.
XUID Contains the user identifier and user-specific settings.
