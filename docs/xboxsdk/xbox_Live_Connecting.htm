Connecting to Xbox Live with Authentication

The entry point into the Xbox Live system is through the authentication system. All titles must log users into the service before providing any Xbox Live features to the player.

Fortunately, the responsibility for title developers is quite small. The Xbox Live libraries implement the true authentication work for the title and expose it almost entirely through the XOnlineLogon function. The Authentication sample in the Xbox SDK demonstrates many of the requirements for this authentication.
