IDirect3DResource8::GetDevice

Retrieves the device associated with a resource.

HRESULT GetDevice(
  IDirect3DDevice8** ppDevice
);

Parameters

ppDevice
[out, retval] Address of a pointer to an IDirect3DDevice8 interface to fill with the device pointer.

Return Values

The return value is always D3D_OK.

Applies To

This method applies to the following interfaces, which inherit from IDirect3DResource8.

  • IDirect3DBaseTexture8
  • IDirect3DCubeTexture8
  • IDirect3DIndexBuffer8
  • IDirect3DTexture8
  • IDirect3DVertexBuffer8
  • IDirect3DVolumeTexture8
  • IDirect3DVolume8
  • IDirect3DSurface8
  • IDirect3DPalette8

Remarks

This method allows navigation to the owning device object.

Note  Calling this method will increase the internal reference count on the IDirect3DDevice8 interface. Failure to call IDirect3DResource8::Release when finished using this IDirect3DDevice8 interface results in a memory leak.

Requirements
