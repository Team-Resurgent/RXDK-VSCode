D3D__NullHardware

Enables infinitely fast hardware.

BOOL D3D__NullHardware;

Remarks

Set D3D__NullHardware to TRUE to enable infinitely fast hardware. While this is set to TRUE, no operations on the GPU will actually take place and there will be no rendering. If you reset D3D_NullHardware back to FALSE, it is more than likely that the program will crash. This is useful for determining how CPU-bound your program is.
