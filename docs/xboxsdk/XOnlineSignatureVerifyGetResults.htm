XOnlineSignatureVerifyGetResults

Retrieves the results of signature verification initiated with a call to XOnlineSignatureVerify.

HRESULT XOnlineSignatureVerifyGetResults(
    XONLINETASK_HANDLE hTask,
    HRESULT **prgHresults,
    DWORD *pdwHresults
);

Parameters

hTask
[in] Online task handle returned by the call to XOnlineSignatureVerify that began the verification.
prgHresults
[out] Pointer to a variable that receives an array of HRESULT values indicating the success or failure of each digest/signature pair.
pdwHresults
[out] Pointer to a DWORD variable that receives the count of results returned in prgHresults.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

For each digest/signature pair passed to XOnlineSignatureVerify, a corresponding HRESULT will be returned in the prgHresults array. An HRESULT of S_OK indicates that the digest/signature pair was successfully verified. Any other result indicates that the verification failed and the data should be treated as invalid.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonlines.lib.
