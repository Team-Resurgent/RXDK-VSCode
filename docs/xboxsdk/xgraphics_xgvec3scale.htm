XGVec3Scale

Scales a 3D vector.

XGVECTOR3* XGVec3Scale(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV,
  FLOAT s
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR3 structure.
s
[in] Scaling value.

Return Values

Pointer to a XGVECTOR3 structure that is the scaled vector.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3Scale function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
