Performance Tools

This section provides information regarding the performance tools in this version of the Xbox Development Kit (XDK). The tools described in this section are subject to change in later versions of the XDK.

The tools in this section include:

  • CPU Profiler for Xbox (CPX)
  • Performance Investigator for Xbox (PIX)
  • Remote Performance Monitor Tool (XbPerfMon)
  • TriggerProfile Tool
  • Xbox Call-Attributed Profile Viewer (XbPerfView)
  • Xbox Call-Attributed Profiler (XbPerf)
  • Performance Library (XPerf) Reference
