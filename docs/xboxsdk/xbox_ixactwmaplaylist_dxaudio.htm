IXACTWmaPlayList

Applications use the methods of the IXACTWmaPlayList interface to manipulate XACT WMA playlists.

An IXACTWmaPlayList object is obtained by calling the IXACTSoundBank::CreateWmaPlayList function.

IXACTWmaPlayList methods Add
GetCurrentSongInfo
GetCurrentSongInfoEx
GetProperties
Next
Previous
Remove
SetCurrent
SetPlaybackBehavior

The IXACTWmaPlayList interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
