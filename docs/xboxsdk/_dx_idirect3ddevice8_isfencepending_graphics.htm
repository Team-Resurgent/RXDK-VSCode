IDirect3DDevice8::IsFencePending

Checks to see if a given fence has been processed.

BOOL IsFencePending(
  DWORD Fence
);

Parameters

Fence
[in] Handle of a valid fence. Use IDirect3DDevice8::InsertFence to insert a fence and get a handle.

Return Values

The function returns TRUE if given fence has not yet been processed.
