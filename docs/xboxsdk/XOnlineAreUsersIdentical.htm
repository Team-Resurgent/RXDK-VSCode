XOnlineAreUsersIdentical

Macro that, given two XUID structures, determines whether the two structures represent the same player.

#define XOnlineAreUsersIdentical(pXUID1,pXUID2) \
    (((pXUID1)->qwUserID == (pXUID2)->qwUserID) && \
    (XOnlineUserGuestNumber((pXUID1)->dwUserFlags) == XOnlineUserGuestNumber((pXUID2)->dwUserFlags)))

Parameters

pXUID1
[in] Pointer to an XUID structure that specifies the first player.
pXUID2
[in] Pointer to an XUID structure that specifies the second player.

Return Values

The return value is TRUE if the two XUID structures represent the same player; FALSE otherwise.

The return value is of type BOOL.

Remarks

Rather than comparing the structures directly, use XOnlineAreUsersIdentical to determine whether two XUID structures represent the same player.

Requirements

Header: Declared in Xonline.h.
