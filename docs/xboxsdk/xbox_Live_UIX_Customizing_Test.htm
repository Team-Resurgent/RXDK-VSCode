The UI Test Feature

 

Illustration  UI Test Feature Displaying a List Box.

 

You can use the UI test feature (UIX_UITEST_FEATURE) to test your title's custom ITitleUIPlugin implementation. The feature is enabled and started just as other features are. It provides controls and an interface to enable testing screen elements and interaction with the custom plug-in. Items can be added to a list box in order to test scrolling and greying behavior. Using the feature does not require that the title log on or be connected to Xbox Live. The feature uses the one-player logon skin.

Note    The UI test feature is available only in debug builds.
