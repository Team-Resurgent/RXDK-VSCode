XMediaObject::Discontinuity

Handles a break in the input stream.

HRESULT Discontinuity();

Return Values

Discontinuity should return DS_OK if successful.

Remarks

An application should call Discontinuity when an Xbox Media Object needs more input data but that data is not available for some reason. Discontinuity resets the internal state of the Xbox Media Object and initializes any state variables needed. For streams with finite duration, discontinuity should be called after submission of the final packet in the stream.
