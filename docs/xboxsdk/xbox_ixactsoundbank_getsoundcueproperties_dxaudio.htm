IXACTSoundBank::GetSoundCueProperties

Returns properties of a sound cue.

HRESULT GetSoundCueProperties(
    DWORD dwSoundCueIndex,
    PXACT_SOUNDCUE_PROPERTIES pSoundCueProperties
);

Parameters

dwSoundCueIndex
[in] Sound cue index to query.
pSoundCueProperties
[out] Pointer to an XACT_SOUNDCUE_PROPERTIES structure that recieves the properties. This parameter cannot be NULL.

Return Values

Returns DS_OK if successful, an error code otherwise. If any wavebank used by the sound is not currently loaded, the function returns E_FAIL.

Remarks

If there is an active cue instance associated with the cue index, the 3D properties will be returned from the cue instance and not the sound bank.

Requirements
