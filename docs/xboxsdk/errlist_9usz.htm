Error Codes


This table lists the system error codes returned by GetLastError for many XTL functions.

Titles can retrieve error codes by calling the GetLastError function. The use of GetLastError is optional; it can be called by a title to find out the specific reason for a function failure. Note that, unless specifically documented in the Return Values section of a function's reference page, the error codes set by a function are intended to be primarily used as a debugging aid, not as a method for controlling execution flow within a title. Also note that there may be errors listed below that are not currently returned by any XTL functions.

See also: SetLastError, Winsock Error Codes, WSAGetLastError, WSASetLastError.
