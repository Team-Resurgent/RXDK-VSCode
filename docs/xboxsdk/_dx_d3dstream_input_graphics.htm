D3DSTREAM_INPUT

Describes a vertex buffer's data stream.

typedef struct _D3DSTREAM_INPUT{  // Xbox extension
  IDirect3DVertexBuffer8 *VertexBuffer;
  UINT Stride;
  UINT Offset;
} D3DSTREAM_INPUT;

Members

VertexBuffer
Pointer to the associated vertex buffer.
Stride
Stride between the elements, in bytes.
Offset
Byte offset from the start of the associated vertex buffer.

Requirements
