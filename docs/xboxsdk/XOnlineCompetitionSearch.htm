XOnlineCompetitionSearch

Searches the competitions dataset for result lists using a specific query ID.

HRESULT XOnlineCompetitionSearch(
    DWORD dwQueryID,
    DWORD dwTarget,
    DWORD dwPage,
    DWORD dwResultsPerPage,
    DWORD dwNumSearchAttributes,
    const XONLINE_ATTRIBUTE *pSearchAttributes,
    DWORD dwNumResultAttributeSpecs,
    const XONLINE_ATTRIBUTE_SPEC *pResultAttributeSpecs,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwQueryID
[in] ID of the search query. Obtain this ID from XLAST.
dwTarget
[in] ID of the target of the query.
dwPage
[in] Page number of the results being requested. Page numbers start with 0.
dwResultsPerPage
[in] Number of entries per page of results. Maximum is 255.
dwNumSearchAttributes
[in] Number of search attributes in the array specified by the pSearchAttributes parameter.
pSearchAttributes
[in] Pointer to an array of XONLINE_ATTRIBUTE structures, each specifying the parameter ID and value for the search to be performed. These attributes must be of the same type and in the same order as listed in the Parameters tab for the query in XLAST.
dwNumResultAttributeSpecs
[in] Number of result attribute specifiers.
pResultAttributeSpecs
[in] Starting address of an array of XONLINE_ATTRIBUTE_SPEC structures that specify the type and size, in bytes, of each result attribute. The dwType member of the first attribute spec must be set to XONLINE_QUERY_ENTITY_ID. The remainder of the attribute specs must match the list and ordering of return values specified under the Return Values tab for this query in XLAST.
hWorkEvent
[in, optional] Handle of an event you want to associate with this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by XOnlineCompetitionSearch. Use this handle in calls to XOnlineTaskContinue to perform work associated with this task. When the task is complete, call XOnlineCompetitionSearchGetResults to get the search results. Call XOnlineTaskClose to close the task.

Return Values

Returns S_OK if successful, an error code otherwise. The error codes returned by this function include the following.

Error Code Description
XONLINE_E_QUERY_UNEXPECTED_ATTRIBUTE The list of attributes in the pSearchAttributes parameter does not match the list of attributes on the server, which were specified by XLAST.
XONLINE_E_QUERY_SPEC_COUNT_MISMATCH The number of attributes listed in the pResultAttributeSpecs parameter does not match the list of return values on the server, which were specified by XLAST.
XONLINE_E_QUERY_ATTRIBUTE_TOO_LONG A string or BLOB (binary block of data) type attribute listed in pSearchAttributes or pResultAttributeSpecs specifies a length that is smaller than the data stored on the server.
XONLINE_E_QUERY_DATASET_NOT_FOUND The target, specified in the dwTarget parameter, was not found on the server.
XONLINE_E_QUERY_PROCEDURE_NOT_FOUND The search query, specified in the dwQueryID parameter was not found on the server.
