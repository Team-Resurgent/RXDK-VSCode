/TESTXBEVERSION (XBE Version)

/TESTXBEVERSION:version

Where:

version
XBE version number for the image.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Select the Xbox Image folder.
  3. Select the Certificate property page.
  4. Modify the XBE Version property.

Specifying /TESTXBEVERSION N is the same as speciying /TESTVERSION N*256. Users should note that /TESTVERSION has been deprecated.
