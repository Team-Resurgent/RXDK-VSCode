XONLINE_FRIENDS_RESULTS_TYPE

Defines the values used to specify types of player list results.

typedef enum {
    XONLINE_FRIENDS_RESULTS_FRIENDREQUESTFAILED = 0,
    XONLINE_FRIENDS_RESULTS_FRIENDREMOVEFAILED,
    NUM_XONLINE_FRIENDS_RESULTS_TYPES,
} XONLINE_FRIENDS_RESULTS_TYPE;

Constants

XONLINE_FRIENDS_RESULTS_FRIENDREQUESTFAILED
The player list update request failed.
XONLINE_FRIENDS_RESULTS_FRIENDREMOVEFAILED
The removal of a user from the player list has failed.
NUM_XONLINE_FRIENDS_RESULTS_TYPES
Indicates the number of player list result types defined. Not a valid value to be used in player list structures or functions.
