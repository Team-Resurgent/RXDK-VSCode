XPhysicalSize

The XPhysicalSize function returns the size, in bytes, of a memory block previously allocated by the XPhysicalAlloc function. 

SIZE_T XPhysicalSize(
  LPVOID lpAddress  // address of memory block
);

Parameters

lpAddress
[in] Pointer to the base address of the region of physical memory.

Return Values

If the function succeeds, the return value is the size, in bytes, of the memory block specified in lpAddress.

The function does not return a failure code, and will either fault or return a size of at least 4 KB if an invalid address is specified.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
