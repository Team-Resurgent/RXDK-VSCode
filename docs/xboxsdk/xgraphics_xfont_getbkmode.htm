XFONT::GetBkMode

Detemines if the text-cell background is opaque or transparent.

int __stdcall GetBkMode();

Parameters

None.

Return Values

The function returns one of the following values:

Value Meaning
XFONT_TRANSPARENT The background of the text cell is filled with the background color of the surface on which the text is rendered.
XFONT_OPAQUE The background of the text cell is filled with the color set using XFONT::SetBkColor.

Requirements
