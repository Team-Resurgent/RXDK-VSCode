IDirect3DDevice8::SetLight

Assigns a set of lighting properties for this device.

HRESULT SetLight(
  DWORD Index,
  CONST D3DLIGHT8* pLight
);

Parameters

Index
[in] Zero-based index of the set of lighting properties to set. The maximum allowed Index value is 4095. If a set of lighting properties exists at this index, it is overwritten by the new properties specified in pLight.
pLight
[in] Pointer to a D3DLIGHT8 structure, containing the lighting-parameters to set.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

To reduce the memory automatically allocated, index values should be kept as low as possible and reused where feasible.
