IDirect3DDevice8::UpdateOverlay

Updates the overlay.

HRESULT WINAPI UpdateOverlay(
  D3DSurface *pSurface, 
  CONST RECT *SrcRect, 
  CONST RECT *DstRect, 
  BOOL EnableColorKey, 
  D3DCOLOR ColorKey
);

Parameters

pSurface
[in] Pointer to an IDirect3DSurface8 interface, representing the overlay. This surface must have been created using the D3DFMT_YUY2 surface format.
SrcRect
[in] Pointer to a RECT structure containing the source rectangle. Cannot be NULL.
DstRect
[in] Pointer to a RECT structure containing the destination rectangle. Cannot be NULL.
EnableColorKey
[in] TRUE to enable color key operations on the surface being used for an overlay. If EnableColorKey is FALSE, the overlay will be rendered opaquely.
ColorKey
[in] D3DCOLOR value to replace with transparent black. This is always a 32-bit ARGB color, independent of the source image format.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

Overlays must have been enabled using IDirect3DDevice8::EnableOverlay before calling UpdateOverlay. In addition, before calling UpdateOverlay, at least one call to either IDirect3DDevice8::Present or IDirect3DDevice8::Swap must occur.

UpdateOverlay does not display the overlay surface immediately. The display occurs on the next vertical blank. Check the status of the overlay by calling IDirect3DDevice8::GetOverlayUpdateStatus. Until GetOverlayUpdateStatus returns TRUE, writing to the previously displayed overlay surface causes visual glitches.
