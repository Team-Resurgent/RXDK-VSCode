Downloading a DSP Effects Image

Before using XACT for audio playback, we download a digital signal processor (DSP) effects image to the global processor in the Xbox Media Communications Processor (MCP). For this example, we'll use the default DSP image (Dsstdfx.bin) which is located in the \source\dsound\dsp\ directory of the XDK. Simply copy the file into your resource folder on the Xbox console. The example below uses a folder named "media", which is a subfolder of the application directory.

Load the sound bank data file using our utility function, XACTLoadFile (listed later in the Input and Support Routines section). Once the DSP image is loaded, we can download it to the processor using the IXACTEngine::DownloadEffectsImage routine. Finally, we want to free the pointer returned from XACTLoadFile.

For more information on DSP effects images, see the DSP Effects section.

void DownloadDSPEffectsImage()
{
    // Load our DSP effects image
    VOID* pDSPImage = NULL;
    DWORD dwFileSize;
    XACTLoadFile( "d:\\media\\dsstdfx.bin", &pDSPImage, &dwFileSize );

    // Specify I3DL2 reverb and crosstalk support
    DSEFFECTIMAGELOC dseil;
    dseil.dwI3DL2ReverbIndex = GraphI3DL2_I3DL2Reverb;
    dseil.dwCrosstalkIndex   = GraphXTalk_XTalk;	
    HRESULT hr = g_pXACT->DownloadEffectsImage( pDSPImage, dwFileSize, &dseil, NULL );
    assert( SUCCEEDED(hr) );

    // Free our copy of the DSP image
    free( pDSPImage );
}
