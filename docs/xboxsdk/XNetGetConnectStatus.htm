XNetGetConnectStatus

Determines the current state of the XSNL secure connection with the specified in_addr.

DWORD XNetGetConnectStatus(
    const IN_ADDR ina
);

Parameters

ina
The in_addr for which the secure connection state is to be determined. The specified in_addr must have been returned from a previous call to XNetXnAddrToInAddr, XNetServerToInAddr, or XOnlineGetServiceInfo.

Return Values

Returns the state of the secure connection, which can be one of the following values:

Value Description
XNET_CONNECT_STATUS_IDLE The connection has not started.
XNET_CONNECT_STATUS_PENDING The connection is being established.
XNET_CONNECT_STATUS_CONNECTED The connection has been established.
XNET_CONNECT_STATUS_LOST The connection has been lost.

Remarks

For more information about secure connections, see the discussion in XNetConnect.

Requirements

Header: Declared in Winsockx.h. for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. For online titles, use the secure version of the library file (XOnlines.lib) for final testing (not available for system link play).
