LD_UPDATE

Used in the XOnlineTitleUpdateEx function to pass launch data to a title across an update.

typedef struct{
    DWORD dwContext;
    DWORD dwReserved[6];
    BYTE Data[MAX_LAUNCH_DATA_SIZE - 28];
} LD_UPDATE, *PLD_UPDATE;

Members

dwContext
Title-defined value that will be returned to the title in the dwContext member of the LD_FROM_UPDATE structure (see XGetLaunchInfo) after the update.
dwReserved
Reserved; do not use.
Data
Title-defined launch data that will be returned to the title in the Data member of the LD_FROM_UPDATE structure after the update.

Remarks

The title can use the Data member for whatever launch data it desires. For example, Data could contain the name of the Xbox executable that initiated the update, so that the default executable (default.xbe, launched by the updater after the update) could reboot into that specific executable.
