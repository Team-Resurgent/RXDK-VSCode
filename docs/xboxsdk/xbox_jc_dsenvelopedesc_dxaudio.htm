DSENVELOPEDESC

Contains parameters for the envelope generators.

typedef struct _DSENVELOPEDESC{
    DWORD dwEG;
    DWORD dwMode;
    DWORD dwDelay;
    DWORD dwAttack;
    DWORD dwHold;
    DWORD dwDecay;
    DWORD dwRelease;
    DWORD dwSustain;
    LONG lPitchScale;
    LONG lFilterCutOff;
} DSENVELOPEDESC, *LPDSENVELOPEDESC;

Members

dwEG
Envelope generator to set data on. Valid values are:
DSEG_AMPLITUDE
Amplitude-only generator generator.
DSEG_MULTI
Multi-function envelope generator.
dwMode
Value that specifies the envelope mode. Can be one of the following values:
DSEG_MODE_DISABLE
The envelope is disabled and the envelope value is always full-scale.
DSEG_MODE_DELAY
Starts with the envelope at zero amplitude with an initial delay.
DSEG_MODE_ATTACK
Bypasses the initial delay and goes directly to the attack envelope.
DSEG_MODE_HOLD
Bypasses the attack segment and immediatly goes full scale.
dwDelay
Delay value for envelope. This 12-bit number is multiplied by 16 and is used as the number of sample frames (32 samples/sample frame) to delay prior to starting the Attack segment. Valid ranges: [0–0xfff]. To convert from time to and from seconds, Time (seconds) = (dwDelay*512)/48000 and dwDelay = (Time (Seconds) * 48000)/512.
dwAttack
Attack value for envelope. This 12-bit number is multiplied by 16 and is used as the number of sample frames (32 samples/sample frame) the Attack segment spans. Valid ranges: [0–0xfff]. To convert from time to and from seconds, Time (seconds) = (dwDelay*512)/48000 and dwAttack = (Time (Seconds) * 48000)/512.
dwHold
Hold value for envelope. This 12-bit number is multiplied by 16 and is used as the number of sample frames (32 samples/sample frame) to hold after completion of the Attack segment. Valid ranges: [0–0xfff]. To convert from time to and from seconds, Time (seconds) = (dwDelay*512)/48000 and dwHold = (Time (Seconds) * 48000)/512.
dwDecay
Decay value for envelope. This 12-bit number is multiplied by 16 and is used as the number of sample frames (32 samples/sample frame) that the Decay segment spans. Valid ranges: [0–0xfff]. To convert from time to and from seconds, Time (seconds) = (dwDecay*512)/48000 and dwDecay = (Time (Seconds) * 48000)/512.

Note   The dwDecay value behaves differently based on the dwSustain level. dwDecay is the time it takes to reach zero sustain. So, if dwSustain is 0, dwDecay will work as expected. If dwSustain is not 0, the hardware voice will decay until it reaches that sustain value and then stop. So a 16 second decay will take 16 seconds only of dwSustain is set to zero. It will take much less time of dwSustain is 30, for example.

dwSustain
Sustain level. 8-bit value from 0-255, with 255 representing 100% level.
dwRelease
Release value for envelope. This 12-bit number is multiplied by 16 and is used as the number of sample frames (32 samples/sample frame) the Release segment spans. Valid ranges: [0–0xfff]. To convert from time to and from seconds, Time (seconds) = (dwDelay*512)/48000 and dwRelease = (Time (Seconds) * 48000)/512.
lPitchScale (multi-function EG only)
Value that specifies the amount the frequency envelope modulates the pitch frequency. Can be a value from -128 to 127. A value of 0 specifies that there should be no frequency modulation, while a value of 127 specifies approximately +1 octave of frequency modulation. This change is linear in terms of pitch.
lFilterCutOff (multi-function EG only)
Value that specifies the amount the frequency envelope modulates the filter cutoff frequency. Can be a value from -128 to 127. A value of -128 specifies that the filter cutoff should be modulated by –8 octaves. A value of 127 indicates a modulation of approximately +8 octaves.
