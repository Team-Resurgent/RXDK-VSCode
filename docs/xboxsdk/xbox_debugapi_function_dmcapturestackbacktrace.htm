DmCaptureStackBackTrace

Retrieves function return addresses from the frames on the call stack.

DMHRAPI DmCaptureStackBackTrace(
    ULONG FramesToCapture,
    PVOID *BackTrace
);

Parameters

FramesToCapture
Number of stack frames to walk, beginning with the current stack frame.
BackTrace
PVOID array that receives the return addresses. The array must be large enough to contain at least FramesToCapture variables of type PVOID.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If more stack frames are specified in FramesToCapture than are available, the available return addresses are written to the BackTrace array first, followed by an address of 0 (zero) to indicate that no more addresses are available.

This function cannot walk the stack if the 'frame pointer omission' compiler optimization is enabled. To use this function with optimized (retail) builds, turn off frame pointer omission by adding the /Oy- flag to the C/C++ command-line options.

The load address for an Xbox executable is always 0x00010000.

Requirements

Header: Declared in XbDm.h.

Import Library: Use XbDm.lib.
