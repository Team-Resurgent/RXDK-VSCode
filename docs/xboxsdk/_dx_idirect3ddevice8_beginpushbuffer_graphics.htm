IDirect3DDevice8::BeginPushBuffer

Begins a push-buffer recording bracket, causing all subsequent Direct3D commands (until a call to IDirect3DDevice8::EndPushBuffer) to be recorded into the specified push-buffer.

HRESULT BeginPushBuffer(
  IDirect3DPushBuffer8 *pPushBuffer
);

Parameters

pPushBuffer
[out] Pointer to the IDirect3DPushBuffer8 structure where the Direct3D commands will be recorded.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

The following functions are not permitted within a IDirect3DDevice8::BeginPushBuffer/IDirect3DDevice8::EndPushBuffer bracket:

  • IDirect3DDevice8::CopyRects
  • IDirect3DDevice8::InsertFence
  • IDirect3DDevice8::Present
  • IDirect3DResource8::MoveResourceMemory
  • IDirect3DDevice8::UpdateTexture

If the pPushBuffer points to a previously recorded push-buffer, the push-buffer data previously recorded will be overwritten.

This function cannot be called if IDirect3DDevice8::SetIndices has been called with its BaseVertexIndex parameter set to a non-zero value. In addition, while recording a push-buffer, IDirect3DDevice8::SetIndices cannt be called with a non-zero value for its BaseVertexIndex parameter.

IDirect3DDevice8::EndPushBuffer should be called when done recording.

Requirements

  Import Library: Use D3d8.lib.
