DSMIXBINVOLUMEPAIR

Contains the identifier for the mixbin and the volume it should be set to (when applicable).

typedef struct _DSMIXBINVOLUMEPAIR{
    DWORD dwMixBin;  // MixBin identifier
    LONG lVolume;  // MixBin volume
} DSMIXBINVOLUMEPAIR, *LPDSMIXBINVOLUMEPAIR;

Members

dwMixBin
Value that contains the mixbin identifier. Can be one of the Mixbin Constants.
lVolume
Value that specifies the volume that this mixbin should be set to (when applicable).

Remarks

For a 3D voice, it is required that the first four mixbins be the ones in DSMIXBINVOLUMEPAIRS_REQUIRED_3D. Then, either a 4-channel algorithm can be used or a 5-channel algorithm can be used. This is determined by whether the fifth mixbin is DSMIXBIN_FRONT_CENTER or not. The 5-channel algorithm is very similar to the 4-channel algorithm except that some of the sound is routed out of the front-left and front-right speakers and routed to the center speaker.
