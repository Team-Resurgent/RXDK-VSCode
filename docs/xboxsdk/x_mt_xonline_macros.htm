Xbox Live Macros

The following macros are used in Xbox Live:

Macro Description
TOPOLOGY_ID

Builds a topology ID from a round number and an event number.
XONLINE_MSG_PROP_TAG

Builds a property tag from a type and an ID.
XOnlineAreUsersIdentical Macro that, given two XUID structures, determines whether the two structures represent the same player.
XOnlineIsUserGuest Macro that, given the dwUserFlags value from an XUID structure, determines whether a player is a guest.
XOnlineIsUserNicknameAllowed Macro that, given the dwUserFlags value from an XUID structure, determines whether a player is permitted to choose nicknames.
XOnlineIsUserPurchaseAllowed Macro that, given the dwUserFlags value from an XUID structure, determines whether a player is permitted to purchase Xbox Live premium content or subscription offerings.
XOnlineIsUserSharedContentAllowed Macro that, given the dwUserFlags value from an XUID structure, determines whether a player is permitted to use shared content.
XOnlineIsUserVoiceAllowed Macro that, given the dwUserFlags value from an XUID structure, determines whether a player is permitted to use voice chat.
XOnlineMessageGetPropId Extracts a property ID from a property tag.
XOnlineMessageGetPropType Extracts a property type from a property tag.
XOnlineOfferingIsFree Macro that, given the fOfferingFlags value from an XONLINEOFFERING_INFO structure, determines whether the offering is free.
XOnlineSetUserGuestNumber Macro that sets the guest number in the dwUserFlags member of an XUID structure.
XOnlineUserCountryId Macro that, given the dwUserFlags value from an XUID structure, returns the player's country identifier.
XOnlineUserDisconnectRating Macro that, given the dwUserFlags value from an XUID structure, determines a player's disconnect rating.
XOnlineUserGuestNumber Macro that, given the dwUserFlags value from an XUID structure, determines the guest number (1–3) of the player.
XOnlineUserNoShowRating Macro that, given the dwUserFlags value from an XUID structure, determines a player's no show rating.
XOnlineXUIDIsTeam Macro that, given a pointer to an XUID structure, determines whether that XUID is a team XUID (as opposed to a player XUID).
