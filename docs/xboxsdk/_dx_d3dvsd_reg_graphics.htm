D3DVSD_REG

Binds a single vertex register to a vertex element from the vertex stream.

D3DVSD_REG( _VertexRegister, _Type ) \
          (D3DVSD_MAKETOKENTYPE(D3DVSD_TOKEN_STREAMDATA) | \
          ((_Type) << D3DVSD_DATATYPESHIFT) | (_VertexRegister))

Parameters

_VertexRegister
Address of the vertex register. For the fixed function pipeline, the input registers have the following fixed mapping:
D3DVSDE_POSITION
Register 0.
D3DVSDE_BLENDWEIGHT
Register 1.
D3DVSDE_NORMAL
Register 2.
D3DVSDE_DIFFUSE
Register 3.
D3DVSDE_SPECULAR
Register 4.
D3DVSDE_FOG
Register 5. Xbox extension.
D3DVSDE_BACKDIFFUSE
Register 7. Xbox extension.
D3DVSDE_BACKSPECULAR
Register 8. Xbox extension.
D3DVSDE_TEXCOORD0
Register 9.
D3DVSDE_TEXCOORD1
Register 10.
D3DVSDE_TEXCOORD2
Register 11.
D3DVSDE_TEXCOORD3
Register 12.
_Type
Specifies the dimensionality and arithmetic data type. The following values are defined.
D3DVSDT_FLOAT1
1-D float expanded to (value, 0.0, 0.0, 1.0).
D3DVSDT_FLOAT2
2D float expanded to (value, value, 0.0, 1.0).
D3DVSDT_FLOAT3
3D float expanded to (value, value, value, 1.0).
D3DVSDT_FLOAT4
4-D float.
D3DVSDT_D3DCOLOR
4-D packed unsigned bytes mapped to 0.0 to 1.0 range. In double word format this is ARGB, or in byte ordering it would be B, G, R, A.
D3DVSDT_SHORT2
2D signed short expanded to (value, value, 0.0, 1.0).
D3DVSDT_SHORT4
4-D signed short.
D3DVSDT_NORMSHORT1
Xbox extension. 1-D signed, normalized short expanded to (value, 0.0, 0.0, 1.0). Signed, normalized shorts map from -1.0 to 1.0.
D3DVSDT_NORMSHORT2
Xbox extension. 2D signed, normalized short expanded to (value, value, 0.0, 1.0). Signed, normalized shorts map from -1.0 to 1.0.
D3DVSDT_NORMSHORT3
Xbox extension. 3D signed, normalized short expanded to (value, value, value, 1.0). Signed, normalized shorts map from -1.0 to 1.0.
D3DVSDT_NORMSHORT4
Xbox extension. 4-D signed, normalized short expanded to (value, value, value, value). Signed, normalized shorts map from -1.0 to 1.0.
D3DVSDT_NORMPACKED3
Three signed, normalized components packed in 32-bits. (11,11,10). Each component ranges from -1.0 to 1.0. Expanded to (value, value, value, 1.0). The following code shows how this packing can be done: 
for (DWORD i = 0; i < dwNumNormals; i++)
{
   pPackedNormals[i] = ( ( ( (DWORD)(pSrcNormals[i].z * 511.0f) )  & 0x3ff ) << 22L ) |
                       ( ( ( (DWORD)(pSrcNormals[i].y * 1023.0f) ) & 0x7ff ) << 11L ) |
                       ( ( ( (DWORD)(pSrcNormals[i].x * 1023.0f) ) & 0x7ff ) <<  0L );
}
D3DVSDT_SHORT1
Xbox extension. 1-D signed short expanded to (value, 0., 0., 1). Signed shorts map to the range [-32768, 32767].
D3DVSDT_SHORT3
Xbox extension. 3D signed short expanded to (value, value, value, 1). Signed shorts map to the range [-32768, 32767].
D3DVSDT_PBYTE1
Xbox extension. 1-D packed byte expanded to (value, 0., 0., 1). Packed bytes map to the range [0, 1].
D3DVSDT_PBYTE2
Xbox extension. 2D packed byte expanded to (value, value, 0., 1). Packed bytes map to the range [0, 1].
D3DVSDT_PBYTE3
Xbox extension. 3D packed byte expanded to (value, value, value, 1). Packed bytes map to the range [0, 1].
D3DVSDT_PBYTE4
Xbox extension. 4-D packed byte expanded to (value, value, value, value). Packed bytes map to the range [0, 1].
D3DVSDT_FLOAT2H
Xbox extension. 3D float that expands to (value, value, 0.0, value). Useful for projective texture coordinates.
D3DVSDT_MAX
This constant is not supported in this release.

Remarks

The following DirectX 8.0 definitions are not supported on the Xbox game system:

  • D3DVSDE_BLENDINDICES
  • D3DVSDE_NORMAL2
  • D3DVSDE_POSITION2
  • D3DVSDE_PSIZE
  • D3DVSDE_TEXCOORD4 through D3DVSDE_TEXCOORD7
