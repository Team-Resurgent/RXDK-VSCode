LD_DEMO

Required for launching into and out of demos. The launch data type is LDT_TITLE.

typedef struct _LD_DEMO{
    DWORD dwID;
    DWORD dwRunmode;
    DWORD dwTimeout;
    CHAR szLauncherXBE[64];
    CHAR szLaunchedXBE[64];
    BYTE Reserved[MAX_LAUNCH_DATA_SIZE - 140];
} LD_DEMO, *PLD_DEMO;

Members

dwID
Value that identifies the structure as demo launch data. This value is set by the demo launcher and is defined as LAUNCH_DATA_DEMO_ID.
dwRunmode
Determines whether the demo was run in kiosk mode or whether the user explicitly selected to run the demo. Can be either of the following values:
Flag Description
XLDEMO_RUNMODE_KIOSKMODE The demo was run in kiosk mode.
XLDEMO_RUNMODE_USERSELECTED The demo was selected by the user.
dwTimeout
Number of milliseconds of inactivity after which the demo game should return to the demo launcher. This value corresponds to the CDX timeout value.
szLauncherXBE
Path to the demo launcher. The demo game should use this value to return to the demo launcher after the timeout has expired or as the result of the player selecting to return to the demo launcher.
szLaunchedXBE
Path to demo game.
Reserved
Reserved for use by the demo launcher. The demo game should not alter this data.

Remarks

The demo should never assume it was launched by CDX. It should instead determine if it was launched by CDX by doing the following:

DWORD dwLaunchStatus = XGetLaunchInfo( &dwLaunchDataType, &LaunchData );
if( dwLaunchStatus == ERROR_SUCCESS )
{
   if( dwLaunchDataType == LDT_TITLE )
   {
      PLD_DEMO pDemoLauncher = (PLD_DEMO)(&LaunchData);
      if( pDemoLauncher->dwID == LAUNCH_DATA_DEMO_ID )
      {
         bLaunchedByCDX = TRUE;
      }
   }
}

A demo game can return to CDX by calling

XLaunchNewImage( pDemoLauncher->szLauncherXBE, &LaunchData );

Notice that the original LaunchData should be passed back to CDX unchanged.
