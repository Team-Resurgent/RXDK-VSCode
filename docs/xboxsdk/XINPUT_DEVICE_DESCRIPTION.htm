XINPUT_DEVICE_DESCRIPTION

Structure returned by the XInputGetDeviceDescription function that contains the vendor identifier, product identifier, and version of a specified device.

typedef struct{
    WORD wVendorID;
    WORD wProductID;
    WORD wVersion;
} XINPUT_DEVICE_DESCRIPTION, *PXINPUT_DEVICE_DESCRIPTION;

Members

wVendorID
Vendor identifier of the device.
wProductID
Product identifier of the device.
wVersion
Firmware version of the device.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
