VolumeFog Sample

The VolumeFog sample demonstrates a technique for displaying fog in a 3D scene.

Overview

Xbox programs can load fog volumes of arbitrary size from files containing models of the fog volumes. Fog volumes are loaded as meshes. When a program uses a fog volume, it finds the objects in the scene that intersect the fog volume. It then renders the appearance of the object based on the object's distance from the viewer.
