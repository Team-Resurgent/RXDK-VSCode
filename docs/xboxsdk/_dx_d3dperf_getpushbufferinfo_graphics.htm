D3DPERF_GetPushBufferInfo

Gets performance information about the push-buffer.

void WINAPI D3DPERF_GetPushBufferInfo(
  D3DPUSHBUFFERINFO *pPushBufferInfo
);

Parameters

pPushBufferInfo
[out, retval] Pointer to a D3DPUSHBUFFERINFO structure which contains information about the push-buffer since the last call to D3DPERF_Reset.

Return Values

None.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
