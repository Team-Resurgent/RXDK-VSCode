UIXPlugin Sample

The UIXPlugin sample shows how to implement a custom user interface plug-in that works with the UIX Drop-in UI library.

Overview

The UIX library provides a complete drop-in UI for Xbox Live authentication (signing on) and friends list. However, developers may want to customize the UI rendering to better match their game. This sample shows the source code for a skeletal UI plug-in, which can be used as a starting point to work with any game. Games implementing a custom UI plug-in will generally need to tie the plug-in into their own font code, and possibly their own resource loading and management code as well.

Path

Source: (XDK Root)\Samples\Xbox\Networking\UIXPlugin

Remarks

Run copytoxb.bat to copy the required fonts and skin files to the Xbox before running this sample.

Be aware that using your own custom UI plug-in means you will not pick up any bug fixes for the default UI plug-in that may appear in later XDK releases.
