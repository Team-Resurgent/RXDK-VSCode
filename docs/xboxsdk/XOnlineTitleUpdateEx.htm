XOnlineTitleUpdateEx

Begins the download and installation of an available update for the game title, and allows passing launch data across the update.

HRESULT XOnlineTitleUpdateEx(
    const LD_UPDATE *pldUpdate
);

Parameters

pldUpdate

[in] LD_UPDATE structure that contains launch data to be passed to the title after the reboot. This data is passed to the updated title in the LD_FROM_UPDATE structure returned by XGetLaunchInfo.

Return Values

If the function succeeds, it does not return; the Xbox reboots into the updater (update.xbe).

Remarks

XOnlineTitleUpdateEx performs the same function as XOnlineTitleUpdate, except that it allows launch data to be passed across the update.

When an update is available for a title, the XOnlineLogon task will fail with XONLINE_E_LOGON_UPDATE_REQUIRED. The title will then need to perform an update before XOnlineLogon will succeed and allow the title to make other online calls.

XOnlineTitleUpdateEx reboots the Xbox console to download and install the update for the game title, and then reboots to the newly updated game title. The update persists on the Xbox console until the next update is available for the title.

Before calling XOnlineTitleUpdateEx, the title should inform the player that an update is about to occur and that the Xbox console will reboot in order to perform the update. The player should be given the option to forego updating, and, if desired, to proceed playing the game without invoking any online functionality.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
