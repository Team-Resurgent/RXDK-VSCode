DmGetXboxName

Retrieves the machine name of the Xbox game system.

DMHRAPI DmGetXboxName(
  LPSTR   szName, 
  LPDWORD pcch
  );

Parameters

szName
Pointer to a buffer that receives the machine name. The maximum length of the machine name string is 255 characters, plus the terminating null character.
pcch
Pointer to a DWORD variable that specifies the size of the szName buffer.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise. If the szName buffer is too small to contain the machine name, the function returns XBDM_BUFFER_TOO_SMALL.

Remarks

When called from the console, DmGetXboxName retrieves the registered name of the Xbox game system. When called from the development system, DmGetXboxName retrieves the name of the last Xbox game system that was accessed from the development system.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
