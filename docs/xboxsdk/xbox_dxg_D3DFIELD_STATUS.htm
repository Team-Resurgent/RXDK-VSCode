D3DFIELD_STATUS

Describes the number of Vblank interrupts since startup and the display field type.

typedef struct _D3DFIELD_STATUS{
    D3DFIELDTYPE   Field;
    UINT           VBlankCount;
} D3DFIELD_STATUS;

Members

Field
The display field type. For a TV display, the returned value is either D3DFIELD_EVEN or D3DFIELD_ODD. For monitor output, the value is D3DFIELD_PROGRESSIVE.
VBlankCount
The number of Vblank interrupts since startup.
