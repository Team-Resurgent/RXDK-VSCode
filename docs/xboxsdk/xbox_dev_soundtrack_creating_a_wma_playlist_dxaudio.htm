Creating and Playing a WMA Playlist

When SetXACTPlayList is called, the previous playlist (if any) is cleared, a new one is created, the playlist is populated based on the soundtrack identifier, and playback is started. By organizing the function this way, we can call SetXACTPlayList several times as needed with the index of the soundtrack we want to play.

Note    The dwIndex parameter of the SetXACTPlayList function represents the vector index of the soundtrack we wish to play, not the ID of the soundtrack. The ID of the soundtrack is obtained through the enumeration process, and does not correspond to soundtrack's position in the list.

void SetXACTPlayList( DWORD dwIndex )
{
    // Release the previous play list as needed
    if( g_pWMAPlaylist )
    {
        g_pWMAPlaylist->Release();
        g_pWMAPlaylist = NULL;
    }

    // Create the new play list
    HRESULT hr = g_pSoundBank->CreateWmaPlayList( g_dwSoundCueIndex,
                                                  XACT_FLAG_WMAPLAYLIST_PLAYBACK_LOOP,
                                                  &g_pWMAPlaylist );
    assert( SUCCEEDED(hr) );

    // Add the new soundtrack songs to the play list
    XACT_WMA_PLAYLIST_ADD xwpAddInfo = { 0 };
    xwpAddInfo.dwType = eXACTWmaPlayListAdd_Soundtrack;
    xwpAddInfo.dwSoundtrackId = g_vSoundtracks[dwIndex].uSoundtrackID;
    g_pWMAPlaylist->Add( &xwpAddInfo, NULL );

    // Play
    g_pSoundBank->Play( g_dwSoundCueIndex, NULL, XACT_FLAG_SOUNDCUE_AUTORELEASE, NULL );
}
