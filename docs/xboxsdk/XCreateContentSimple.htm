XCreateContentSimple

Creates a simple content metadata file that contains no content signatures. 

BOOL XCreateContentSimple(
    DWORD dwTitleID,
    XOFFERING_ID xOfferingID,
    DWORD dwContentFlags,
    LPCWSTR lpContentName,
    LPCSTR lpDestinationDirectory
);

Parameters

dwTitleID
[in] Title identifier that will be associated with the content. If zero is specified for TitleID, the title identifier of the calling title is used.
xOfferingID
[in] Offering identifier of the content.
dwContentFlags
[in] Title-specific flags describing the content. These flags can be used to filter content enumeration, and this value is returned in the XCONTENT_FIND_DATA structure when content is enumerated.
lpContentName
[in] Pointer to a null-terminated Unicode string that specifies the display name that describes the content.
lpDestinationDirectory
[in] Pointer to a null-terminated string that specifies the full path of the destination directory into which the metadata file will be written (and into which the content has been or will be installed). Use XGetContentInstallLocationFromIDs to determine the appropriate content installation directory.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

Installation of packaged contents is typically handled by either a content download installer or a DVD-based package installer. However, a title that wishes to perform its own content installation and use its own content verification scheme can call XCreateContentSimple. XCreateContentSimple will create a simple content metadata file that contains no content signatures, but makes the content visible to both the Xbox Dashboard and to the content enumeration functions.

The XCreateContentSimple function creates metadata files for offline content only. That is, content that is delivered from the title DVD.

Requirements
