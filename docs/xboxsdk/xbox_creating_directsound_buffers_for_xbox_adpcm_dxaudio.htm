Creating DirectSound Buffers for Xbox ADPCM Compressed Waves

This article will cover filling in the XBOXADPCMWAVEFORMAT structure and then using it to create a DirectSound buffer for a wave file compressed using the Xbox ADPCM codec.

  • Prerequisites and Requirements
  • Implementation Steps
  • Related Topics

Prerequisites and Requirements

To create a DirectSound buffer for Xbox ADPCM compressed wave files, you will need a mono or stereo wave file compressed using the Xbox ADPCM codec and a valid DirectSound object.

Implementation Steps

Complete the following steps to create a DirectSound buffer for a Xbox ADPCM compressed wave file.

  1. Declare a XBOXADPCMWAVEFORMAT structure.
    2. XBOXADPCMWAVEFORMAT xboxadpcmwf;
  2. Fill in the members of the embedded WAVEFORMATEX structure (which is the wfx member of the XBOXADPCMWAVEFORMAT structure).
    1. Set the wFormatTag member to WAVE_FORMAT_XBOX_ADPCM.
      2. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
    2. Set the nChannels member to the number of channels in the wave file. For ADPCM compressed waves, it is currently limited to one (mono) or two (stereo). For this example, lets say that we have a stereo wave file.
      3. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
    3. Set the nSamplesPerSec member to the frequency of the wave file in hertz. For this example, lets use 48,000.
      4. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
    4. Set the nBlockAlign member of the structure to 36 times the number of channels specified by nChannels. For this example, we're using a stereo wave file (2 channels), so the value should be 72.
      5. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
    5. Set the nAvgBytesPerSec member to be equal to number of samples per second (specified by nSamplesPerSec) times nBlockAlign divided by 64. So for this example, that would be 48000*72/64, which comes out to 54000.
      6. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
    6. Set the wBitsPerSample member to the number of bits per sample. Xbox ADPCM uses 4 bits per sample, so we'll use that number here.
      7. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
    7. Set the cbSize member to the size of the XBOXADPCMWAVEFORMAT structure minus the size of the embedded WAVEFORMATEX structure.
      8. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
  3. Set the wSamplesPerBlock member to the number of samples per block. Must be set to 64.
    4. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;
  4. Declare a DSBUFFERDESC structure.
    5. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;
  5. Fill in the members of the DSBUFFERDESC structure.
    1. Set the dwSize member to the size of the structure.
      2. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;

dsbdesc.dwSize = sizeof(DSBUFFERDESC);
    2. Set the dwFlags member to one or more of the flags that specify the capabilities for the DirectSound buffer. For this example, lets just make a plain buffer by specifying zero.
      3. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;

dsbdesc.dwSize = sizeof(DSBUFFERDESC);
dsbdesc.dwFlags = 0;
    3. Set the dwBufferBytes member to the size of the new buffer. For this example, we'll set it to zero, which tells the DirectSound not to allocate any memory for this DirectSound buffer. This makes the title responsible for allocating the memory and must call IDirectSoundBuffer8::SetBufferData to tell DirectSound where to found the sound buffer data.
      4. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;

dsbdesc.dwSize = sizeof(DSBUFFERDESC);
dsbdesc.dwFlags = 0;
dsbdesc.dwBufferBytes = 0;
    4. Set the lpwfxFormat member equal to your XBOXADPCMWAVEFORMAT structure. You will need to cast the XBOXADPCMWAVEFORMAT structure to a WAVEFORMATEX structure.
      5. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;

dsbdesc.dwSize = sizeof(DSBUFFERDESC);
dsbdesc.dwFlags = 0;
dsbdesc.dwBufferBytes = 0;
dsbdesc.lpwfxFormat = (LPWAVEFORMATEX)&xboxadpcmwf;
  6. Declare a pointer to a DirectSound buffer.
    7. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;

dsbdesc.dwSize = sizeof(DSBUFFERDESC);
dsbdesc.dwFlags = 0;
dsbdesc.dwBufferBytes = 0;
dsbdesc.lpwfxFormat = (LPWAVEFORMATEX)&xboxadpcmwf;

LPDIRECTSOUNDBUFFER pBuffer;
  7. Use DirectSoundCreateBuffer or IDirectSound8::CreateSoundBuffer to create the DirectSound buffer. For this example, lets use DirectSoundCreateBuffer.
    8. XBOXADPCMWAVEFORMAT xboxadpcmwf;

xboxadpcmwf->wfx.wFormatTag = WAVE_FORMAT_XBOX_ADPCM;
xboxadpcmwf->wfx.nChannels = 2;
xboxadpcmwf->wfx.nSamplesPerSec = 48000;
xboxadpcmwf->wfx.nBlockAlign = 72;
xboxadpcmwf->wfx.nAvgBytesPerSec = 54000;
xboxadpcmwf->wfx.wBitsPerSample = 4;
xboxadpcmwf->wfx.cbSize = sizeof(xboxadpcmwf) - sizeof(xboxadpcmwf->wfx);
xboxadpcmwf->wSamplesPerBlock = 64;

DSBUFFERDESC dsbdesc;

dsbdesc.dwSize = sizeof(DSBUFFERDESC);
dsbdesc.dwFlags = 0;
dsbdesc.dwBufferBytes = 0;
dsbdesc.lpwfxFormat = (LPWAVEFORMATEX)&xboxadpcmwf;

LPDIRECTSOUNDBUFFER pBuffer;

DirectSoundCreateBuffer(&dsbdesc;, &pBuffer);
