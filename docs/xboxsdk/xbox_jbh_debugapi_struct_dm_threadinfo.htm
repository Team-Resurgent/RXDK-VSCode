DM_THREADINFO

The DM_THREADINFO structure is used by the DmGetThreadInfo function.

typedef struct _DM_THREADINFO {
    DWORD SuspendCount;
    DWORD Priority;
    LPVOID XapiData;
    LPVOID TlsBase;
} DM_THREADINFO, *PDM_THREADINFO;

Members

SuspendCount
Number of times this thread has been suspended.
Priority
Priority of this thread.
XapiData
Pointer to the XTL data used by this thread.
TlsBase
Pointer to the base of the thread local storage.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
