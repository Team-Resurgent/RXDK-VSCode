Downloading Images to the Global Processor

This tutorial describes how to download a DSP effects image to the global processor in the Xbox Media Communications Processor (MCP).

Currently, any application using 3D sound in either DirectMusic or DirectSound must download an effects image containing both Crosstalk and I3DL2 reverberation effects to the global processor.

Prerequisites and Requirements

To complete this tutorial, you will either need the default DSP effects image (Dsstdfx.bin) included with the samples in the XDK or you will need to create your own DSP effects image using the either the Xgpimage.exe or DSPBuilder tool.

  7. Specify the indexes of the effects images that are used by DirectSound in the DSEFFECTIMAGELOC structure. The indexes are found in the .h file generated by the xgpimage.exe tool. For this example, let's say that the DSP effects image being used doesn't include the I3DL2 reverb effect (so we'll use DSFX_IMAGELOC_UNUSED for the index) and the index for the Crosstalk cancellation effect is MIXBIN0_CHAIN_XTALK.

EffectLoc.dwCrosstalkIndex = MIXBIN0_CHAIN_XTALK;
