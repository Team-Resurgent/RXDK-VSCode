DmGo

Resumes suspended Xbox title threads.

DMHRAPI DmGo(void);

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If a debugging event handler is registered for DM_EXEC events, it will receive a notification that execution on the system has resumed.

Note    Threads that have been stopped as a result of a notification (exceptions, breakpoints, single-step, DmStopOn) will not resume after a call to DmGo unless a call to DmContinueThread is made.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
