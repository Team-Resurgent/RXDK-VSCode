Leaderboard Types

Past versions of XLAST supported only one type of leaderboard. XLAST now offers the following leaderboard types.

  • Standard Leaderboards
  • Competition Leaderboard Templates

Note    All leaderboards for a given title must have unique numeric identifiers, regardless of the leaderboard type. For example, if your title's Standard Leaderboards are numbered 1–20, no other leaderboards for that title may be numbered 1–20. If you were to then add Team Leaderboards, you would probably begin numbering them at 21.

Standard Leaderboards

Standard Leaderboards contain stats for individual players. This is the type of leaderboard supported by XLAST before version 3. Therefore, Standard Leaderboards are available in all versions of XLAST.

Competition Leaderboard Templates

Competition Leaderboards are created dynamically at run time on Xbox Live. You cannot create them in XLAST Stats version 3. Instead, you use XLAST Stats version 3 to define Competition Leaderboard Templates. These templates look almost identical to the final Competition Leaderboards. However, you typically make only a handful of them, as compared to the potentially hundreds of generic leaderboards a game can have.

Note    The IDs for Competition Leaderboard Templates must be in the range of 1–15, inclusive. Therefore, the total number of Competition Leaderboard Templates is limited to 15.
