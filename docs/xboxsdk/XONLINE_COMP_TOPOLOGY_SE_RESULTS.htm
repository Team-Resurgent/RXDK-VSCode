XONLINE_COMP_TOPOLOGY_SE_RESULTS

Used to hold the results of a topology search of a single-elimination competition.

typedef struct{
    DWORD dwBaseWidth;
    DWORD dwRoundsReturned;
    DWORD dwTotalResultEntries;
    DWORD dwResultsSize;
    PBYTE pbResults;
    DWORD dwNumResultAttributeSpecs;
    PXONLINE_ATTRIBUTE_SPEC pResultAttributeSpecs;
} XONLINE_COMP_TOPOLOGY_SE_RESULTS, *PXONLINE_COMP_TOPOLOGY_SE_RESULTS;

Members

dwBaseWidth
Same as the dwTopWidth parameter of the XOnlineCompetitionTopologySingleElimination function.
dwRoundsReturned
Number of rounds returned in the results.
dwTotalResultEntries
Total number of items in the array specified by the pbResults member of this structure.
dwResultsSize
Size, in bytes, of the array specified by the pbResults member of this structure.
pbResults
Array that contains the results of the topology search.
dwNumResultAttributeSpecs
Number of attribute specifiers in the pResultAttributeSpecs member of this structure.
pResultAttributeSpecs
Array that contains the attribute specifiers for the topology search. Attribute specifiers are the column numbers of each attribute in the data set being searched. These must match the attribute specs specified in the original call to XOnlineCompetitionTopologySingleElimination.
