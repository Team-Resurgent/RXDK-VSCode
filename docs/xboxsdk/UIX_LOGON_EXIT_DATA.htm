UIX_LOGON_EXIT_DATA

When logon is successful, this structure contains a list of controllers that were used to log on.

typedef struct _UIX_LOGON_EXIT_DATA{
    DWORD pMappedControllers[XONLINE_MAX_LOGON_USERS];
} UIX_LOGON_EXIT_DATA, *UIX_LOGON_EXIT_DATA;

Members

pMappedControllers
Array containing controller and port assignments for all players logged onto Xbox Live. For more information, see the Remarks section below.

Remarks

The pMappedControllers member contains an array that maps controllers to specific ports. Each DWORD in the array corresponds to a port number. In other words, array location 0 indicates controller port 0, array location 1 corresponds to controller port 1, and so on. The values that each DWORD in the array holds is a controller number. A value of −1 indicates no controller is connected to the port. So, for example, if controller 0 is plugged into port 2, then pMappedControllers[2] contains a 0. If controller 1 is plugged into port 3, then pMappedControllers[3] contains a 1. Suppose the array contains the values 0, −1, 1, 2. In this case, controller 0 is plugged into port 0, there is no controller plugged into port 1, and controllers 1 and 2 are in ports 2 and 3, respectively.
