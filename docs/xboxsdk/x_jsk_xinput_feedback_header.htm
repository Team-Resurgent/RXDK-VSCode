XINPUT_FEEDBACK_HEADER

The XINPUT_FEEDBACK_HEADER structure describes the status of a current feedback function call to a device attached to the system. 

typedef struct _XINPUT_FEEDBACK_HEADER
{
    DWORD           dwStatus;
    HANDLE OPTIONAL hEvent;
    BYTE            Reserved[XINPUT_FEEDBACK_HEADER_INTERNAL_SIZE];
} XINPUT_FEEDBACK_HEADER, *PXINPUT_FEEDBACK_HEADER;

Members

dwStatus
[out] Flag of the stats of an operation called by XInputSetState.
hEvent
[in] Handle to an event created with CreateEvent. This parameter is optional.
Reserved[XINPUT_FEEDBACK_HEADER_INTERNAL_SIZE]
Reserved for internal use.

Remarks

Objects defined with this structure are used to track the status of current function calls to provide feedback to an attached device.

The large Reserved member is allocated to avoid dynamically allocating and freeing memory on every call, increasing efficiency.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
