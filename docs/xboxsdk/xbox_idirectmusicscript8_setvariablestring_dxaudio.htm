IDirectMusicScript8::SetVariableString

Sets the value of the specified string value in a DirectMusic script.

HRESULT SetVariableString(
    char *pszVariableName,
    const char *pszValue,
    DMUS_SCRIPT_ERRORINFO *pErrorInfo
);

Parameters

pszVariableName
Address of string value that contains the variable name.
pszValue
Address of a string buffer that contains the value to set.
pErrorInfo
Address of a DMUS_SCRIPT_ERRORINFO structure.

Return Values

Returns S_OK if successful, an error code otherwise. Returns DMUS_E_SCRIPT_VARIABLE_NOT_FOUND if the specified variable doesn't exist in the script. Will return DISP_E_TYPEMISMATCH if the value passed cannot be converted to a string.
