DmUnloadExtension

Unloads debugger extensions.

DMHRAPI DmUnloadExtension(
  HANDLE hModule
  );

Parameters

hModule
Handle of the extension to unload.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.
