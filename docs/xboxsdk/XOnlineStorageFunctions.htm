Storage Functions

The Xbox Live storage functions include the following.

Function Description
XOnlineStorageCreateServerPath Creates a server path from constituent information.
XOnlineStorageDeleteFile

Allows a title to delete files per user title managed storage.
XOnlineStorageDownload Downloads a storage item from the specified storage facility.
XOnlineStorageDownloadToMemory

Downloads a storage item that was previously uploaded using the XOnlineStorageUploadFromMemory function.
XOnlineStorageDownloadToMemoryGetResults

Retrieves the storage download results from a call to the XOnlineStorageDownloadToMemory function.
XOnlineStorageEnumerate

Enumerates files that are stored on global storage.
XOnlineStorageEnumerateGetResults

Returns the results of a call to XOnlineStorageEnumerate.
XOnlineStorageGetInstallLocation Retrieves the path to the storage item file downloaded by XOnlineStorageDownload.
XOnlineStorageGetProgress Retrieves the amount of work performed so far on an Xbox Live storage upload or download operation. The function returns the amount of work performed as both a percentage and a fraction.
XOnlineStorageSetFamilyTitleID

Sets the family title ID for storage.
XOnlineStorageUpload Uploads a file attachment to accompany player statistics written by the XOnlineStatWrite function.
XOnlineStorageUploadByServerPath

Uploads the contents of the specified directory to the specified storage facility as a single storage item.
XOnlineStorageUploadFromMemory

Uploads a file attachment to title storage from memory.
