DM_UTILITY_DRIVE_INFO

The DM_UTILITY_DRIVE_INFO structure, returned by the DmGetUtilityDriveInfo function, contains information about the Xbox console's utility drive.

typedef struct{
    DWORD dwFlags;
    DWORD rgdwTitleId[3];
} DM_UTILITY_DRIVE_INFO, *DM_UTILITY_DRIVE_INFO;

Members

dwFlags
Combination of one or more of the following flags.
Constant Description
DM_UTILITY_DRIVE_0_NEVER_USED The cache partition on drive 0 has never been used.
DM_UTILITY_DRIVE_1_NEVER_USED The cache partition on drive 1 has never been used.
DM_UTILITY_DRIVE_2_NEVER_USED The cache partition on drive 2 has never been used.
DM_UTILITY_DRIVE_0_NOT_USED The cache partition on drive 0 does not appear in the list of most recently used cache partitions.
DM_UTILITY_DRIVE_1_NOT_USED The cache partition on drive 1 does not appear in the list of most recently used cache partitions.
DM_UTILITY_DRIVE_2_NOT_USED The cache partition on drive 2 does not appear in the list of most recently used cache partitions.
rgdwTitleId
Array of three title identifiers

Requirements

Header: Declared in XbDm.h, XboxDbg.h.
