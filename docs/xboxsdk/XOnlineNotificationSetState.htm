XOnlineNotificationSetState

Sets the current online state information for the specified player.

HRESULT XOnlineNotificationSetState(
    DWORD dwUserIndex,
    DWORD dwStateFlags,
    XNKID sessionID,
    DWORD dwStateData,
    const BYTE *pbStateData
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player whose state information is being set. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwStateFlags
[in] Online state flags to set. This parameter can be a combination of the values in the following table:
Value Description
XONLINE_FRIENDSTATE_FLAG_JOINABLE The player is in a session that other players can join.
XONLINE_FRIENDSTATE_FLAG_ONLINE The player is online.
XONLINE_FRIENDSTATE_FLAG_PLAYING The player is playing a game.
XONLINE_FRIENDSTATE_FLAG_VOICE The player has voice capability.

To have a player appear to be offline—that is, "cloaked"—clear the XONLINE_FRIENDSTATE_FLAG_ONLINE flag; even though the player is online, he or she will not appear on the friends lists of other players. However, if the cloaked player sends an invitation to a player on their friends list, he or she will appear online to that player until the invitation is resolved.

sessionID
[in] XNKID structure that identifies the player's current session. Setting this structure to all zeroes expires any previously sent game invitations.
dwStateData
[in] Size, in bytes, of the buffer pointed to by pStateData. This value can range from zero to MAX_STATEDATA_SIZE. If the value is zero, pStateData must be NULL.
pbStateData
[in] Pointer to a buffer containing title-specific state information. This parameter can be NULL if no such information is provided.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The data specified in pStateData is included in the XONLINE_FRIEND structure returned by other online functions (such as XOnlineFriendsGetLatest).

State changes are sent via the XOnlineLogon task handle. Titles must pump this handle in order for state changes to be sent to the online service.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
