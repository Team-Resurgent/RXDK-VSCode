UIXCreateAudioPlugin

Creates the default Xbox Drop-in UI audio plug-in.

HRESULT UIXCreateAudioPlugin(
    VOID *pXactEngine,
    VOID *pXactSoundBank,
    ITitleAudioPlugin **ppAudioPlugin
);

Parameters

pXactEngine
[in] Pointer to the title's initialized XACT engine previously returned by XACTEngineCreate.
pXactSoundBank
[in] Pointer to an initialized XACT sound bank previously returned by IXACTEngine::CreateSoundBank.
ppAudioPlugin
[out] Address of an ITitleAudioPlugin pointer that receives the audio plug-in interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

ITitleAudioPlugin implements the audio plug-in interface to the Xbox Drop-in UI engine. The engine calls methods on the interface to provide sound for the user interface. Titles can provide their own implementation or use the default ITitleAudioPlugin obtained with UIXCreateAudioPlugin.

To free the audio plug-in interface when it is no longer needed, call Release on the ITitleAudioPlugin pointer.

Requirements
