IDirect3DDevice8::SetPixelShaderProgram

Sets the current pixel shader to a previously created pixel shader.

HRESULT WINAPI SetPixelShaderProgram(
  CONST D3DPIXELSHADERDEF *pPSDef
);

Parameters

pPSDef
[in] Pointer to a D3DPIXELSHADERDEF structure describing the pixel shader program to execute. If pPSDef is NULL, the previously set pixel shader program will be released.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

This function is identical to IDirect3DDevice8::SetPixelShader except that it does not specify a pixel shader object; it specifies pixel shader microcode. If a title uses this function, Direct3D doesn't have to copy the microcode into an internally allocated buffer. Instead, the title will simply use the microcode pointed to by pPSDef.

The title must keep the microcode resident in memory as long as the pixel shader is active.

Calling this function will reset any shader constants set by SetPixelShaderConstant to their values as defined in the pixel shader program.
