GlobalLock

The GlobalLock function locks a global memory object and returns a pointer to the first byte of the object's memory block.

Note  All Globalxxx and Localxxx memory management functions use the title's default heap, and provide less flexibility than the heap functions. Titles should use the heap functions instead.

LPVOID GlobalLock(
  HGLOBAL hMem   // handle to global memory object
);

Parameters

hMem
[in] Handle to the global memory object. This handle is returned by either the GlobalAlloc or GlobalReAlloc function.

Return Values

The return value is a pointer to the first byte of the memory block.

Remarks

Because all memory allocated with the global or local memory functions is fixed, the address returned is the same as the handle passed in.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
