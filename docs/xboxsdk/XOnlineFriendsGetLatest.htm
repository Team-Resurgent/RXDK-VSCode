XOnlineFriendsGetLatest

Returns the latest friends list for the specified player.

DWORD XOnlineFriendsGetLatest(
    DWORD dwUserIndex,
    DWORD dwFriendBufferCount,
    PXONLINE_FRIEND pFriendBuffer
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player whose friends list is to be enumerated. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwFriendBufferCount
[in] Number of XONLINE_FRIEND structures that will fit into the buffer pointed to by pFriendBuffer. This value should always be MAX_FRIENDS.
pFriendBuffer
[out] Pointer to a buffer that receives the friends list as an array of XONLINE_FRIEND structures. The buffer must be large enough to hold at least dwFriendBufferCount structures of type XONLINE_FRIEND.

Return Values

The return value is the actual number of friend records written to pFriendBuffer.

Remarks

XOnlineFriendsGetLatest returns the latest friends list maintained by the friends list enumeration task. The enumeration task is created when the title calls XOnlineFriendsEnumerate. The title should not call XOnlineFriendsGetLatest until XOnlineTaskContinue returns XONLINETASK_S_RESULTS_AVAIL for this task. If the title calls XOnlineFriendsGetLatest before receiving this return value, XOnlineFriendsGetLatest returns a zero-length list.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
