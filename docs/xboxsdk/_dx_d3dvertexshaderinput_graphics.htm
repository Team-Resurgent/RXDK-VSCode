D3DVERTEXSHADERINPUT

Defines how vertex attibutes are read from one or more streams.

typedef struct{
    DWORD StreamIndex;
    DWORD Offset;
    DWORD Format;
    BYTE TessType;
    BYTE TessSource;
} D3DVERTEXSHADERINPUT;

Members

StreamIndex
The index of the stream to be used for input.
Offset
Byte offset from the start of the stream given in StreamIndex.
Format
The dimensionality and data type for this attribute. This may be one of the following values:
Value Description
D3DVSDT_NONE This flag is only set to indicate that the vertex shader input will be unused.
D3DVSDT_FLOAT1 1D float expanded to (value, 0.0, 0.0, 1.0).
D3DVSDT_FLOAT2 2D float expanded to (value, value, 0.0, 1.0).
D3DVSDT_FLOAT3 3D float expanded to (value, value, value, 1.0).
D3DVSDT_FLOAT4 4D float.
D3DVSDT_D3DCOLOR 4D packed unsigned bytes mapped to 0.0 to 1.0 range. In double-word format this is ARGB, or in byte ordering it would be B, G, R, A.
D3DVSDT_SHORT2 2D signed short expanded to (value, value, 0.0, 1.0).
D3DVSDT_SHORT4 4D signed short.
D3DVSDT_NORMSHORT1 Xbox extension. 1D signed, normalized short expanded to (value, 0.0, 0.0, 1.0). Signed, normalized shorts map from −1.0 to 1.0.
D3DVSDT_NORMSHORT2 Xbox extension. 2D signed, normalized short expanded to (value, value, 0.0, 1.0). Signed, normalized shorts map from −1.0 to 1.0.
D3DVSDT_NORMSHORT3 Xbox extension. 3D signed, normalized short expanded to (value, value, value, 1.0). Signed, normalized shorts map from −1.0 to 1.0.
D3DVSDT_NORMSHORT4 Xbox extension. 4D signed, normalized short expanded to (value, value, value, value). Signed, normalized shorts map from −1.0 to 1.0.
D3DVSDT_NORMPACKED3 Three signed, normalized components packed in 32-bits (11, 11, 10). Each component ranges from −1.0 to 1.0. Expanded to (value, value, value, 1.0). Refer to the Remarks section to see how this flag can be used.
D3DVSDT_SHORT1 Xbox extension. 1D signed short expanded to (value, 0.0, 0.0, 1.0). Signed shorts map to the range [−32768, 32767].
D3DVSDT_SHORT3 Xbox extension. 3D signed short expanded to (value, value, value, 1.0). Signed shorts map to the range [−32768, 32767].
D3DVSDT_PBYTE1 Xbox extension. 1D packed byte expanded to (value, 0.0, 0.0, 1.0). Packed bytes map to the range [0, 1].
D3DVSDT_PBYTE2 Xbox extension. 2D packed byte expanded to (value, value, 0.0, 1.0). Packed bytes map to the range [0, 1].
D3DVSDT_PBYTE3 Xbox extension. 3D packed byte expanded to (value, value, value, 1.0). Packed bytes map to the range [0, 1].
D3DVSDT_PBYTE4 Xbox extension. 4D packed byte expanded to (value, value, value, value). Packed bytes map to the range [0, 1].
D3DVSDT_FLOAT2H Xbox extension. 3D float that expands to (value, value, 0.0, value). Useful for projective texture coordinates.
TessType

The tessellation type. This may be one of the following values:
Value Description
0 Perform no tessellation.
1 Enable tessellator-generated normals.
2 Enable tessellator-generated UV texture coordinates.

TessSource
The source register used for tessellation.

Remarks

The following code shows how packing should be done when using D3DVSDT_NORMPACKED3:

for (DWORD i = 0; i < dwNumNormals; i++)
{
   pPackedNormals[i] = ( ( ( (DWORD)(pSrcNormals[i].z * 511.0f) )  & 0x3ff ) << 22L ) |
                       ( ( ( (DWORD)(pSrcNormals[i].y * 1023.0f) ) & 0x7ff ) << 11L ) |
                       ( ( ( (DWORD)(pSrcNormals[i].x * 1023.0f) ) & 0x7ff ) <<  0L );
}
