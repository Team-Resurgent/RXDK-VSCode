XACT_WMA_PLAYLIST_ADD_TYPE

XACT WMA playlist add types.

typedef enum _XACT_WMA_PLAYLIST_ADD_TYPE{
    eXACTWmaPlayListAdd_Directory = 1,
    eXACTWmaPlayListAdd_File,
    eXACTWmaPlayListAdd_Soundtrack,
    eXACTWmaPlayListAdd_SoundtrackSong,
    eXACTWmaPlayListAdd_Max,
} XACT_WMA_PLAYLIST_ADD_TYPE;

Constants

eXACTWmaPlayListAdd_Directory
Indicates adding all WMA files from a given directory to a WMA playlist.
eXACTWmaPlayListAdd_File
Indicates adding a single WMA file to a WMA playlist.
eXACTWmaPlayListAdd_Soundtrack
Indicates adding all files from a soundtrack to a WMA playlist.
eXACTWmaPlayListAdd_SoundtrackSong
Indicates adding a single song from a soundtrack to a WMA playlist.
eXACTWmaPlayListAdd_Max
Maximum number of XACT WMA playlist add types.
