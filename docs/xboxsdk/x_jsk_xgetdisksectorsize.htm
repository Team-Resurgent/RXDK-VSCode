XGetDiskSectorSize

The XGetDiskSectorSize function returns the sector size used by the specified file system device.

DWORD XGetDiskSectorSize(
    LPCSTR lpRootPathName
);

Parameters

lpRootPathName
[in] Pointer to a null-terminated string that specifies the root directory of the device being queried (for example, T:\).

Return Values

If the function succeeds, the return value is the sector size for the specified device.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

The path specified by lpRootPathName must be a valid root directory of a disk device on the Xbox game system (for example, D:\, T:\, U:\, or Z:\).

This function is useful for determining the sector size of a disk device on which a file will be opened for unbuffered access (by calling CreateFile with FILE_FLAG_NO_BUFFERING). Reads and writes to such files must be done for a number of bytes that are integer multiples of the sector size.

The size of a fixed sector on the hard disk is 512 bytes. The size of a sector on a CD or DVD drive is 2048 bytes.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
