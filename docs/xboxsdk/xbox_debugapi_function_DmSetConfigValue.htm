DmSetConfigValue

Programmatically sets hardware configuration values on the Xbox development console.

DMHRAPI DmSetConfigValue(
    ULONG ulValueIndex,
    ULONG ulType,
    PVOID pValue,
    ULONG cbValueLength
);

Parameters

ulValueIndex
Zero-based index number of the value to set.
ulType
Type of the value to set. Currently, this parameter must be set to REG_DWORD.
pValue
Address of the data to copy into the configuration value.
cbValueLength
Number of bytes in the data. Currently, this must be set to sizeof(DWORD).

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
