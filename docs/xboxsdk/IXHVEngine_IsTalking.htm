IXHVEngine::IsTalking

Determines whether voice data is currently being received by a specified remote talker.

BOOL IsTalking(
    XUID xuidRemoteTalker
);

Parameters

xuidRemoteTalker
[in] XUID of the registered remote talker.

Return Values

Returns TRUE if voice data is currently being received by a remote talker, FALSE otherwise.

Remarks

This function returns TRUE if voice data is currently being decoded for the specified remote talker. Due to network and processing delays, the remote talker may not be speaking into the microphone at the precise moment that IsTalking returns TRUE.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
