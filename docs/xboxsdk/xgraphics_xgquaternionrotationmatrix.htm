XGQuaternionRotationMatrix

Builds a quaternion from a rotation matrix.

XGQUATERNION* XGQuaternionRotationMatrix(
  XGQUATERNION* pOut,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGQUATERNION structure that is the result of the operation.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to the XGQUATERNION structure built from a rotation matrix.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGQuaternionRotationMatrix function can be used as a parameter for another function.

Requirements

  Import Library: Use xgraphics.lib.
