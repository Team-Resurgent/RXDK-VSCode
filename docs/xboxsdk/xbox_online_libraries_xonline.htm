Xbox Networking Libraries

The Xbox Title Libraries contain services used to enable Xbox Live and system link play. Most of the Xbox Live–related libraries are described in this section.

To initialize a network connection, all titles designed to enable Xbox Live or system link play must use Xbox Secure Network Library (SNL).
