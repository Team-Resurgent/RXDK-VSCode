XOnlineMatchSessionFindFromID

Searches for the online-game session that corresponds to the specified session identifier.

HRESULT XOnlineMatchSessionFindFromID(
    XNKID SessionID,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

SessionID
[in] Session identifier for the online-game session to search for.
hWorkEvent
[in, optional] Optional. Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.
phTask
[out] Address of an XONLINETASK_HANDLE that uniquely identifies this search. After the task has been pumped to completion, call XOnlineMatchSearchGetResults to obtain information about the session.

Return Values

If XOnlineMatchSessionFindFromID succeeds, the return value is S_OK.

If XOnlineMatchSessionFindFromID fails, the return value is one of the following error codes.

Value Description
E_OUTOFMEMORY Insufficient memory. Free some memory.
WSAEADDRNOTAVAIL See Winsock Error Codes for more information.
WSACONNRESET See Winsock Error Codes for more information.
Other See Online Errors for information about other possible errors.

Remarks

The XOnlineMatchSessionFindFromID function is called from an Xbox that wishes to join an online-game session that is hosted by a different Xbox.

The hSession task handle must be checked repeatedly until there is no more work to do.

The title is responsible for freeing this handle using the XOnlineTaskClose function.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
