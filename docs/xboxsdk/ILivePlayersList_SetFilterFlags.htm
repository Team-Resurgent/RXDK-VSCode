ILivePlayersList::SetFilterFlags

Sets flags that filter which players are shown on the players list.

HRESULT SetFilterFlags(
    CONST DWORD Flags
);

Parameters

Flags
[in] Bitmask of flags that determines the displayed players. The mask is combined (using a bitwise AND) with the player flags returned by ITitlePlayersListItem::GetBitflags, and players with a nonzero result are displayed. If Flags is zero, all players (regardless of flags) are displayed.

Return Values

Returns S_OK.

Remarks

SetFilterFlags does not need to be called unless filter flags different from those passed in the UIX_PLAYERS_PARAMS structure at feature start are desired. Filter flags may be set only while the players list is active (calling SetFilterFlags at any other time performs no action).

Requirements
