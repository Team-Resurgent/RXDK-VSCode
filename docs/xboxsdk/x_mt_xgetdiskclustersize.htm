XGetDiskClusterSize

The XGetDiskClusterSize function returns the cluster size used by the specified file system device.

DWORD XGetDiskClusterSize(
    LPCSTR lpRootPathName
);

Parameters

lpRootPathName
[in] Pointer to a null-terminated string that specifies the root directory of the device being queried (for example, T:\).

Return Values

If the function succeeds, the return value is the cluster size for the specified device.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

The path specified by lpRootPathName must be a valid root directory of a device on the Xbox game system (for example, D:\, T:\, U:\, or Z:\).

The function returns the cluster size of the device, which is the number of bytes used on the device per allocation unit (essentially, the bytes per sector times the number of sectors per allocation unit). Any allocation from the device will use at least this many bytes, and allocations will be made in integral multiples of this cluster size.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
