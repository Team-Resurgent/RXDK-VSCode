XONLINE_STAT_ELO

Contains information for calculating the ELO rating for a pair of competitors.

typedef struct{
    XUID xuid1;
    XUID xuid2;
    DWORD dwLeaderBoardID;
    DWORD dwConditionalIndex;
    double W;
    double C1;
    double C2;
} XONLINE_STAT_ELO, *PXONLINE_STAT_ELO;

Members

xuid1
XUID structure identifying player 1.
xuid2
XUID structure identifying player 2.
dwLeaderBoardID
Leaderboard to be updated. Leaderboards are also referred to as scoreboards.
dwConditionalIndex
One-based index of the XONLINE_STAT_CONDITIONAL procedure that determines whether this update occurs, or zero if always updated.
W
Double float value between 0.0 and 1.0, inclusive, containing the relative score of the winner to the loser. The value 0.5 indicates a tie. A value in the range 0.5 <= W <= 1.0 indicates a win for player 1, while a value in the range and 0.0 <= W <= 0.5 indicates a loss for player 1. A value of W == 1.0 indicates a complete and crushing victory for player 1, and a value of W == 0.0 indicates a total victory for player 2—and an utter, humiliating defeat for player 1.
C1
Double float value between 0.0 and 1.0 that represents a fraction applied to the final point change added or subtracted from player 1's current rating. This constitutes a weighting value of the range in which player 1's rating currently falls.
C2
Double float value between 0.0 and 1.0 that represents a fraction applied to the final point change added or subtracted from the player 2's current rating. This constitutes a weighting value of the range in which player 2's rating currently falls.

Remarks

For an explanation of how the ELO rating system works, see the Stats and the ELO Rating System.
