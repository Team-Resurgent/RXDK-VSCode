DSI3DL2LISTENER

Contains the information related to listener for Interactive 3-D Audio Level 2 (I3DL2) standard support.

typedef struct _DSI3DL2LISTENER{
    LONG lRoom;
    LONG lRoomHF;
    FLOAT flRoomRolloffFactor;
    FLOAT flDecayTime;
    FLOAT flDecayHFRatio;
    LONG lReflections;
    FLOAT flReflectionsDelay;
    LONG lReverb;
    FLOAT flReverbDelay;
    FLOAT flDiffusion;
    FLOAT flDensity;
    FLOAT flHFReference;
} DSI3DL2LISTENER, *LPDSI3DL2LISTENER;

Members

lRoom
Value that adjusts the intensity level and low-pass filter for the room effect. Can be a value between –10000 mB and 0 mB. The default value is –10000 mB.
lRoomHF
Value that specifies the attenuation at high frequencies relative to the intensity at low frequencies. Can be a value between –10000 mB and 0 mB. The default value is 0 mB.
flRoomRolloffFactor
Value that controls the rolloff effect intensity vs. distance. Can be a value between 0.0 and 10.0. The default value is 0.0.
flDecayTime
Value that specifies the decay time at low frequencies. Can be a value between 0.1 and 20.0 seconds. The default value is 1.0 seconds.
flDecayHFRatio
Value that specifies the ratio of high-frequency decay time relative to the low-frequency decay time. Can be a value between 0.1 and 2.0. The default value is 0.5.
lReflections
Value that adjusts the intensity level of early reflections, relative to the lRoom value. Can be a value between –10000 mB and 1000 mB. The default value is –10000 mB.
flReflectionsDelay
Value that specifies the delay time of the first reflection, relative to the direct path. Can be a value between 0.0 and 0.3 seconds. The default value is 0.02 seconds.
lReverb
Value that adjusts the intensity of late reverberations, relative to the lRoom value. Can be a value between –10000 mB and 2000 mB. The default value is –10000 mB.
flReverbDelay
Value that specifies the time limit between the early reflections and the late reverberation, relative to the time of the first reflection. Can be a value between 0.0 and 0.1 seconds. The default value is 0.04 seconds.
flDiffusion
Value that controls the echo density in the late reverberation decay. Can be a value between 0.0% and 100.0%. The default value is 100.0%.
flDensity
Value that controls the modal density in the late reverberation decay. Can be a value between 0.0% and 100.0%. The default value is 100.0%.
flHFReference
Value that contains the reference high frequency. Can be a value between 20.0 Hz and 20000.0 Hz. The default value is 5000.0 Hz.
