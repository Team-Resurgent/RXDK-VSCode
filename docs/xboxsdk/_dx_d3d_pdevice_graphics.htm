D3D__pDevice

Points to the current IDirect3DDevice8 object.

IDirect3DDevice8* D3D__pDevice;

Remarks

This variable points the same object that would be returned by IDirect3D8::CreateDevice. If no device has been created, this variable will be NULL.

This pointer should only be used for reading values. Do not use it to write any values.

Requirements

Import Library: Use D3d8.h.

Import Library: Use d3d8.lib.
