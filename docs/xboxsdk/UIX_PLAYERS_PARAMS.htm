UIX_PLAYERS_PARAMS

Initialization parameters for the players feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_PLAYERS_FEATURE.

typedef struct _UIX_PLAYERS_PARAMS{
    SIZE_T StructSize;
    DWORD UserPort;
    ITitlePlayersListItem *pPlayerControllingScreen;
    CONST XUID *pSelectedPlayerXUID;
    BOOL SortCurrentPlayersList;
    DWORD FilterFlags;
    DWORD DisplayType;
} UIX_PLAYERS_PARAMS, *PUIX_PLAYERS_PARAMS;

Members

StructSize
Size of the structure. Must be set to the structure size before calling ILiveEngine::StartFeature.
UserPort
The controller port index (zero-based) of the user for which the players list will be displayed. Specify UIX_INVALID_VALUE to use lobby mode.
pPlayerControllingScreen
The player that is given control of the players screen (this is usually the player on the port specified in UserPort). The specified player will be displayed as the first player in the list (unless SortCurrentPlayersList is TRUE), and will not be selectable. Specify NULL to designate no such player.
pSelectedPlayerXUID
Pointer to an XUID of the user to initially select in the players list. If a user with the specified XUID is in the list, that user will be highlighted when the list is created. Specify NULL to select the first player.
SortCurrentPlayersList
Specify TRUE to sort the players list (based on the ITitlePlayersListItem::Compare method), FALSE to perform no sorting.
FilterFlags
Initial filter flags to determine which players are displayed in the list. Specify zero to show all players. Can be changed later by calling ILivePlayersList::SetFilterFlags.
DisplayType

The type of players list to display. One or more of the following flags must be specified:
Value Description
UIX_PLAYERS_DISPLAY_CURRENT_PLAYERS Display current players.
UIX_PLAYERS_DISPLAY_DEPARTED_PLAYERS Display departed players.

The following additional flags may optionally be specified:

Value Description
UIX_PLAYERS_DISPLAY_LOBBY_MODE Use lobby mode for the players list (automatically selected if UserPort is UIX_INVALID_VALUE).

Remarks

This structure must be initialized to zero and StructSize set to the size of the structure before calling ILiveEngine::StartFeature.
