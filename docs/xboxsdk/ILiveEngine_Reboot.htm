ILiveEngine::Reboot

Reboots the Xbox console.

HRESULT Reboot(
    DWORD Context
);

Parameters

Context
[in] Title-defined context value that will be returned to the title when the console reboots back to the title.

Return Values

If the function succeeds, the console reboots and there is no return value. Otherwise, the function returns an error value.

Remarks

Call Reboot when DoWork indicates that a reboot is necessary. Reboot will fail and return an error code if the engine does not need to reboot. Reboot will boot to the appropriate location based on the event that necessitated a reboot. For example, if the user needs to perform account management, Reboot will boot to the account management screen in the Xbox Dashboard.

When the console reboots to the title after performing whatever operation necessitated the initial reboot, the title can retrieve the Context value by calling XGetLaunchInfo and examining the dwContext member of the returned launch information structure.

Requirements

Import Library: Use Uix.lib.
