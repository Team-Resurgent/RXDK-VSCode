IXHVEngine::VoiceMailPlay

Starts playback of the voice message for the local talker.

HRESULT VoiceMailPlay(
    DWORD dwLocalPort,
    DWORD dwInputBufferSize,
    const BYTE *pbInputBuffer,
    BOOL bOutputToSpeakers
);

Parameters

dwLocalPort
[in] Controller port of the local talker.
dwInputBufferSize
[in] Size, in bytes, of the buffer containing the voice message data.
pbInputBuffer
[in] Pointer to the buffer containing the voice message data.
bOutputToSpeakers
[in] Flag indicating the game should use the front speaker instead of the default target read from the DSP image.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method allocates memory that is freed upon receiving either a ITitleXHV::VoiceMailStopped or a ITitleXHV::VoiceMailDataReady callback.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
