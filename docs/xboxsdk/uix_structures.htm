Drop-in UI Structures

The following structures are used with Xbox Drop-in UI:

Structure Description
UIX_EXIT_INFO

Feature exit information obtained with the ILiveEngine::GetExitInfo method.
UIX_FRIENDS_PARAMS

Initialization parameters for friends feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_FRIENDS_FEATURE.
UIX_LOGON_EXIT_DATA

When logon is successful, this structure contains a list of controllers that were used to log on.
UIX_LOGON_PARAMS

Logon parameters for the logon feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_LOGON_FEATURE.
UIX_PLAYERS_PARAMS

Initialization parameters for the players feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_PLAYERS_FEATURE.
UIX_UITEST_PARAMS

Initialization parameters for the UI test feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_UITEST_FEATURE. Available only in debug builds.
