DmSetTitle

Tells the system software which title to launch after the console is rebooted with the DMBOOT_WAIT flag.

DMHRAPI DmSetTitle(
  LPCSTR szDir, 
  LPCSTR szTitle, 
  LPCSTR szCmdLine
  );

Parameters

szDir
Path that contains the title.
szTitle
File name of the title. The file name must be relative to the szDir parameter. (No absolute paths.)
szCmdLine
Command line to pass to the title. See XGetLaunchInfo for more information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

This function is meaningful only if the console was rebooted with the DMBOOT_WAIT flag. When the console is rebooted with DMBOOT_WAIT, it will wait 15 seconds after rebooting before launching a title. During this 15-second period (and only during this period), calling DmSetTitle will set the title to launch.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
