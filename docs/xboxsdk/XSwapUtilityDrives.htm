XSwapUtilityDrives

Swaps the partitions assigned to the N: and Z: drives. This allows a game to decide which partition's data will be preserved. The partition that is last assigned to the Z: drive will be preserved.

BOOL XSwapUtilityDrives();

Parameters

None.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

When successfully executed, this function will remap the partitions in use to the opposite drive letter. In other words, the partition originally mapped to N: will become Z: and vice versa. This also implicitly means that the partition that becomes Z: will be the one preserved by the system and will be potentially available for re-use in the future (assuming no other title is assigned that partition in the meantime).

If either partition is not mounted when the function is called, the system will halt.

Requirements

Header: Declared in Xbox.h.

Import Library: Use Xapilib.lib.
