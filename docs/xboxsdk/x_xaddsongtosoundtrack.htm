XAddSongToSoundtrack

The XAddSongToSoundtrack function adds a new Microsoft® Windows Media® Audio (WMA) song to a soundtrack.

HRESULT WINAPI XAddSongToSoundtrack(
  UINT dwSoundtrackId,
  LPCSTR pszSongPath,
  LPCWSTR pszSongName,
  LP_SOUNDTRACK_PROGRESS_ROUTINE lpRoutine,
  LPVOID Context,
  PUINT pdwSongId
);

Parameters

dwSoundtrackId
[in] Soundtrack identifier of the soundtrack that the song will be added to. This value can be obtained by calling the XAddSoundtrack function.
pszSongPath
[in] Pointer to a null-terminated string that specifies the full directory path of the song to be added.
pszSongName
[in] [optional] Pointer to a null-terminated string that specifies the name of the song. If no name is specified, the WMA file title is used.
lpRoutine
[in] [optional] Pointer to the starting address of the function created for reporting progress as the song is added. See the LP_SOUNDTRACK_PROGRESS_ROUTINE section for more information on this callback function.
Context
[in] [optional] A user-defined value to be passed to the progress callback routine.
pdwSongId
[out] [optional] Song identifier of the song that has been added. This is the value used when calling XOpenSoundtrackSong, for example.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The XAddSongToSoundtrack function allows applications to add WMA songs to soundtracks on the user's hard drive. Songs (background music) are then available for playback by other applications, or directly through the Xbox Dashboard.

Files added to a soundtrack must be in 44-kHz, 128-kbit WMA format.

Requirements

Header: Declared in xbox.h; for include use xtl.h.
Import Library: Use xsndtrk.lib.
