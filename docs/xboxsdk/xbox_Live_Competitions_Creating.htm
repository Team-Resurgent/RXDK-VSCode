Creating a Competition Instance

With a competition configuration on Xbox Live, you (or your users) can create a competition instance based on the template in the configuration. This can be done via the client (your title) by calling the XOnlineCompetitionCreateSingleElimination function.

Note    Competition instances can also be instantiated via the Web.

The XOnlineCompetitionCreateSingleElimination function begins the asynchronous task of instantiating a competition. XOnlineCompetitionCreateSingleElimination gives your title a handle that is signaled when there is work to be performed on the task. To get work done on the task, you use a technique called "pumping the task handle." Specifically, your title calls XOnlineTaskContinue to let the Xbox Live service know that the client is still connected and waiting for results, and to see if results are ready. To obtain the results of this task, your title must call the XOnlineCompetitionCreateGetResults function. After it obtains the results, your title calls XOnlineTaskClose.

Note    Before players can instantiate or participate in a competition, your game must log them in to Xbox Live with the XOnlineLogon function.

When you create a competition instance, you must specify certain attributes. These include the registration start time, the registration end time, the competition start time, the time at which reminders are sent, the round intervals, and whether this is a team competition.

You can create competitions with either public or private slots. Typically, a title uses private slots for those players who are explicitly invited to join. Public slots are accessible to anyone.

By default, competitions are removed from the server seven days after they are completed or canceled. However, to change this setting, your title must create the competition using the XOnlineCompetitionCreate function rather than XOnlineCompetitionCreateSingleElimination. Pass the new time limit (in days) as the value for the XONLINE_COMP_ATTR_COMP_CLEANUP attribute in the pCompetitionAttributes parameter. When the competition is removed from the server, its associcated leaderboards (also called scoreboards) are removed as well.
