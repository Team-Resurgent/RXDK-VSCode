XOnlineGameInviteRevoke

Revokes a pending game invitation issued to a specified list of players.

HRESULT XOnlineGameInviteRevoke(
    DWORD dwUserIndex,
    DWORD dwPeerCount,
    const XUID *pxuidPeersToRevoke,
    XNKID SessionID,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player revoking the game invitation. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwPeerCount
[in] Number of peers from which to revoke the invitation. This is the number of XUID structures represented by the pxuidPeersToRevoke argument.
pxuidPeersToRevoke
[in] Pointer to an array of XUID structures representing the peers from which to revoke the invitation.
SessionID
[in] An XNKID structure representing the session.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on this task. This parameter can be NULL if no such event is required.
phTask
[out] Address of an XONLINETASK_HANDLE that must be pumped to completion by XOnlineTaskContinue. Call XOnlineTaskClose to close the task.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function is similar to the XOnlineFriendsRevokeGameInvite function, with the exception that the sender can be a teammate or fellow peer in a competition.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
