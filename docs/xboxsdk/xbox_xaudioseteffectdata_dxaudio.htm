XAudioSetEffectData

Sets the effects parameters effects such as IIR2, Distortion, and I3DL2.

HRESULT XAudioSetEffectData(
    DWORD dwEffectIndex,
    LPCDSFX_HIGH_LEVEL_EFFECT_DESCRIPTION pDesc,
    LPDSFX_RAW_EFFECT_DESCRIPTION pRawDesc
);

Parameters

dwEffectIndex
[in] Index of effect to set.
pDesc
[in] A pointer to a DSFX_HIGH_LEVEL_EFFECT_DESCRIPTION structure indicating the high level parameters to set.
pRawDesc
[out] A pointer to a DSFX_RAW_EFFECT_DESCRIPTION structure indicating the raw parameters to be returned. This parameter is optional and can be NULL.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This API offers the developer a high level interface for setting the effects parameters for effects such as IIR2, Distortion, and I3DL2. The user supplies high level parameters for the effect with are translated internally into the raw representation required by the DSP.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
