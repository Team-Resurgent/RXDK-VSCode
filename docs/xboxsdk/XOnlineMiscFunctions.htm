Miscellaneous Functions

The following Xbox Live functions do not fit into any particular category.

Function Description
XOnlineCleanup Performs required cleanup for the Xbox Live functions.
XOnlineFeedbackSend Sends feedback about a user of the Xbox Live service.
XOnlineGetNatType Returns a value that describes the "strictness" of the network address translation (NAT) device between the Xbox console and the Xbox Live service.
XOnlineGetServiceInfo Indicates whether a specified service has been successfully logged on, and optionally retrieves additional information about the service.
XOnlineGetSession Returns the session information for the current title.
XOnlineGetUsers Retrieves information about the online users stored on the Xbox hard disk and any attached Xbox MUs.
XOnlineGetUserSession Retrieves the session information for a user.
XOnlineStartup Performs initialization and startup for Xbox Live functions.
XOnlineStringLookup Retrieves Unicode strings corresponding to an array of IDs for a given language.
XOnlineStringLookupEx Retrieves Unicode strings corresponding to an array of IDs for a given language.
XOnlineStringLookupGetResults Retrieves the results from a call to XOnlineStringLookup.
XOnlineStringVerify Checks text strings to ensure that they do not contain objectionable content. Use this function to verify the strings that a user provides before uploading or storing them where other users can retrieve and display them.
XOnlineStringVerifyGetResults Retrieves the results from a call to XOnlineStringVerify.
XOnlineThrottleGet Retrieves the current throttle settings for the given throttle type.
XOnlineThrottleSet Sets the current throttle settings for the given throttle type.
XOnlineTitleUpdate Begins the download and installation of an available update for the game title.
XOnlineTitleUpdateEx Begins the download and installation of an available update for the game title, and allows passing launch data across the update.
XOnlineVerifyNickname Creates an asynchronous task to verify that the specified nickname is valid.
