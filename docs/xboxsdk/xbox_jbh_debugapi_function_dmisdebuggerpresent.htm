DmIsDebuggerPresent (PC)

Allows code on the development PC to determine whether there is a debugger connected.

DMHRAPI DmIsDebuggerPresent(void);

Return Values

Returns XBDM_NOERR if there is a debugger connected, or an error value if not.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XbDm.lib
