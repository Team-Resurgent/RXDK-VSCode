Xbox Tools

This section provides information about the tools included in this version of the Xbox Development Kit (XDK) for the Xbox® video game system from Microsoft. The tools described here are subject to change in later versions of the XDK.

You can use the Xbox tools as soon as you receive your XDK. Tool executables that have a graphical user interface are available from the Start menu on your development system. Command-line tool executables are included in your system path after you have installed the XDK. For details on installing and configuring your development system and Xbox console, see the XDK Release Notes (RelNotes.htm in the root of the Xbox SDK disc).

The tools in this section are divided into the following categories:

  • Graphics
  • Audio
  • (Remote) Target System
  • Xbox Live and Networking
  • Build, Submission, and Certification
  • Debugging
  • Performance
  • Visual C++ 7 Compiler
