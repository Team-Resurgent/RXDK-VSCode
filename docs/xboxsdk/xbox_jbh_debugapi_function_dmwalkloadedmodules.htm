DmWalkLoadedModules

Provides a way to examine information about each loaded module. Call DmWalkLoadedModules repeatedly to iterate over the list of loaded modules.

DMHRAPI DmWalkLoadedModules(
  PDM_WALK_MODULES *ppWalkMod, 
  PDMN_MODLOAD     pModLoad
  );

Parameters

ppWalkMod
[in, out] Pointer to a variable that maintains a pointer to a internal structure used by DmWalkLoadedModules. Set *ppWalkMod to NULL on the first call to DmWalkLoadedModules; the function allocates memory to store a list of loaded modules, and sets the variable pointed to by ppWalkMod to point to that memory.
pModLoad
[out] Pointer to a DMN_MODLOAD structure that receives information about the next loaded module.

Return Values

Returns XBDM_NOERR if the call is successful and the end of the list has not been reached. XBDM_ENDOFLIST is returned when the walk is complete. Otherwise, an error value is returned.

Remarks

If the return is not XBDM_NOERR or XBDM_ENDOFLIST, then an error has occurred and the walk should be discontinued. After using DmWalkLoadedModules, you must free the memory allocated to store the list of loaded modules by calling DmCloseLoadedModules, passing it the value stored in *ppWalkMod.

The following example illustrates how to use DmWalkLoadedModules and DmCloseLoadedModules to iterate over the list of loaded modules:

DMHRAPI error;
PDM_WALK_MODULES pWalkMod = NULL;
DMN_MODLOAD modLoad;

while( XDBM_NOERR == (error = DmWalkLoadedModules(&pWalkMod, &modLoad)) ) {
    // Examine the contents of modLoad.
}
if (error != XBDM_ENDOFLIST)
{
    // Handle error.
}
DmCloseLoadedModules(pWalkMod);
