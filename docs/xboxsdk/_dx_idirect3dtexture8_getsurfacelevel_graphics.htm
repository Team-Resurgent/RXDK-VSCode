IDirect3DTexture8::GetSurfaceLevel

Retrieves a surface for a texture level.

HRESULT GetSurfaceLevel(
  UINT Level,
  IDirect3DSurface8** ppSurfaceLevel
);

Parameters

Level
[in] The level of the texture.
ppSurfaceLevel
[out, retval] Address of a pointer to an IDirect3DSurface8 interface, representing the returned surface.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value is E_OUTOFMEMORY.

Note  Calling this method will increase the internal reference count on the IDirect3DSurface8 interface. Failure to call IDirect3DTexture8::Release when finished using this IDirect3DSurface8 interface results in a memory leak.

Requirements
