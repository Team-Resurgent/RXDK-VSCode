XOnlineQuerySearch

Searches a competition's dataset for entities matching a set of attributes.

HRESULT XOnlineQuerySearch(
    DWORD dwDatasetId,
    DWORD dwProcIndex,
    DWORD dwPage,
    DWORD dwResultsPerPage,
    DWORD dwNumResultSpecs,
    const XONLINE_ATTRIBUTE_SPEC *pAttributeSpecs,
    DWORD dwNumAttributes,
    const XONLINE_ATTRIBUTE *pAttributes,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE *phTask
);

Parameters

dwDatasetId
[in] ID of the dataset to query.
dwProcIndex
[in] Index number of a search procedure.
dwPage
[in] Current page number of the result list. Page numbers start with 0.
dwResultsPerPage
[in] Number of results per page. Maximum is 255.
dwNumResultSpecs
[in] Number of competition attributes in the array specified by the pAttributeSpecs parameter.
pAttributeSpecs
[in] Array of XONLINE_ATTRIBUTE_SPEC structures that specify the type and size, in bytes, of each result attribute. The first structure in this array must have the X_ATTRIBUTE_DATATYPE_ENTITY_ID as the value of its dwType member. Each subsequent structure must match the order and type of return value from the query.
dwNumAttributes
[in] Number of competition attributes in the array specified by the pAttributes parameter.
pAttributes
[in] Pointer to an array of XONLINE_ATTRIBUTE structures, each specifying the parameter ID and value for the Search to be performed. These attributes must be of the same type and in the same order as configured in XLAST for the procedure specified in dwProcIndex.
hWorkEvent
[in, optional] Handle to an event you want to associate with this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by XOnlineQuerySearch. Use this handle in calls to XOnlineTaskContinue to perform work associated with this task. When the task is complete, call XOnlineCompetitionSearchGetResults to get the search results. Call XOnlineTaskClose to close the task.

Return Values

Returns S_OK if successful, an error code otherwise. The error codes can include those listed in the following table.

Error Code Description
XONLINE_E_QUERY_UNEXPECTED_ATTRIBUTE The list of attributes specified in the pSearchAttributes parameter does not match the list of attributes on the server, which were configured with XLAST.
XONLINE_E_QUERY_SPEC_COUNT_MISMATCH The list of attributes specified in the pResultAttributeSpecs parameter does not match the list of attributes on the server, which were configured with XLAST.
XONLINE_E_QUERY_ATTRIBUTE_TOO_LONG A string or BLOB (binary block of data) type attribute listed in pSearchAttributes or pResultAttributeSpecs specifies a length that is shorter than the data stored on the server.
XONLINE_E_QUERY_DATASET_NOT_FOUND The dataset specified in the dwDatasetID parameter was not found.
XONLINE_E_QUERY_PROCEDURE_NOT_FOUND The query specified in the dwQueryID parameter was not found.
