DmGetNameOfXbox

Retrieves the name of the currently connected Xbox.

DMHRAPI DmGetNameOfXbox(
  LPSTR szName, 
  LPDWORD lpdwSize,
  BOOL fResolvable
  );

Parameters

szName
Pointer to a buffer that receives the Xbox name.
lpdwSize
Pointer to a variable of type DWORD that specifies the size of the szName buffer. Upon return, the variable is set to the actual number of characters (excluding the terminating null character) returned in szName.
fResolvable
Specifies that the name returned should be verified as actually resolving to the currently connected Xbox console.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib.
