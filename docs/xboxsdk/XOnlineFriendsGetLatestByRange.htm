XOnlineFriendsGetLatestByRange

Retrieves a subset of the friends list. The title describes the subset by specifying a starting offset and the number of entries to retrieve.

HRESULT XOnlineFriendsGetLatestByRange(
    DWORD dwUserIndex,
    DWORD dwRangeStart,
    DWORD *pdwFriendBuffer,
    PXONLINE_FRIEND pFriendBuffer,
    DWORD *pdwFriendsBefore,
    DWORD *pdwFriendsAfter
);

Parameters

dwUserIndex
[in] Controller index of the current player. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwRangeStart
[in] Offset from the top of the friends list at which to start the range.
pdwFriendBuffer
[in, out] On input, specifies the number of XONLINE_FRIEND structures in the buffer pointed to by the pFriendBuffer parameter. On output, receives the number of XONLINE_FRIEND structures filled with data that were copied to this buffer.
pFriendBuffer
[out] Pointer to a buffer that receives an array of XONLINE_FRIEND structures for members of the friends list.
pdwFriendsBefore
[out] Receives the number of entries that occur in the friends list before the entries returned by this function.
pdwFriendsAfter
[out] Receives the number of entries that occur in the friends list after the entries returned by this function.

Return Values

XOnlineFriendsGetLatestByRange always succeeds with a return value of S_OK.

Remarks

You must call the XOnlineFriendsStartup function before calling XOnlineFriendsGetLatestByRange.

The XOnlineFriendsGetLatestByRange function attempts to fill in the entire buffer provided by the title. If the number of entries from the start offset to the end of the friends list isn't sufficient to fill the buffer, the function automatically moves the starting offset upward so as to fill the buffer as much as possible. For this reason, the title should always check the values returned in the pdwFriendsBefore and pdwFriendsAfter parameters.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
