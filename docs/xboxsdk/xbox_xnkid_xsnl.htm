XNKID

Contains the session ID for an online-game session.

typedef struct {
    BYTE ab[8];
} XNKID;

Members

ab
Array of bytes that contains the key identifier.

Remarks

The information on this structure is provided for reference only, so that titles will be aware of the size of the data being used.

Requirements

  Header: Declared in Winsockx.h, for include use Xtl.h.
