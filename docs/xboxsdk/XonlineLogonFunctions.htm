Logon Functions

The Xbox Live logon functions include the following.

Function Description
XGetAutoLogonFlag Retrieves the autologon flags.
XOnlineChangeLogonUsers Creates an asynchronous task to change the set of local players logged onto Xbox Live.
XOnlineChangeLogonUsersTaskGetResults Returns the results of a completed (or failed) XOnlineChangeLogonUsers call.
XOnlineGetLogonUsers Retrieves the status of each user participating in a logon session, including the success or failure of player logons performed by XOnlineLogon.
XOnlineLogon Creates an asynchronous task to log the specified players on to Xbox Live and to acquire access to the specified online services.
XOnlineLogonTaskGetResults Returns the result of a completed XOnlineLogon or XOnlineSilentLogon call.
XOnlineRetrieveLogonState Retrieves logon state information from an XONLINE_LOGON_STATE structure obtained with a previous call to XOnlineSaveLogonState. This information can be used to perform a logon with the same players and services that were logged on at the time XOnlineSaveLogonState was called.
XOnlineSaveLogonState Retrieves the current state of an Xbox Live logon, including the logged on users and services. The returned data can be saved, then later used in a call to XOnlineRetrieveLogonState to retrieve information about the logon state.
XOnlineSilentLogon Enables titles to automatically sign into the Xbox Live service using the most recently signed-in account.
