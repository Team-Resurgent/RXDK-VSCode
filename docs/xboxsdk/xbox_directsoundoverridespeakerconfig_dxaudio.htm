DirectSoundOverrideSpeakerConfig

Overrides the current speaker configuration.

void DirectSoundOverrideSpeakerConfig(
    DWORD dwSpeakerConfig
);

Parameters

dwSpeakerConfig

Value that specifies the speaker configuration to use. Can be one of the following values:
Value Description
DSSPEAKER_MONO Set the speaker configuration to mono.
DSSPEAKER_STEREO Set the speaker configuration to stereo.
DSSPEAKER_SURROUND Set the speaker configuration to surround mode.
DSSPEAKER_USE_DEFAULT Set the speaker configuration back to the settings in the Xbox Dashboard.

Return Values

None.

Remarks

Before the DirectSound object is created, DirectSoundOverrideSpeakerConfig can be used to change the default speaker configuration. Titles should check the user's current audio configuration using XGetAudioFlags when setting a speaker configuration that may not be supported by the user's audio equipment (for example, if the user only has mono support and the title wants to set the speaker configuration to stereo).

Calling DirectSoundOverrideSpeakerConfig will turn off Dolby Digital encoding, if it is enabled in the Xbox Dashboard. This function will only turn off audio features in the Xbox Dashboard, and will not enable features that are disabled by the user. The features are disabled until the console is rebooted, at which point they revert to the user's settings.
