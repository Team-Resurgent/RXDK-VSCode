XHV Structures

This section contains the structures associated with XHV.

Structure Description
XHV_CODEC_HEADER Exposes internal XHV statistics.
XHV_LOCAL_TALKER_STATUS Describes the current status of a local talker. This structure is returned by IXHVEngine::GetLocalTalkerStatus.
XHV_RUNTIME_PARAMS Contains initialization parameters for the XHV engine.
XHV_SR_ITEM Contains a word ID and the confidence level of the recognized word.
XHV_VOICE_COMMUNICATOR_STATUS Indicates the status of the voice communicator.
XHV_VOICE_MASK Contains input voice mask parameters for a specified local talker.
