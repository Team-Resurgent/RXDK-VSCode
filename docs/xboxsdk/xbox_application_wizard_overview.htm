Xbox Application Wizard Overview

The Xbox Application Wizard allows you to create two types of Xbox projects, either an Xbox game title or an Xbox static library. By default, the Xbox Application Wizard creates a non-empty Xbox game title. If you wish to alter these settings, click the Application Settings tab in the left column of the wizard and make the desired changes.

You can view the files the wizard creates for your project in Solution Explorer. For more information about the files the wizard creates for your project, see the project-generated file ReadMe.txt.
