DSSTREAMDESC

Describes a sound stream.

typedef struct _DSSTREAMDESC{
    DWORD dwFlags;
    DWORD dwMaxAttachedPackets;
    LPWAVEFORMATEX lpwfxFormat;
    LPFNXMEDIAOBJECTCALLBACK lpfnCallback;
    LPVOID lpvContext;
    LPCDSMIXBINS lpMixBins;
} DSSTREAMDESC, *LPDSSTREAMDESC;

Members

dwFlags
Value indicating stream attributes. Can be the following:
Value Description
DSSTREAMCAPS_CTRL3D The stream will be a 3D HRTF stream. Can only be set for streams with a mono format.
DSSTREAMCAPS_MUTE3DATMAXDISTANCE The stream is muted and no rolloff calculations are performed if the distance from the listener meets or exceeds the stream's MaxDistance value.
DSSTREAMCAPS_LOCDEFER The stream will not allocate a hardware voice until it is played.
DSSTREAMCAPS_NOCOALESCE The stream will not coalesce data packets when writing to the hardware. Normally, if multiple packets have been queued for playback, DirectSound will attempt to coalesce them into one packet before mapping them into the hardware. Because of this, all coalesced packets will complete at exactly the same time. Therefore, if more accurate completion notifications are desired, create the stream using this flag.
DSSTREAMCAPS_ACCURATENOTIFY This flag is similar to the DSSTREAMCAPS_NOCOALESCE flag except that it also completes packets from the stream's interrupt deferred procedure call (DPC) instead of from DoWork.
dwMaxAttachedPackets
Maximum count of packets that will be simultaneously submitted to the stream.
lpwfxFormat
Address of a WAVEFORMATEX structure that describes the stream format.
lpfnCallback
Address of the packet completion callback routine.
lpvContext
Address of a buffer containing the packet completion callback context.
lpMixBins
Address of a DSMIXBINS structure that contains the output mixbin identifiers and volume pairs. Can be set to NULL, which specifies that the default mixbin volume pairs are used. (See Mixbin Volume Pair Combinations for more information about the default mixbin volume pairs.)
