WAVEBANKDATA

Contains the data from an XACT wave bank.

typedef struct _WAVEBANKDATA{
    DWORD dwFlags;
    DWORD dwEntryCount;
    CHAR szBankName[WAVEBANK_BANKNAME_LENGTH];
    DWORD dwEntryMetaDataElementSize;
    DWORD dwEntryNameElementSize;
    DWORD dwAlignment;
    WAVEBANKMINIWAVEFORMAT CompactFormat;
} WAVEBANKDATA, *LPWAVEBANKDATA;

typedef const WAVEBANKDATA *LPCWAVEBANKDATA;

Members

dwFlags
Wave bank type and data flags. dwFlags specifies one of the following wave bank types.
Value Description
WAVEBANK_TYPE_BUFFER Indicates that the data for this wave bank uses an in-memory buffer. When this flag is specified, the WAVEBANK_TYPE_STREAMING flag may not be specified.
WAVEBANK_TYPE_STREAMING Indicates that the data for this wave bank uses a streaming buffer. When this flag is specified, the WAVEBANK_TYPE_BUFFER flag may not be specified.

Additionally, dwFlags may contain a combination of the following wave bank flags.

Value Description
WAVEBANK_FLAGS_ENTRYNAMES Indicates that the wave bank includes entry names. The names will be an array of strings found in the WAVEBANK_SEGIDX_ENTRYNAMES section of the wavebank.
WAVEBANK_FLAGS_COMPACT Indicates that the wave bank is a compact streaming bank.

Use the WAVEBANK_TYPE_MASK mask to isolate the wave bank type, and the WAVEBANK_FLAGS_MASK mask to isolate the wave bank flags.

dwEntryCount
Number of elements found in the WAVEBANK_SEGIDX_ENTRYMETADATA section (and if names are included, the number of elements in the WAVEBANK_SEGIDX_ENTRYNAMES section).
szBankName
Name of the wave bank.
dwEntryMetaDataElementSize
Size of each entry metadata element in bytes.
dwEntryNameElementSize
Size of each entry name element in bytes.
dwAlignment
Entry alignment in bytes. Can be one of the following values:
Value Description
WAVEBANK_ALIGNMENT_MIN Indicates alignment for in-memory wave banks.
WAVEBANK_ALIGNMENT_HARDDRIVE Indicates alignment for hard disk streamed wave banks.
WAVEBANK_ALIGNMENT_DVD Indicates alignment for DVD streamed wave banks.
CompactFormat
A WAVEBANKMINIWAVEFORMAT structure containing the mini-wave format for the wave bank. This member is valid only if dwFlags contains WAVEBANK_FLAGS_COMPACT.

Requirements
