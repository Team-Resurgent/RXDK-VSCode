IDirect3DDevice8::GetVertexBlendModelView

Gets the last model-view transform.

HRESULT GetVertexBlendModelView(
  D3DMATRIX *pModelView,
  D3DMATRIX *pProjectionViewport
);

Parameters

pModelView
[out] Pointer to a D3DMATRIX structure containing the last model-view transform, as specified by IDirect3DDevice8::SetVertexBlendModelView.
pProjectionViewport
[out] Pointer to a D3DMATRIX structure containing the last projection/viewport matrix transform, as specified by IDirect3DDevice8::SetVertexBlendModelView.

Return Values

If IDirect3DDevice8::SetVertexBlendModelView currently has control of the GPU's transform registers, the return value is S_OK, otherwise, the return value is S_FALSE.
