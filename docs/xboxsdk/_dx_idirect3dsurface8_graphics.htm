IDirect3DSurface8

Applications use the methods of the IDirect3DSurface8 interface to query and prepare surfaces. IDirect3DSurface8 inherits from IDirect3DResource8.

The IDirect3DSurface8 interface inherits the following IDirect3DResource8 methods, which can be organized into these groups.

Devices GetDevice
Information GetType
Private Surface Data FreePrivateData
Miscellaneous Register
  BlockUntilNotBusy
  IsBusy
  MoveResourceMemory

  AddRef
  Release

The methods of the IDirect3DSurface8 interface can be organized into the following groups.

Information GetContainer
  GetDesc
Locking Surfaces LockRect
  UnlockRect

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
