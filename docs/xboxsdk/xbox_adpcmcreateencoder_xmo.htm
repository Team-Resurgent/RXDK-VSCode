AdpcmCreateEncoder

Encodes linear PCM data into ADPCM format.

HRESULT AdpcmCreateEncoder(
    LPCWAVEFORMATEX pwfxDecoded,
    DWORD dwBytesPerDecodedBlock,
    LPIMAADPCMWAVEFORMAT pwfxEncoded,
    LPXMEDIAOBJECT *ppMediaObject
);

Parameters

pwfxDecoded
Address of a WAVEFORMATEX struture that describes the input format.
dwBytesPerDecodedBlock
DWORD value that specifies the number of bytes in each block.
pwfxEncoded
Address of the ADPCM format that describes the output.
ppMediaObject
Address of a pointer to the Xbox media object that is returned.

Return Values

Returns DS_OK if successful, a Xbox media object specific error code otherwise.
