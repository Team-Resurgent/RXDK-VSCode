XAc97MediaObject::SetMode

Sets the channel mode.

HRESULT SetMode(
    DWORD dwMode,
);

Parameters

dwMode
Value that specifies the digital channel mode. Can be one of the following values:
DSAC97_MODE_PCM
DSAC97_MODE_ENCODED

Return Values

Returns S_OK if successful, an error code otherwise.
