WMAXMOFileHeader

Contains the header information from a Microsoft® Windows Media® Audio (WMA) file.

typedef struct _WMAXMOFileHeader{
    DWORD dwVersion;
    DWORD dwSampleRate;
    DWORD dwNumChannels;
    DWORD dwDuration;
    DWORD dwBitrate;
} WMAXMOFileHeader;

Members

dwVersion
Value that contains the version of the codec.
dwSampleRate
Value that contains the sampling rate in Hertz.
dwNumChannels
Value that contains the number of audio channels.
dwDuration
Value that contains the duration of the file in milliseconds.
dwBitrate
Value that contains the bit-rate of the WMA bitstream.

Requirements

  Header: Declared in DSound.h.
