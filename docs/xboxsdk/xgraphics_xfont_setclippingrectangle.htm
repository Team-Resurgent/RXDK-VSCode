XFONT::SetClippingRectangle

Sets the clipping rectangle used for the text cell.

void __stdcall SetClippingRectangle(
  D3DRECT *pRectangle
);

Parameters

pRectangle
[in] Pointer to a D3DRECT structure that describes the clipping rectangle. pRectangle should be NULL to disable the clipping.

Return Values

None.

Requirements

  Header: Declared in xfont.h
  Import Library: Use xgraphics.lib
