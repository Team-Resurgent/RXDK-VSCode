DmGetMemory

Copies memory from the Xbox console to the local PC.

DMHRAPI DmGetMemory(
  LPCVOID  lpbAddr, 
  DWORD   cb, 
  LPVOID lpbBuf, 
  LPDWORD pcbRet
  );

Parameters

lpbAddr
Address on the Xbox console of the first byte of memory to be copied.
cb
Number of bytes to be copied.
lpbBuf
Address on the local PC of the buffer that will receive the data.
pcbRet
Pointer to a DWORD that will receive the number of bytes actually copied. This value can be NULL. If pcbRet is NULL and the number of bytes requested does not equal the number of bytes received, DmGetMemory will return an error.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise. It is possible to for DmGetMemory to return no error and the DWORD pointed to by pcbRet will hold a value of zero (if the memory pointed to by lpbAddress is not accessible).

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
