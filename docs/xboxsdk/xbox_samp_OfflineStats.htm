OfflineStats Sample

The OfflineStats sample demonstrates the Offline stats API for feeding data to Stats.xbe. This lets offline titles implement basic Xbbox Live statistics with just a few lines of code.

Note    Offline stats and Stats.xbe are no longer supported and should not be used in games.

Overview

For this sample to function perfectly, it needs to be renamed to Default.xbe so that Stats.xbe can launch it when done.

Copy the media tree to the target machine before running this sample. Be sure to rename the OfflineStats.xbe to Default.xbe so that it will be launched when the Xbox Stats is exited. Also, Stats.xbe, from XDK\redist\stats, must be stamped with the title ID of the sample: (xbecert /TESTID:0xFFED9B77 Stats.xbe) and copied into the same location as the Offline Stats sample.

You also need to copy from XDK\redist\stats\media to the media directory the following files:

  • ArialUni.Ttf
  • stats.inx
  • stats.xpr
  • stats.xsb
  • stats.xwb

These files can be modified to change the look and sound of Stats.xbe.

The Leaderboards.inx file is the only file that should not be copied from this directory.
