IDirect3DCubeTexture8::UnlockRect

This function does nothing. It has been provided to ease in the porting of code from DirectX 8.0 to DirectX for the Xbox.

HRESULT UnlockRect(
  D3DCUBEMAP_FACES FaceType,
  UINT Level
);

Parameters

FaceType
[in] Member of the D3DCUBEMAP_FACES enumerated type, identifying a cube map face.
Level
[in] Specifies a level of a mipmapped cube texture.

Return Values

The method returns D3D_OK.
