XOnlineTeamSetProperties

Sets the properties for a team.

HRESULT XOnlineTeamSetProperties(
    DWORD dwUserIndex,
    XUID xuidTeam,
    const XONLINE_TEAM_PROPERTIES *pTeamProperties,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player setting the team properties.
xuidTeam
[in] XUID that uniquely identifies the team.
pTeamProperties
[in] Pointer to an XONLINE_TEAM_PROPERTIES structure containing the properties of the team.
hWorkEvent
[in, optional] Handle to an event.
phTask
[out] Pointer to an XONLINETASK_HANDLE returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The strings in the XONLINE_TEAM_PROPERTIES structure are automatically checked to ensure they do not contain offensive text. Titles do not need to call XOnlineStringVerify before calling XOnlineTeamSetProperties. If a string does have inappropriate content, the error XONLINE_E_TEAMS_NAME_CONTAINS_BAD_WORDS, XONLINE_E_TEAMS_DESCRIPTION_CONTAINS_BAD_WORDS, XONLINE_E_TEAMS_MOTTO_CONTAINS_BAD_WORDS, or XONLINE_E_TEAMS_URL_CONTAINS_BAD_WORDS will be returned from XOnlineTaskContinue, depending on the offending string.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
