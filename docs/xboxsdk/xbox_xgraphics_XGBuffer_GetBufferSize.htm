XGBuffer_GetBufferSize

Retrieves the size of a buffer.

DWORD XGBuffer_GetBufferSize(
  XGBuffer* pThis
);

Parameters

pThis
[in, out] A pointer to a valid XGBuffer structure.

Return Values

If this method succeeds, it returns a pointer to a DWORD that contains the size of the buffer in BYTES.
