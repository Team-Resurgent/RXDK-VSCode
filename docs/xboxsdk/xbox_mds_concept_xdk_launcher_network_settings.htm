Network Settings

The network settings allow the developer to configure the network information on the target system for connecting to the local network. These settings can be adjusted by selecting Network Settings in the XDK Launcher Settings and Tools menu.

When changing network settings, or unplugging and replugging in network cables, the network connection is automatically re-acquired.

Illustration  An image of the Network Settings screen.

The following items are listed in the Network Settings menu:

  • Game Configuration
  • Debug Configuration
  • Test Connection to PartnerNet

Game Configuration

This section provides a brief overview of the commands available on the Game Configuration menu.

IP Settings

  • Configure IP — Determines whether the IP should be configured automatically or manually. By default, the IP configuration is determined automatically.

    Note    If the IP configuration is set automatically, the IP Address, Subnet Mask, and Gateway options are disabled. Use the A button on the controller to toggle between automatic and manual settings.

  • IP Address — Provides a keypad for entering the appropriate static IP address of the Xbox.
  • Subnet Mask — Provides a keypad for entering the appropriate subnet mask for the Xbox.
  • Gateway — Provides a keypad for entering the appropriate gateway for the Xbox.
  • MAC Address — Specifies the media access control (MAC) address of the Xbox. This is a factory-assigned value that can be overridden in the Advanced Settings menu (see below).

    Note    This refers to the game MAC address for the Xbox. The debug MAC address can be set in the Debug Configuration menu below.

DNS Servers

  • Configure DNS — Determines whether the domain name servers (DNS) should be configured automatically or manually. By default, the DNS configuration is determined automatically.

    Note    If the DNS configuration is set automatically, the Primary DNS and Secondary DNS options are disabled. Use the A button on the controller to toggle between automatic and manual settings.

  • Primary DNS — Provides a keypad for entering the appropriate primary DNS for the Xbox.
  • Secondary DNS — Provides a keypad for entering the appropriate secondary DNS for the Xbox.

PPPoE Settings

  • Configure PPPoE — Turns on Point-to-Point Protocol over Ethernet (PPPoE) configuration.

    Note    If the Configure PPPoE configuration is off, the user name, password, and service name options are disabled. Use the A button on the controller to toggle between off and on states.

  • User Name — Provides a keypad for entering the appropriate PPPoE user name for the Xbox.
  • Password — Provides a keypad for entering the appropriate PPPoE password for the Xbox.
  • Service Name — Provides a keypad for entering the appropriate PPPoE service name for the Xbox.

Advanced Settings

  • Host Name — Provides a keypad for entering the name of a host, if required by the ISP.
  • MAC Address — Provides a keypad for entering the appropriate MAC address for the Xbox. This settingoverrides the factory-assigned MAC address described in the IP Settings section (see above).

    Note    This refers to the game MAC address for the Xbox. The debug MAC address can be set in the Debug Configuration menu below.

    • Note    A small number of the original Xbox development consoles shipped with MAC addresses that were set to all zeros. If you have such a console, you must enter a valid MAC address or you won't be able to connect to Xbox Live or Partnernet because the console cannot generate a valid gamertag.

Debug Configuration

This section provides a brief overview of the commands available on the Debug Configuration menu. Note that these network settings are separate from the Game Configuration settings listed above.

  • Machine Name — Provides a keypad for entering the name of the Xbox.
  • Configure IP - Determines whether the Dynamic Host Configuration Protocol (DHCP) assigned IP should be configured automatically or manually. By default, the DHCP-assigned IP configuration is determined automatically.

    Note    If the DHCP-assigned IP configuration is set automatically, the DHCP-assigned IP Address, DHCP-assigned Subnet Mask, and DHCP-assigned Gateway options are disabled. Use the A button on the controller to toggle between automatic and manual settings.

  • IP Address — Provides a keypad for entering the appropriate static IP address of the Xbox.
  • Subnet Mask — Provides a keypad for entering the appropriate subnet mask for the Xbox.
  • Gateway — Provides a keypad for entering the appropriate gateway for the Xbox.
  • MAC Address — Specifies the media access control (MAC) address of the Xbox. This is a debug address and is read-only.

Test Connection to PartnerNet

Attempts to connect the Xbox to the Xbox Live service. A message of success or failure is then displayed. In addition to the success message, QOS (Quality Of Service) information is displayed. That information includes:

  • Number of test packets sent.
  • Minimum round-trip time for those packets, in milliseconds.
  • Median round-trip time for those packets, in milliseconds.
  • Average bandwidth (down) to the Xbox from PartnerNet, in kilobits per second.
  • Average bandwidth (up) from the Xbox to PartnerNet, in kilobits per second.
