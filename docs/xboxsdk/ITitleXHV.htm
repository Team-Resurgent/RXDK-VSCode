ITitleXHV

The ITitleXHV interface supports the following methods.

Method Description
CommunicatorStatusUpdate Notification indicating the voice communicator status.
LocalChatDataReady Notification indicating that the local chat data is ready.
MicrophoneRawDataReady Title-defined callback called by the XHV engine to submit raw microphone (voice) data to the title before that data is encoded by the engine.
SpeechRecognized Returns speech recognition information about the specified local talker.
VoiceMailDataReady Notification indicating that the voice message data is ready.
VoiceMailStopped Notification indicating that the voice message has stopped.

The ITitleXHV interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
