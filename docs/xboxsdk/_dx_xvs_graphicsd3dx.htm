xvs

Provides a method to specify the type and version of the Xbox vertex shader code.

xvs.mainVer.subVer

Registers

mainVer
Main version number of the vertex shader.
subVer
Sub version number of the vertex shader.

Remarks

An Xbox vertex shader is like a normal vertex shader, except that:

  • It can write to the output registers oB0 and oB1.
  • It can read from the extended constant registers c[-96]..c[-1].
  • It can use the dph and rcc instructions.

The following example illustrates how the xvs instruction might be used.

xvs.1.1  ; Xbox vertex shader version 1.1.

This instruction is required to be at the beginning of all Xbox vertex shaders.
