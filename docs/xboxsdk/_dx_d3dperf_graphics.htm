D3DPERF

Describes all performance information gathered by the driver.

typedef struct _D3DPERF
{
    ULONGLONG m_TSCLastResetVal;
    ULONGLONG m_TSCLastPresentVal;
    D3DWAITINFO m_FrameTimeMin;
    D3DWAITINFO m_FrameTimeMax;
    DWORD m_BreakCount;
    D3DPERF_PerformanceCounters m_Breakperfctr;
    DWORD *m_pPutLocationAtReset;
    ULONGLONG m_PushBufferEndDwordsSkipped;
    DWORD m_FrameRateIntervalTime;
    DWORD m_dwDumpFPSInfoMask;
    PFNDumpFrameRateInfoHandler m_pfnDumpFrameRateInfoHandler;
    DWORD m_DumpWaitCycleTimesThreshold;
    PFNCycleThresholdHandler m_pfnCycleThresholdHandler;
    BOOL m_SetRenderStateOverridesOnReset;
    DWORD m_ProfileSamples;
    ULONGLONG m_TSCProfileStartTime;
    WORD m_ProfileData[105000];
    DWORD m_ProfileBusyCounts[D3DPERF_PROF_MAX];
    DWORD RecordD3DPerfEvents;
    DWORD D3DPerfEventSpewFrameStart;
    DWORD D3DPerfEventSpewFrameCount;
    DWORD m_DumpPerfCycleTimesThresholdCpu;
    DWORD m_DumpPerfCycleTimesThresholdGpu;
    DWORD CurrentD3DPerfEvent;
    DWORD CountD3DPerfEvents;
    D3DPERFEvent *pD3DPerfEvents;
    D3DOVERRIDERENDERSTATE m_RenderStateOverrides[D3DRS_MAX];
    D3DWAITINFO m_PerformanceCounters[D3DPERF_MAX];
    DWORD m_APICounters[D3DAPI_MAX];
    DWORD m_RenderStateCounters[D3DRS_MAX];
    DWORD m_TextureStateCounters[D3DTSS_MAX];
} D3DPERF;

Members

m_TSCLastResetVal
Time Stamp Counter value of last D3DPERF_Reset call.
m_TSCLastPresentVal
Time Stamp Counter value of last IDirect3DDevice8::Present call.
m_FrameTimeMin
Describes the minimum frame number and cycle counts. See D3DWAITINFO.
m_FrameTimeMax
Describes the maximum frame number and cycle counts. See D3DWAITINFO.
m_BreakCount
Describes the number of times the performance counter specified by the m_Breakperfctr member of D3DPERF should be hit before breaking.
m_Breakperfctr
A member of the D3DPERF_PerformanceCounters enumeration. Specifies on which performance counter a breakpoint should be set.
m_pPutLocationAtReset
Pointer to a DWORD value containing the location of the push-buffer at the time of the call to D3DPERF_Reset. This is used to determine the value of PushBufferBytesWritten in the D3DPUSHBUFFERINFO structure as it is returned by D3DPERF_GetPushBufferInfo.
m_PushBufferEndDwordsSkipped
Count of DWORDs skipped jumping back to pushbuffer start. This is used to determine the value of PushBufferBytesWritten in the D3DPUSHBUFFERINFO structure as it is returned by D3DPERF_GetPushBufferInfo.
m_FrameRateIntervalTime
Interval time to dump framerate information, in milliseconds. By default, this value is zero.
m_dwDumpFPSInfoMask
Mask that controls what information D3DPERF_DumpFrameRateInfo spits out. The mask can be any of the following values ORed together:
D3DPERF_DUMP_FPS_INTINFO
Dump interrupts and context information.
D3DPERF_DUMP_FPS_MEMORY
Dump memory information.
D3DPERF_DUMP_FPS_PUSHBUFFER
Dump push-buffer information.
D3DPERF_DUMP_FPS_WAITINFO
Dumps the various wait timings.
D3DPERF_DUMP_FPS_PERFPROFILE
Dumps times for percent busy and percent idle using a 100-ns timer that pings the GPU. Dumping this information is very expensive and should be avoided unless this is specifically the information you want.
D3DPERF_DUMP_FPS_VERTEXINFO
Dumps information about vertices processed.
D3DPERF_DUMP_FPS_PERFEVENTS
Dumps information about performance events. Setting this value will cause output similar to that generated by D3DPERF_DumpPerfEvents.

The default value is (D3DPERF_DUMP_FPS_INITINFO | D3DPERF_DUMP_FPS_MEMORY | D3DPERF_DUMP_FPS_PUSHBUFFER | D3DPERF_DUMP_FPS_WAITINFO | D3DPERF_DUMP_FPS_VERTEXINFO).

m_pfnDumpFrameRateInfoHandler
Pointer to the function called that handles framerate information. The default value is D3DPERF_DumpFrameRateInfo. PFNDumpFrameRateHandler is defined as follows: 
typedef void (WINAPI *PFNDumpFrameRateInfoHandler)(void);
m_DumpWaitCycleTimesThreshold
Holds threshold cycle count for spewing wait times.
m_pfnCycleThresholdHandler
Pointer to the function called when the threshold cycle count is hit as was set in m_DumpWaitCycleTimesThreshold. The default value is D3DPERF_DumpCounterCycleInfo. PFNCycleThresholdHandler is defined as follows: 
typedef void (WINAPI *PFNCycleThresholdHandler)(
  D3DPERF_PerformanceCounters perfctr,
  ULONGLONG Cycles
);
m_SetRenderStateOverridesOnReset
If TRUE, the next call to D3DPERF_Reset will go through the array stored in m_RenderStateOverrides and set all render states that have the Override member set to TRUE. The default value is FALSE.
m_ProfileSamples
Total number of profile samples read.
m_TSCProfileStartTime
Time profiler was started. Initialized in D3DPERF_StartPerfProfile.
m_ProfileData[105000]
Performance state bitmask history over last 10 ms where it then wraps. Each element of the array represents what was busy during the sampling at that time. The sampling occurs every 100 ns. The bitmask can be interpreted as (1 << value) where value is a member of the D3DPERFPROF enumeration. For example, since 0x7 = (1 << D3DPERF_PROF_GPU) & (1 << D3DPERF_PROF_FE) & (1 << D3DPERF_PROF_BE), this value would indicate that the GPU, the front end, and the back end were all busy. This array is cleared when D3DPERF_StartPerfProfile or D3DPERF_Reset is called.
m_ProfileBusyCounts[D3DPERF_PROF_MAX]
Total busy counts for each unit read while profiler was enabled.
RecordD3DPerfEvents
DWORD value that represents a bit mask of the performance events that are getting recorded. To determine if a given event is getting recorded, bitwise-AND the DWORD with (1 << Event) where Event is an element of the D3DPERFEventsType enumeration.
D3DPerfEventSpewFrameStart
Number of frames to wait before D3DPERF_DumpPerfEvents begins displaying performance information.
D3DPerfEventSpewFrameCount
Number of frames for which performance event information should be spewed.
m_DumpPerfCycleTimesThresholdCpu
Minimum number of cycles required to spew CPU performance times.
m_DumpPerfCycleTimesThresholdGpu
Minimum number of cycles required to spew GPU performance times.
CurrentD3DPerfEvent
Current open event in pD3DPerfEvents.
CountD3DPerfEvents
Total number of elements in pD3DPerfEvents.
pD3DPerfEvents
Pointer to an array of D3DPERFEvent enumerated types, which describe the performance events getting monitored.
m_RenderStateOverrides[D3DRS_MAX]
Used to override any calls to IDirect3DDevice8::SetRenderState at the next call of D3DPERF_Reset. This array is indexed using the D3DRENDERSTATETYPE enumeration. For example, to change the fill mode: 
D3DOVERRIDERENDERSTATE OverrideFillMode = { TRUE, D3DFILL_WIREFRAME };

D3DPERF *pPerf = D3DPERF_GetStatistics;
pPerf->m_RenderStateOverrides[ D3DRS_FILLMODE ] = OverrideFillMode;
m_PerformanceCounters[D3DPERF_MAX]
The count of the number of times certain performance-related events happen. The D3DPERF_PerformanceCounters enumeration is used to index this array.
m_APICounters[D3DAPI_MAX]
The count of the number of times each Direct3D function is called. The D3DPeformanceCounters enumeration is used to index this array.
m_RenderStateCounters[D3DRS_MAX]
The count of the number of times each render state is set. The D3DRENDERSTATETYPE enumeration is used to index this array.
m_TextureStateCounters[D3DTSS_MAX]
The count of the number of times each texture state is set. The D3DTEXTURESTAGESTATETYPE enumeration is used to index this array.

Remarks

This structure is generally obtained, accessed, and modifed using a pointer returned by D3DPERF_GetStatistics.

D3DPERF *pPerf = GetStatistics();
pPerf->m_SetRenderStateOverridesOnReset = TRUE;

The pointer returned by D3DPERF_GetStatistics should not be freed since it is statically allocated.
