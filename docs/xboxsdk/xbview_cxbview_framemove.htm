CXBView::FrameMove (XbView)

CXBView::FrameMove is the frame-by-frame update for XbView. It polls the controllers and interprets the thumbsticks, triggers, and button presses. It also affects the camera motion, lights, and so on, based on input from the user. FrameMove calls Scene::Update so that animation updates happen every frame.
