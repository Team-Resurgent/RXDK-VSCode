UIX_PROPERTY_TYPE

ILiveEngine properties.

typedef enum _UIX_PROPERTY_TYPE{
    UIX_PROPERTY_DISPLAY_NOTIFICATIONS = 0,
    UIX_PROPERTY_HOST_NO_MIGRATION = 1,
    UIX_PROPERTY_DISPLAY_CONNECTION_ERRORS = 2,
    UIX_PROPERTY_ALLOW_GAME_INVITES = 3,
    UIX_PROPERTY_VOICE_MAIL_ENGINE = 4,
    UIX_PROPERTY_VOICE_MAIL_TO_SPEAKERS = 5,
    UIX_PROPERTY_FORCE_DWORD = 0x7fffffff,
} UIX_PROPERTY_TYPE;

Constants

UIX_PROPERTY_DISPLAY_NOTIFICATIONS
Determines whether the Xbox Drop-in UI engine displays in-game notifications automatically or whether the title is responsible for doing so. A value of TRUE (the default) indicates that the engine is displaying the notifications.
UIX_PROPERTY_HOST_NO_MIGRATION
Indicates that this is a host that does not support migration. The default is FALSE. Set this constant to TRUE only if this Xbox console is hosting a game session and the game does not support host migration.
UIX_PROPERTY_DISPLAY_CONNECTION_ERRORS
Determines whether the Xbox Drop-in UI engine displays connection errors when the service connection fails. The default is TRUE. Xbox Live–Aware titles should set this constant to FALSE so that connection errors are not displayed to the player.
UIX_PROPERTY_ALLOW_GAME_INVITES
Determines whether the friends feature displays a menu item that allows game invitations to be sent. The default is TRUE. Set this constant to FALSE to remove from the friends screen the menu item to send game invitations.
UIX_PROPERTY_VOICE_MAIL_ENGINE
Specifies an Xbox High-Level Voice (XHV) engine for use with voice message processing. If NULL is specified, voice messages are no longer processed.
UIX_PROPERTY_VOICE_MAIL_TO_SPEAKERS
Determines whether voice message output is sent to the speakers, rather than the voice communicator. The default is TRUE. Set this constant to FALSE to redirect voice message output to the voice communicator.
UIX_PROPERTY_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.

Remarks

By default, the Xbox Drop-in UI engine displays in-game notifications in the lower right corner of the screen for three seconds per TCR requirements. If the title wants to display the notifications instead, it can set UIX_PROPERTY_DISPLAY_NOTIFICATIONS to FALSE, and then use ILiveEngine::GetNotifications (with UIX_NOTIFICATION_IN_GAME_FLASH) to determine when notifications should be displayed. Titles may wish to render notifications themselves in order to incorporate the notifications within a moving portion of their game (for example, within a car's dashboard). Titles may also wish to turn off automatic notifications if delayed notifications are desired (the Xbox Drop-in UI engine does not currently implement delayed notifications).

If UIX_PROPERTY_HOST_NO_MIGRATION is set to TRUE, the player will receive a warning message (per TCR requirements) when choosing to exit the current game to join a friend in another game. Because the player is hosting and the game has indicated that it does not support host migration, the player is notified that leaving the game will end the session for all the other players.
