IDirectMusicSegment8::GetClockTimeLength

Retrieves the length of the segment in REFERENCE_TIME.

HRESULT GetClockTimeLength(
    REFERENCE_TIME *prtLength,
    BOOL *pfClockTime
);

Parameters

prtLength
Address of a REFERENCE_TIME value that receives the segment length if it is using clock time, zero if it is using music time.
pfClockTime
Address of a Boolean value that indicates whether the REFERENCE_TIME length is being used by the segment. TRUE indicates that the REFERENCE_TIME value received by the prtLength parameter is being used. FALSE indicates that the MUSIC_TIME value is being used.

Return Values

Returns S_OK.
