LocalHandle

The LocalHandle function retrieves the handle associated with the specified pointer to a memory block. Because all memory allocated with the global or local memory functions is fixed, a handle to a memory block is the same as the address of the memory block.

Note  All Globalxxx and Localxxx memory management functions use the title's default heap, and provide less flexibility than the heap functions. Titles should use the heap functions instead.

HLOCAL LocalHandle(
  LPCVOID pMem   // pointer to local memory block
);

Parameters

pMem
[in] Pointer to the first byte of the memory block. This pointer is returned by the LocalLock function.

Return Values

The return value is a handle to the specified memory block. Because all memory allocated with the global or local memory functions is fixed, the handle returned is the same as the address passed in.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
