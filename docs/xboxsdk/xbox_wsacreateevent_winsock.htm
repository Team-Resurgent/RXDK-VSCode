WSACreateEvent

Creates a new event object.

WSAEVENT  WSACreateEvent (void);

Parameters

This function has no parameters.

Return Values

If no error occurs, WSACreateEvent returns the handle of the event object. Otherwise, the return value is WSA_INVALID_EVENT. To get extended error information, call WSAGetLastError.

Remarks

The WSACreateEvent function is used to create an event object created that is manual reset with an initial state of nonsignaled. Winsock event objects are system event objects. Therefore, if a title desires auto reset events, it can call the CreateEvent function directly.

The WSACreateEvent function is equivalent to:

CreateEvent( NULL, TRUE, FALSE, NULL );

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
