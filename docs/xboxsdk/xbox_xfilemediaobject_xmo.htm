XFileMediaObject

This section contains references for XFileMediaObject methods.

The XFileMediaObject interface contains the following methods.

XMediaObject Methods Discontinuity
  Flush
  GetInfo
  GetStatus
  Process
XFileMediaObject Methods DoWork
  GetLength
  Seek

The XFileMediaObject interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
