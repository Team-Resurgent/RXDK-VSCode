XFONT::SetTextHeight

Sets the height the text cell for the current font.

HRESULT __stdcall SetTextHeight(
  unsigned uHeight
);

Parameters

uHeight
[in] The total height of the text cell, in pixels.

Return Values

On success, the function returns S_OK.

Remarks

The default height of a text cell is 16 pixels high.

Requirements
