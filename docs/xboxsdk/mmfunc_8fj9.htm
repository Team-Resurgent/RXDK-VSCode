timeGetSystemTime

The timeGetSystemTime function retrieves the number of milliseconds that have elapsed since the system was started. It is limited to the resolution of the system timer, which is one millisecond (1 ms). Other than using the MMTIME structure to return the elapsed time, this function is identical to the timeGetTime and GetTickCount functions.

MMRESULT timeGetSystemTime(
  LPMMTIME pmmt,  
  UINT cbmmt      
);

Parameters

pmmt
Pointer to an MMTIME structure.
cbmmt
Size, in bytes, of the MMTIME structure.

Return Values

Returns TIMERR_NOERROR. The elapsed time is returned in the ms member of the MMTIME structure. The wType member of the MMTIME structure is set to TIME_MS.

Remarks

The elapsed time is stored as a DWORD value. Therefore, the time will wrap around to zero if the system is run continuously for 2^32 milliseconds, about 49.71 days. This can cause problems in code that uses the return value of timeGetSystemTime directly in computations, particularly where the value is used to control code execution. You should always use the difference between two timeGetSystemTime return values in computations, as shown in the example below.

If you need a higher resolution timer, use a high-resolution timer.

It is not necessary to use the timeBeginPeriod and timeEndPeriod functions in order to use timeGetSystemTime. The timer resolution is always 1 millisecond. The timeBeginPeriod and timeEndPeriod functions are not implemented in the XTL.

The only difference between timeGetTime and timeGetSystemTime is that timeGetSystemTime uses the MMTIME structure to return the system time. Thus timeGetTime has slightly less overhead than timeGetSystemTime.

Example

The following example demonstrates how to handle timer wrap around:

MMTIME mmtStart, mmtCur;

timeGetSystemTime( &mmtStart, sizeof(mmtStart) );

do {
    // Do processing here.
	
    // Check for elapsed time.
    timeGetSystemTime( &mmtCur, sizeof(mmtCur) );
} while( mmtCur.ms - mmtStart.ms < TIMELIMIT );

where TIMELIMIT is the time interval of interest to the application.
