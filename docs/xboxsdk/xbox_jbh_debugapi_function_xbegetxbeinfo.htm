XbeGetXbeInfo

Retrieves information about an XBE on the development system.

DMHRAPI XbeGetXbeInfo(
  LPCTSTR szXbe, 
  PDM_XBE pXbe  
  );

Parameters

szXbe
The XBE file name.
pXbe
Pointer to an DM_XBE structure to receive the information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

This function is similar to DmGetXbeInfo, but it deals with xbe files that are located on the development system.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
