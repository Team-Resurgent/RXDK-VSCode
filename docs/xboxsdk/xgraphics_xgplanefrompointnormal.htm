XGPlaneFromPointNormal

Constructs a plane from a point and a normal.

XGPLANE* XGPlaneFromPointNormal(
  XGPLANE* pOut,
  CONST XGVECTOR3* pPoint,
  CONST XGVECTOR3* pNormal
);

Parameters

pOut
[in, out] Pointer to the XGPLANE structure that is the result of the operation.
pPoint
[in] Pointer to a XGVECTOR3 structure, defining the point used to construct the plane.
pNormal
[in] Pointer to a XGVECTOR3 structure, defining the normal used to construct the plane.

Return Values

Pointer to the XGPLANE structure constructed from the point and the normal.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGPlaneFromPointNormal function can be used as a parameter for another function.
