UIX_LOGON_TYPE

Logon type to perform. Used in UIX_LOGON_PARAMS.

typedef enum _UIX_LOGON_TYPE{
    UIX_LOGON_TYPE_NORMAL = 0,
    UIX_LOGON_TYPE_SILENT = 1,
    UIX_LOGON_TYPE_RETRIEVED_STATE = 2,
    UIX_LOGON_TYPE_RETRIEVED_GAME_INVITE = 3,
    UIX_LOGON_FORCE_DWORD = 0x7fffffff,
} UIX_LOGON_TYPE;

Constants

UIX_LOGON_TYPE_NORMAL
Perform a normal logon.
UIX_LOGON_TYPE_SILENT
Perform a silent logon.
UIX_LOGON_TYPE_RETRIEVED_STATE
Log on using a saved state in an XONLINE_LOGON_STATE structure.
UIX_LOGON_TYPE_RETRIEVED_GAME_INVITE
Log on using a game invite in an XONLINE_ACCEPTED_GAMEINVITE structure.
UIX_LOGON_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.
