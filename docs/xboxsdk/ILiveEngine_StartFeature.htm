ILiveEngine::StartFeature

Starts an Xbox Drop-in UI feature.

HRESULT StartFeature(
    UIX_FEATURE *FeatureID,
    const VOID *pFeatureParams
);

Parameters

FeatureID

[in] Feature to start. Must be one of the following UIX_FEATURE values.
Value Description
UIX_FRIENDS_FEATURE The friends feature.
UIX_LOGON_FEATURE The logon feature.
UIX_PLAYERS_FEATURE The players feature.
UIX_UITEST_FEATURE The UI test feature, for testing a custom ITitleUIPlugin interface. Available in debug builds only.

pFeatureParams

[in] Pointer to a startup parameters structure passed to the feature. The structure must be zero-initialized and the StructSize member set to the structure size. The structure to use depends on the feature.
Value Description
UIX_FRIENDS_FEATURE Pointer to a UIX_FRIENDS_PARAMS structure.
UIX_LOGON_FEATURE Pointer to a UIX_LOGON_PARAMS structure.
UIX_PLAYERS_FEATURE Pointer to a UIX_PLAYERS_PARAMS structure.
UIX_UITEST_FEATURE Pointer to a UIX_UITEST_PARAMS structure.

Return Values

Returns S_OK if successful, an error code otherwise.

Returns ERROR_NOT_FOUND if the feature has not been enabled before calling StartFeature.

Remarks

A feature must have been enabled with the EnableFeature method before it can be started.

Titles using Xbox High-Level Voice (XHV) should initialize the voice engine (with the XHVEngineCreate function) only after the logon feature reports a successful logon.

The Xbox Live engine should not be released while a feature is active. The ILiveEngine::EndFeature method should be called for an active feature before the engine is released.

A UI plug-in must be set using the ILiveEngine::SetUIPlugin method before calling StartFeature.
