Matchmaking Functions

The Xbox Live matchmaking functions include the following.

Function Description
XOnlineMatchSearch Searches for online-game sessions that match the specified attributes.
XOnlineMatchSearchGetResults Retrieves the results from a previous call to XOnlineMatchSearch or XOnlineMatchSessionFindFromID.
XOnlineMatchSearchParse Parses the search results returned by the XOnlineMatchSearchGetResults function.
XOnlineMatchSearchResultsLen Returns the size in bytes of the results returned by the XOnlineMatchSearch function. Pass this size as the value of the dwResultsLen parameter to the XOnlineMatchSearch function.
XOnlineMatchSessionCreate Initializes a session creation request.
XOnlineMatchSessionDelete Creates a delete session request.
XOnlineMatchSessionFindFromID Searches for the online-game session that corresponds to the specified session identifier.
XOnlineMatchSessionGetInfo Obtains information for the specified online match session.
XOnlineMatchSessionUpdate Updates the information associated with an online-game session.
