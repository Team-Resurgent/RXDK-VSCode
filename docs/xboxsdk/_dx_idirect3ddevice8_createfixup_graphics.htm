IDirect3DDevice8::CreateFixup

Creates an empty fix-up object for use in creating per-call fix-ups with IDirect3DPushBuffer8::BeginFixup.

HRESULT CreateFixup(
  UINT Size,
  IDirect3DFixup8 **ppFixup
);

Parameters

Size
[in] Size in bytes of the fix-up object to create.
ppFixup
[out, retval] Address of a pointer to an IDirect3DFixup8 interface, representing the created fix-up object.

Return Values

If the method succeeds, the return value is S_OK. If the method fails due to lack of memory, the return value is E_OUTOFMEMORY.

Remarks

The parameter Size can be zero in order to create a fix-up object that can be used to query the necessary fix-up size.

As with any Direct3D resource, Release should be called when done with the fix-up resource.
