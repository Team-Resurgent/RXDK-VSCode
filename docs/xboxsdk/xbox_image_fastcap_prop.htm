/FASTCAP (Fast Call-Attribute Profiling)

/FASTCAP

The /FASTCAP option enables fast call-attribute profiling, which is required when using XbPerf to profile the performance of a game.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the C\C++ folder.
  3. Click the Advanced property page.
  4. To enable fast call-attribute profiling, set the FastCAP property to Yes.

When using this compiler switch, you must link your project to xbdm.lib or you will get link errors resembling the following:

error LINK2019: unresolved external symbol __CAP_End_Profiling@4 referenced in function ...
error LINK2019: unresolved external symbol __CAP_Start_Profiling@4 referenced in function ...
In addition, Visual Studio's incremental linking, and features that use it such as edit-and-continue, should not be enabled. An error similar to the following indicates that the incremental linking is currently enabled: 
<yourfile>.obj : warning LNK4202 : spawning full build due to '/INCREMENTAL:NO' directive
LINK : warning LNK4075 : ignoring '/EDITANDCONTINUE' due to '/INCREMENTAL:NO' directive

To disable incremental linking in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click Linker.
  3. Set the Enable Incremental Linking property to No.
