XWmaFileMediaObject

Provides access to an Xbox Media Object (XMO), which is used to decode WMA files.

The XWmaFileMediaObject interface contains the following methods:

XMediaObject Methods Discontinuity
  Flush
  GetInfo
  GetStatus
  Process
XFileMediaObject Methods DoWork
  GetLength
  Seek
XWmaFileMediaObject Methods GetFileContentDescription
  GetFileHeader
  SeekToTime

The XWmaFileMediaObject interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

When working with WMA XMO's, error output may be generated for debugging purposes. The following table displays the error codes that can be returned while using WMA XMO's:

Error Code Description
0 Success
1 General Failure
2 Bad argument passed in
3 Bad ASF header
4 Bad packet header
5 Broken frame, data is inconsistent
6 No more data to decode
7 Invalid or unsupported sampling rate
8 Invalid or unsupported number of channels
9 Invalid or unsupported version number
10 Invalid or unsupported weighting mode
11 Invalid or unsupported packetisation
