/TESTRATINGS (Xbox Game Rating)

/TESTRATINGS:number

Where:

number
Specifies the ESRB rating for the image.

Remarks

The ESRB rating can be any of the following values:

ConstantValue
XC_PC_ESRB_ALL 0x00000000
XC_PC_ESRB_ADULT 0x00000001
XC_PC_ESRB_MATURE 0x00000002
XC_PC_ESRB_TEEN 0x00000003
XC_PC_ESRB_EVERYONE 0x00000004
XC_PC_ESRB_KIDS_TO_ADULTS 0x00000005
XC_PC_ESRB_EARLY_CHILDHOOD 0x00000006

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the Certificate property page.
  4. Modify the Xbox Game Rating property.
