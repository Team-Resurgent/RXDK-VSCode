XWriteStatStore

Stores the specified statistic for the specified player on the local Xbox hard disk. This statistic is later stored on the specified Xbox Live leaderboard.

Note    Leaderboards are also referred to as scoreboards.

HRESULT XWriteStatStore(
    LPCWSTR lpLocalUserName,
    DWORD dwLeaderBoardIndex,
    WORD wAttributeIndex,
    XOFFLINE_STAT_TYPE Type,
    VOID *pStatValue
);

Parameters

lpLocalUserName
[in] Pointer to a Unicode string that contains the title-defined identifier for the player. This string is not the player's gamertag on the Xbox Live service.
dwLeaderBoardIndex
[in] Leaderboard on which to store the statistic.
wAttributeIndex
[in] Title-defined identifier for the statistic.
Type

Type of the attribute (statistic) to store. This parameter is a value from the enumeration type XOFFLINE_STAT_TYPE:
Value Meaning
XOFFLINE_STAT_LONG The statistic is an integer (32-bit).
XOFFLINE_STAT_LONGLONG The statistic is a big integer (64-bit).
XOFFLINE_STAT_DOUBLE The statistic is an real number (64-bit).

pStatValue
Pointer to the value to store for this statistic.

Remarks

XWriteStatStore does not digitally sign the statistics that it writes to the Xbox hard disk.

2 MB of free blocks are reserved in the Xbox hard disk when XWriteStatStore is executed. If there are no free blocks on the Xbox hard disk, the function fails. Therefore, your title should ensure that that 2 MB is available on the Xbox hard disk at the time it loads.
