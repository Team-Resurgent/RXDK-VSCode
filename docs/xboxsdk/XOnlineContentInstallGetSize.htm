XOnlineContentInstallGetSize

Retrieves the full install size (in blocks) of a piece of downloaded content.

HRESULT XOnlineContentInstallGetSize(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwTotalInstalledSizeInBlocks,
    DWORD *pdwAdditionalBlocksRequired
);

Parameters

hTask
[in] Task handle created by XOnlineContentInstall (but before XOnlineTaskClose).
pdwTotalInstalledSizeInBlocks
[out] Number of blocks required to install the content.
pdwAdditionalBlocksRequired
[out] Returns zero if the disk has enough space to install the content. If more disk space is required, this value returns the number of additional blocks needed to install the content. This value factors in the additional overhead for download, and parts of the content that are already installed. Because there is a download overhead, note that this value can be greater than the value returned in pdwTotalInstalledSizeInBlocks.

Return Values

Returns S_OK if the there is enough space, or S_FALSE, if additional disk space is required to install the content.

Remarks

There can be a delay before XOnlineContentInstall gets the install size from the server. If XOnlineContentInstallGetSize is called before the server returns the install size, both pdwTotalInstalledSizeInBlocks and pdwAdditionalBlocksRequired return zero, and the function returns S_OK.

The following is an example of using this function:

hr = XOnlineContentInstall( OfferingID, ... , &hTask);
do
{
   hr = XOnlineTaskContinue(hTask);
   if (hr == HRESULT_FROM_WIN32(ERROR_DISK_FULL))
   {
      DWORD dwInstallSize, dwBlocksRequired;
      hr = XOnlineContentInstallGetSize(hTask, &dwInstallSize, &dwBlocksRequired);
      Assert(dwBlocksRequired > 0);
      XOnlineTaskClose(hTask);
      DisplayDiskFullMessageWithRequiredBlockCount( dwBlocksRequired );
      BootToDashMemoryArea( );
   }
} while (hr == XONLINETASK_S_RUNNING)
