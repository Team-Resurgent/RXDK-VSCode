IXVoiceDecoder

Provides access to one of four available voice chat decoders.

Method Description
GetVoiceProperties Retrieves an array of XVOICE_VOICE_PROPERTIES structures that contain voice properties (lip-synchronization information) for this decoder's input streams.
ProcessMultiple Decompresses the data packets in the specified input array, and outputs the decompressed data to the specified output array.
XMediaObject Method
Discontinuity
Flush
GetInfo
GetStatus
Process

The IXVoiceDecoder interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
