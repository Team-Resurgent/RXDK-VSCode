BaseSkeletonNode::Sort (XbView)

BaseSkeletonNode::Sort calls the STL sort routine for a list with the functor CompareBaseFrameFrameNodeLessThan. It also checks a Boolean to make sure you do not want a rigorous sort and checks the Boolean that is set when you have sorted once. BaseSkeletonNode::Sort does not implement a rigorous depth-based sort. Instead, geometry with alpha blend turned on is pushed to the bottom of the list. This needs to happen only once, and the list's order need not change again, so you should store a Boolean to insure that you don't call the sort needlessly. It is somewhat redundant to have the Boolean here, because the Scene::Sort will not even call this routine if its Boolean for checking that the sort has already occurred is set. This redundancy is here in case the end user wants to call this routine directly and bypass the Scene::Sort.
