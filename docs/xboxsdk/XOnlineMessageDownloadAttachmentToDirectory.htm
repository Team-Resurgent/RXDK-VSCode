XOnlineMessageDownloadAttachmentToDirectory

Downloads a large attachment to a directory.

HRESULT XOnlineMessageDownloadAttachmentToDirectory(
    XONLINETASK_HANDLE hDetailsTask,
    WORD wPropTag,
    LPCSTR lpLocalPath,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phDownloadTask
);

Parameters

hDetailsTask
[in] Task handle obtained from XOnlineMessageDetails.
wPropTag
[in] Unique identifier of the attachment.
lpLocalPath
[in] Local path (on the Xbox) to the download directory.
hWorkEvent
[in, optional] Handle to an event to fire whenever the download is occurring. This enables your title to call the XOnlineTaskContinue function. If you do not want to associate an event with this task, set this parameter to NULL.
phDownloadTask
[out] Receives a handle to the download task. This handle can be passed to XOnlineMessageDownloadAttachmentGetProgress to report the progress of the download.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

A property tag specifies the type of the property in its most significant byte. Use the XOnlineMessageGetPropType macro to retrieve it. Before calling the XOnlineMessageDownloadAttachmentToDirectory function, ensure that the property type is XONLINE_MSG_PROP_TYPE_ATTACHMENT. Also, call the XOnlineMessageDetailsGetResultsProperty function to ensure that its pdwAttachmentFlags parameter contains the value XONLINE_MSG_ATTACHMENT_FLAG_DIRECTORY and its return type is XONLINE_E_MESSAGE_PROPERTY_DOWNLOAD_REQUIRED.

Only one download operation at a time is supported.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
