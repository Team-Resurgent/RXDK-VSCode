UIX_EXIT_INFO

Feature exit information obtained with the ILiveEngine::GetExitInfo method.

typedef struct _UIX_EXIT_INFO{
    UIX_FEATURE FeatureID;
    UIX_EXIT_CODE_TYPE ExitCode;
    HRESULT hr;
    PVOID pExitData;
} UIX_EXIT_INFO, *PUIX_EXIT_INFO;

Members

FeatureID
Feature for which exit information was returned. The exit information is for the feature that most recently exited and thus caused the UIX_DOWORK_FEATURE_EXIT flag to be set.
ExitCode

Value indicating the exit code for the feature.
Exit Code Description
UIX_EXIT_NONE The feature is still active (has not exited).
UIX_EXIT_LOGON_SUCCESSFUL The logon feature logged on successfully.
UIX_EXIT_LOGON_FAILED The logon feature exited but failed to log on. This value is also returned when the network connection is lost and the logon feature is not currently active.
UIX_EXIT_LOGON_USER_EXIT The player exited the logon feature.
UIX_EXIT_FRIENDS_NORMAL_EXIT The friends feature exited as usual.
UIX_EXIT_FRIENDS_JOIN_GAME The friends feature exited because the player accepted a game invite.
UIX_EXIT_FRIENDS_JOIN_GAME_CROSS_TITLE The friends feature exited because the player accepted a cross-title game invite.
UIX_EXIT_FRIENDS_SIGNED_OUT The friends feature exited because the player selected the Sign Out button on the friends screen. Before the friends feature exits and returns this code, it calls the ILiveEngine::LogOff method.
UIX_EXIT_PLAYERS_NORMAL_EXIT The UI test feature exited normally. This exit code is available in debug builds only.

After a successful logon (with the value UIX_EXIT_LOGON_SUCCESSFUL), the title must call the XOnlineGetServiceInfo function for each service requested at logon in the LogonServiceIDs member of the UIX_LOGON_PARAMS structure. The title then checks the returned HRESULT types to verify which services were successfully connected to. If a service critical to the title failed, the title should log off the Xbox Live service.

If a title must determine the problem that caused the UIX_EXIT_LOGON_FAILED value to be returned, the title can check the hr member of this structure.

hr
Success or failure result code returned by the feature.
pExitData

Exit data returned by the feature. This data varies depending on the feature and exit code, as follows.
FeatureID value ExitCode value pExitData value
UIX_LOGON_FEATURE UIX_EXIT_LOGON_SUCCESSFUL A pointer to a UIX_LOGON_EXIT_DATA structure that contains the controller port assignments
UIX_FRIENDS_FEATURE UIX_EXIT_FRIENDS_JOIN_GAME A pointer to the XONLINE_FRIEND structure of the friend that sent the game invite

The pExitData member is NULL for any other combination of FeatureID and ExitCode member values.
