X

x-axis
The axis that normally extends horizontally across a coordinate space.
Xbox
The whole brand: game system, games, Xbox Live service, everything.
Xbox Controller
The gamepad provided with the Xbox video game system.
Xbox Dashboard
The applications/utilities that reside permanently on the Xbox hard disk and run when there is no game running on the game system. The configuration user interface (UI), the DVD movie playback capability and the CD audio player are all components of the Xbox Dashboard. The default software that Microsoft provides in the absence of an inserted game.
Xbox Dashboard—configuration
Group of core settings that are available through the main shell interface of the Xbox game system. The Dashboard provides users with a simple and intuitive interface for changing parental control, audio, video, and other necessary settings on the Xbox. See Xbox Dashboard.
Xbox Development Kit (XDK)
The Xbox software development package. This includes the Xbox console, printed material, Microsoft® Visual C++® compiler, Microsoft® Windows® 2000 Professional, and the Xbox system software. It also includes all Xbox documentation, samples, and tools.
Xbox graphics
The Xbox video system.
Xbox Live
Xbox Live is an online service provided by Microsoft to enhance the functionality of Xbox game titles. Xbox Live includes the following services: Content Download, Friends List, Matchmaking, Statistics, and Voice. For more information, see Introduction to Xbox Live.
Xbox Live Aware
Feature that allows non–Xbox Live games to take advantage of the cross-title invitation and notification features of the Xbox Live service. Xbox Live Aware titles expose presence and notification functionality so that, even when playing an offline game, a player can be notified when friends send game invitations.
Xbox Live Now
Feature that enables Xbox Live users to see their friends' status, receive game invites in the Xbox Dashboard, set up and join in voice chats with their buddies, and view timely information on Xbox Live events and features.
Xbox video game system
The end-user Xbox entertainment console and peripherals.
Xbox System Link Cable
An Ethernet crossover cable used to connect Xbox consoles, enabling System Link Play between players.
Xbox system software
The operating system for the Xbox game system and all titles that play on it. The components include the operating system kernel (in the Xbox game system ROM (read-only memory), the Xbox Title Libraries (linked into titles that ship on discs), the utilities on the Xbox hard disk (such as the Xbox Dashboard), and all drivers for peripherals, audio, video, and so forth.
Xbox Title Libraries (XTL)
The libraries used to develop software for the Xbox. Includes a subset of the Microsoft® Win32® API set, a subset of Microsoft® DirectX® API set, and new Xbox-specific services.
