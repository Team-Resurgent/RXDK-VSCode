XOnlineUserNoShowRating

Macro that, given the dwUserFlags value from an XUID structure, determines a player's no show rating.

#define XOnlineUserNoShowRating(dwUserFlags) \
	(((dwUserFlags) & XONLINE_USER_NOSHOW_RATING_MASK) >> 2)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is the new value of dwUserFlags.

The return value is of type BYTE.

Requirements

Header: Declared in Xonline.h.
