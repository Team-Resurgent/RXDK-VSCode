Xbox Live Enumerations

This section contains the enumerations used by the Xbox Live functions.

Enumeration Description
XONLINE_COMP_FREQUENCY_UNIT Specifies when a reminder should be sent to participants in a competition.
XONLINE_FEEDBACK_TYPE Defines the possible types of feedback that can be sent to the Xbox Live service using the XOnlineFeedbackSend function.
XONLINE_FRIENDS_RESULTS_TYPE Defines the possible types of player list results.
XONLINE_GAMEINVITE_ANSWER_TYPE Defines the possible answers to a game invitation.
XONLINE_NAT_TYPE Defines the type of network address translation (NAT).
XONLINE_NOTIFICATION_EVENT_TYPE Defines the possible notification event types.
XONLINE_NOTIFICATION_TYPE Specifies the notification types that can be tested for using the XOnlineGetNotification function.
XONLINE_OFFERING_FREQUENCY Specifies how often charges are made for an offering.
XONLINE_PASSCODE_TYPE Defines values that allow players to enter pass codes with an Xbox controller.
XONLINE_PEER_ANSWER_TYPE Defines the possible responses to a peer request.
XONLINE_REQUEST_ANSWER_TYPE Defines the possible responses to a request.
XONLINE_STAT_PARAM_TYPE Describes parameter types for a title-specific statistics update procedure.
XONLINE_STAT_SORTORDER Specifies values that determine how to sort statistics data.
XONLINE_STAT_TYPE Defines the possible types for statistics used by the statistics service.
XONLINE_TAX_TYPE Defines the possible types for tax used by the online offering pricing.
XONLINE_TEAM_LIVE_PERMISSIONS Specifies reserved values used for permissions.
XONLINESTORAGE_CONTENT_TYPE Indicates the type of content and the function that must be used to download it.
XONLINESTORAGE_FACILITY Defines the types of storage facility.
