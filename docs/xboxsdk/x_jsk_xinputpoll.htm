XInputPoll

The XInputPoll function can be used together with the pPollingParameters argument to XInputOpen in order to exercise very fine control over polling.

DWORD XInputPoll(
  HANDLE hDevice     
);

Parameters

hDevice
[in] Handle to a device opened via XInputOpen.

Return Values

If the function succeeds, the return values is ERROR_SUCCESS.

If the function fails, the return value is an error code defined in Winerror.h. The function does not use SetLastError to set the calling thread's last-error code.

Remarks

XInputPoll should be used only if manual polling was specified when the device was opened. If the device was not opened for manual polling, XInputPoll returns success but performs no operation.

Manually polling is set through the pPollingParameters argument to XInputOpen. The XINPUT_POLLING_PARAMETERS topic provides detailed information on setting polling options.
