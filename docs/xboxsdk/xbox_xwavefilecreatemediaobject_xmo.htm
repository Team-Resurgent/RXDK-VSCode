XWaveFileCreateMediaObject

Creates an Xbox Media Object (XMO) from a wave file on a disk.

HRESULT XWaveFileCreateMediaObject(
    LPCSTR pszFileName,
    LPCWAVEFORMATEX *ppwfxFormat,
    XFileMediaObject **ppMediaObject
);

Parameters

pszFileName
String value that contains the file name to use as the source of the streamed data.
ppwfxFormat
Address of a pointer to a WAVEFORMATEX structure that describes the input format. This will be filled in by the XMO as specified by the header information in the .wav file.
ppMediaObject
Address of a pointer to the XFileMediaObject that is returned.

Return Values

Returns DS_OK if successful, an XMO-specific error code otherwise.

Remarks

The title is responsible for freeing the XWaveFileMediaObject that is created by using the Release method.
