IXACTEngine::FlushNotification

Flush all pending notifications of a specific type from the notification queue.

HRESULT FlushNotification(
    PCXACT_NOTIFICATION_DESCRIPTION pNotificationDesc
);

Parameters

pNotificationDesc
[in] A pointer to an XACT_NOTIFICATION_DESCRIPTION structure describing the notification(s) to flush. If this parameter is NULL, all notifications will be flushed.

Note    The pSoundCue parameter of the XACT_NOTIFICATION_DESCRIPTION structure is reserved when calling this function and should be set to NULL.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
