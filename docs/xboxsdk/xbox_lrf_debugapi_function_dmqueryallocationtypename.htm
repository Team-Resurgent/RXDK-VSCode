DmQueryAllocationTypeName

Retrieves the string description for both internal and custom memory allocations. This function is used in conjunction with the XbMemDump tool when tracking memory allocations.

HRESULT DmQueryAllocationTypeName(
  USHORT  AllocationType,
  LPSTR   pszName,
  SIZE_T  nSize
  );

Parameters

AllocationType
The allocation type value whose string description should be retrieved. Internal allocation types are in the range 0–13, inclusive.
pszName
Pointer to a buffer that receives the string description of the allocation type.
nSize
Size, in bytes, of the pszName buffer.

Return Values

If DmQueryAllocationTypeName successfully retrieves the name of an allocation type, it returns XBDM_NOERR. If the AllocationType specified has not been assigned, DmQueryAllocationTypeName returns E_INVALIDARG.

Remarks

Internal memory allocation types and their corresponding type values are outlined in the following table:

Constant Value Description
0 DM_ALLOCTYPE_VIRTUAL_MEMORY Virtual memory -- A section of memory reserved by the CPU and the system code to hold the tables that map physical addresses to virtual addresses.
1 DM_ALLOCTYPE_CONTIGUOUS_MEMORY Physically contiguous memory -- Allocated using XPhysicalAlloc or D3D_AllocContiguousMemory. This type of memory is guaranteed to always be in sequential physical pages.
2 DM_ALLOCTYPE_SYSTEM_MEMORY System memory -- A section of memory used by the CPU and the system code to hold the tables that identify physical pages and map them into the physical address space.
3 DM_ALLOCTYPE_DEBUG_MEMORY Debug memory -- Allocated by the XBDM debug manager to allow remote debugging of a title over the network.
4 DM_ALLOCTYPE_KERNEL_POOL Kernel pool -- A section of memory reserved by the system for system memory allocations.
5 DM_ALLOCTYPE_DIRECTORY_OBJECT Directory object handle
6 DM_ALLOCTYPE_EVENT Event handle
7 DM_ALLOCTYPE_FILE File handle
8 DM_ALLOCTYPE_IO_COMPLETION I/O completion handle
9 DM_ALLOCTYPE_MUTANT Mutant handle
10 DM_ALLOCTYPE_SEMAPHORE Semaphore handle
11 DM_ALLOCTYPE_TIMER Timer handle
12 DM_ALLOCTYPE_THREAD Thread handle
13 DM_ALLOCTYPE_HEAP Heap memory
