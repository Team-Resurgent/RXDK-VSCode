XMVRESULT

Used in the return value of DOCERROR: name is not a valid attribute...XMVDecoder::GetNextFrame to describe the status of the new frame.

typedef enum _XMVRESULT {
    XMV_NOFRAME,
    XMV_NEWFRAME,
    XMV_ENDOFFILE,
    XMV_FAIL,
    XMV_FORCELONG = 0xFFFFFFFF,
} XMVRESULT;

Constants

XMV_NOFRAME
There is no new data, and the surface passed into DOCERROR: name is not a valid attribute...XMVDecoder::GetNextFrame is unchanged.
XMV_NEWFRAME
The surface passed into DOCERROR: name is not a valid attribute...XMVDecoder::GetNextFrame contains a new frame that needs to be displayed.
XMV_ENDOFFILE
The movie is over. The decoder has reached the end of the file.
XMV_FAIL
An error has occured that prevents the video from being decoded. This could happen through either major data corruption or a read error.
XMV_FORCELONG
Forces the enumeration to compile to 32 bits in size. This value is not used.
