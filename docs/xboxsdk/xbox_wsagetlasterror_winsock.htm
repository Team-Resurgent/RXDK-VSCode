WSAGetLastError

Gets the error status for the last operation that failed.

int  WSAGetLastError (void);

Parameters

This function has no parameters.

Return Values

The return value indicates the error code for this thread's last Winsock operation that failed.

Remarks

The WSAGetLastError function returns the last network error that occurred for any socket belonging to the calling thread. When a particular Winsock function indicates that an error has occurred, this function should be called to retrieve the appropriate error code.

A successful function call, or a call to WSAGetLastError, does not reset the error code. To reset the error code, use the WSASetLastError function call with iError set to zero.

WSAGetLastError is equivalent to GetLastError.

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
