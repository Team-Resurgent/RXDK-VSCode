DmConnectDebugger

Sets the response that DmIsDebuggerPresent will return.

DMHRAPI DmConnectDebugger(
  BOOL fConnect
  );

Parameters

fConnect
Boolean value indicating whether a debugger is connected or not.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

When a debugger connects to the target system, it should call DmConnectDebugger(TRUE) so that calls to DmIsDebuggerPresent will return TRUE, indicating that a debugger is connected.

Requirements
