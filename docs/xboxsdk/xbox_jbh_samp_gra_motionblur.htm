MotionBlur Sample

The MotionBlur sample illustrates one way of performing motion blur on the Xbox.

Overview

The motion blur technique is implemented as follows:

  1. Create a blend texture with Z buffer and a composite texture.
  2. The blend texture gets ( 1 / numpasses ) contribution each time the blurred object is rendered into it.
  3. The composite texture has the blend texture added to it for each pass. So numpasses additions of ( 1 / numpasses ) contributions sums up to 1.0 max.
  4. To handle proper Z buffering into the scene, the scene Z buffer is copied to the blend texture Z buffer for each pass.
  5. After the composite is complete, it is rendered to the screen using alpha blending.

Controls
