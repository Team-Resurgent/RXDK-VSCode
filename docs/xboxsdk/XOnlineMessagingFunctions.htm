Messaging Functions

The Xbox Live messaging functions include the following.

Function Description
XOnlineMessageCreate Creates a new message to send.
XOnlineMessageDelete Deletes a message received from the server.
XOnlineMessageDestroy Releases the memory associated with a message.
XOnlineMessageDetails Retrieves the details of a particular message.
XOnlineMessageDetailsGetResultsProperty Retrieves the specified property from a message's details.
XOnlineMessageDetailsGetResultsSummary Retrieves the summary for a particular message after XOnlineMessageDetails finishes.
XOnlineMessageDownloadAttachmentGetProgress Reports the progress of downloading an attachment.
XOnlineMessageDownloadAttachmentToDirectory Downloads a large attachment to a directory.
XOnlineMessageDownloadAttachmentToMemory Downloads an attachment to memory.
XOnlineMessageDownloadAttachmentToMemoryGetResults Returns the results from calling XOnlineMessageDownloadAttachmentToMemory.
XOnlineMessageEnableReceivingFamilyTitleIDs Specifies which family title IDs are valid for messages.
XOnlineMessageEnumerate Retrieves a list of message summaries from the Xbox Live service.
XOnlineMessageRevoke Revokes a message that has already been sent.
XOnlineMessageSend Sends a message to one or more players.
XOnlineMessageSendGetProgress Retrieves the progress of a message transmission.
XOnlineMessageSendGetResults Retrieves the results of a message transmission.
XOnlineMessageSetFlags Sets the flags associated with a message.
XOnlineMessageSetProperty Associates a property with a message.
XOnlineMessageSetSendingFamilyTitleID Sets the family title ID for messaging.
XOnlineMessageSetSummaryRefresh Enables or disables automatic summary refreshing, and returns the previous setting.
XOnlineMessageSummary Retrieves the summary for a particular message.
