XMVCompare Sample

The XMVCompare sample compares two XMV videos side-by-side for visual quality differences.

Overview

This sample enumerates videos from media\videos directory and enables you to select two XMV files for comparison. It displays the videos in split-screen mode.

Path

Source: (XDK Root)\Samples\Xbox\Video\XMVCompare

Remarks

Before running this sample, please be sure and copy the media tree to the target Xbox.

Prior to using this sample, perform two different encodings of a video. Copy the files to media\videos directory.You can then run this sample and compare the visual quality of encoder's settings.

Note    You'll need to encode video only with no audio using -a_setting 0

The sample enumerates videos stored in the media\videos directory. Use the A and B buttons to cycle through the enumerated list. Press the A button to select the video for the left pane. Pressing the B button selects the video for the right pane.

To start both videos, press the X button. Pause playback with the Start button. While the video is paused, you can use the left thumb stick to move the splitter left or right. If the videos get out of sync, press the left or right thumb sticks to advance the left or right video one video frame, respectively.

This program uses the left and right triggers to display the left and right panes, respectively, in "full screen" mode. The splitter actually moves to far left or right. Use the left thumbstick to scroll it back to viewable position.

Other options:

Black and white buttons
Select and change display modes supported with the current AV pack.
Back button
Display the help screen.
Start button
Pause both videos.
Dpad left/right
Change splitter color.
Dpad up/down
Change splitter scroll rate.
Y button
Toggle on-screen text display.

The example videos have the following encodings:

VideoA: -v_mode 1 -v_quality 100 -v_bitrate 2000000 -v_buffer 5000

VideoB: -v_mode 1 -v_quality 50 -v_bitrate 500000 -v_buffer 5000
