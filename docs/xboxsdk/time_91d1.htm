CompareFileTime

Compares two 64-bit file times. 

LONG CompareFileTime(
  CONST FILETIME *lpFileTime1,  // first file time
  CONST FILETIME *lpFileTime2   // second file time
);

Parameters

lpFileTime1
[in] Pointer to a FILETIME structure that specifies the first 64-bit file time.
lpFileTime2
[in] Pointer to a FILETIME structure that specifies the second 64-bit file time.

Return Values

The return value is one of the following values.

Value Meaning
–1 First file time is less than second file time.
0 First file time is equal to second file time.
+1 First file time is greater than second file time.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
