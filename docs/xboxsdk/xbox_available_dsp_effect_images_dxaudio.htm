Effect Image Parameters

Each effect in the DSP effects image has a block of data (represented by the parameter structure) that contains the state information and parameters for each effect. Titles can use IDirectSound8::SetEffectData to modify those parameters.

Note    For the I3DL2 reverberation effect, titles must use the IDirectSound8::SetI3DL2Listener, IDirectSoundBuffer8::SetI3DL2Source, and IDirectSoundStream::SetI3DL2Source methods.
