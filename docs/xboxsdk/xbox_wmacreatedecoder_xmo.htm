WmaCreateDecoder

Creates and initializes an Xbox Media Object that performs Microsoft® Windows Media® Audio (WMA) decoding.

HRESULT WINAPI WmaCreateDecoder(
    LPCSTR pszFileName,
    HANDLE hFile,
    BOOL fAsyncMode,
    DWORD dwLookaheadBufferSize,
    DWORD dwMaxPackets,
    DWORD dwYieldRate,
    LPWAVEFORMATEX pwfxCompressed,
    XFileMediaObject **ppMediaObject
);

Parameters

pszFileName
[in] The file name of the source data. If this is NULL, a valid handle has to be passed to hfile.
hFile
[in] A valid handle to a WMA file. If this is NULL, a valid file name must be given to pszFileName.
fAsyncMode
[in] Boolean value that determines how the decoder will be created. If TRUE, the decoder will be created for asynchronous operation, which means the Process method will return immediately after it is called. It is important to note that this causes a separate thread to be created that will be used to perform the asynchronous processing. If FALSE, the decoder will be created for synchronous operation. This is the recommended method as it allows synchronous decoding on threads managed by the application.
dwLookaheadBufferSize
[in] The number of bytes the decoder will look ahead when decoding.
dwMaxPackets
Value that indicates the maximum number of attached packets at any given time.
dwYieldRate
Value that specifies how often the decoder should yield execution while decoding data. The value zero indicates it should never yield, the value one indicates it should always yield, and a value two or higher indicates it should yield once every n iterations (where n = dwYieldRate).
pwfxCompressed
[out] Address of a WAVEFORMATEX that describes the input format.
ppMediaObject
[out, retval] Address of a pointer to the Xbox Media Object returned.

Return Values

Returns DS_OK if successful.

Remarks

WmaCreateDecoder creates an Xbox Media Object synchronously. If a title needs the creation function to return immediately, the title should use the WmaCreateInMemoryDecoder instead.

Either the pszFileName or hFile parameter must be NULL.
