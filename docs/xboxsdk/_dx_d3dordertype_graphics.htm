D3DORDERTYPE

Defines the order type of a high-order surface.

typedef enum _D3DORDERTYPE {
   D3DORDER_LINEAR      = 1,
   D3DORDER_CUBIC       = 3,
   D3DORDER_QUINTIC     = 5, 

   D3DORDER_FORCE_DWORD = 0x7fffffff
} D3DORDERTYPE;

Constants

D3DORDER_LINEAR
Linear order type.
D3DORDER_CUBIC
Cubic order type.
D3DORDER_QUINTIC
Quintic order type.
D3DORDER_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.
