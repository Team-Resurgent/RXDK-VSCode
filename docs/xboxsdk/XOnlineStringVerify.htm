XOnlineStringVerify

Checks text strings to ensure that they do not contain objectionable content. Use this function to verify the strings that a user provides before uploading or storing them where other users can retrieve and display them.

Note    XOnlineTeamCreate and XOnlineTeamSetProperties automatically verify the strings in the XONLINE_TEAM_PROPERTIES structure. Titles do not need to verify the strings themselves before calling those functions.

HRESULT XOnlineStringVerify(
    WORD wNumStrings,
    LPCWSTR *ppwStrings,
    DWORD dwLanguage,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

wNumStrings
[in] Number of strings to verify.
ppwStrings
[in] Array of Unicode strings to verify.
dwLanguage

[in] Value that specifies the language of the text to verify. This parameter must be one of the following values:
Value Description
XC_LANGUAGE_ENGLISH English
XC_LANGUAGE_JAPANESE Japanese
XC_LANGUAGE_GERMAN German
XC_LANGUAGE_FRENCH French
XC_LANGUAGE_SPANISH Spanish
XC_LANGUAGE_ITALIAN Italian
XC_LANGUAGE_KOREAN Korean
XC_LANGUAGE_TCHINESE Chinese (Traditional)
XC_LANGUAGE_PORTUGUESE Portuguese

To specify the language currently in use by the Xbox console (in other words, the language set by the user through the Dashboard), pass the return value of XGetLanguage as dwLanguage.

hWorkEvent
[in, optional] Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue.
phTask
[out] Pointer to a variable of type XONLINETASK_HANDLE that receives a task handle representing the verify task. Pump this task to completion using XOnlineTaskContinue. When the task is complete, call XOnlineTaskClose to close the task.

Return Values

This function returns S_OK when the string verification task has started successfully.

XOnlineTaskContinue will return XONLINETASK_S_SUCCESS if all of the strings pass the vetting algorithm.

Returns S_FALSE if one or more of the strings fail. To determine which string(s) failed, call XOnlineStringVerifyGetResults.

Remarks

XOnlineStringVerify can check strings up to 256 characters in length. A maximum of four strings may be checked at once.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
