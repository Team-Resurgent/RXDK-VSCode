XNetRandom

Generates a specified number of bytes of random data.

INT XNetRandom(
    BYTE *pb,
    UINT cb
);

Parameters

pb
Address of a value that receives the random data generated.
cb
Number of bytes to generate.

Return Values

Returns zero if successful, an error code otherwise.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.
