IDirectMusicSegment8::SetWavePlaybackParams

Sets the wave playback settings for the segment.

HRESULT SetWavePlaybackParams(
    DWORD dwFlags,
    DWORD dwReadAhead
);

Parameters

dwFlags
Value specifying the flags set for this method. Can be a combination of the following values:
Value Description
DMUS_WAVEF_OFF Not supported.
DMUS_WAVEF_STREAMING Indicates that this wave file is a stream. Use this flag to avoid shutting off streaming on a wave. Omission of this flag indicates a buffer.
DMUS_WAVEF_NOINVALIDATE Indicates that the wave should be allowed to continue playing after a call made to IDirectMusicPerformance8::Invalidate. Omission indicates that the wave should stop playing on a call made to IDirectMusicPerformance8::Invalidate.
DMUS_WAVEF_NOPREROLL Indicates that the contents of the wave should not be pre-rolled. Omission of this flag indicates that content should be pre-rolled.
DMUS_WAVEF_IGNORELOOPS Indicates that the wave should ignore segment loops. Omission of this flags indicates that this wave will retrigger every time the segment loops (in other words, looping waves will keep looping when the segment loops).
dwReadAhead
Value that specifies the amount of audio data (in milliseconds) to read ahead. Must be greater than or equal to DMUS_READAHEAD_MIN.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method is used to adjust the source of a segment, not the segment that is currently playing. Once the segment starts playing, none of the flags supported by dwFlags can be used.

Calling SetWavePlaybackParams is particularly helpful for a wave file segment which does not contain wave header data in its source. For more information, see IDirectMusicPerformance8::PlaySegmentEx.
