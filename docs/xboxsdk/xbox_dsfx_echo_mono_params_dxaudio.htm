DSFX_ECHO_MONO_PARAMS

Contains the parameters used by the mono echo effect.

typedef struct _DSFX_ECHO_MONO_PARAMS{
    DSFX_ECHO_MONO_STATE    State;
    DWORD                   dwGain;
} DSFX_ECHO_MONO_PARAMS, *LPDSFX_ECHO_MONO_PARAMS;

Members

State
DSFX_ECHO_MONO_STATE structure that contains the state information for the effect.
dwGain
Value that specifies the gain for the effect. The higher the gain, the more wet gets into the signal. Can be a value between 0 and 2^23 (which is 0.0 to 1.0 in 24 bit).
