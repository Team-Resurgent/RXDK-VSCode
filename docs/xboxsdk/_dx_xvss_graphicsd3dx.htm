xvss

Provides a method to specify the type and version of the Xbox vertex state shader code.

xvss.mainVer.subVer

Registers

mainVer
Main version number of the Xbox vertex state shader.
subVer
Sub version number of the Xbox vertex state shader.

Remarks

An Xbox vertex state shader is like a normal Xbox vertex shader, except that:

  • It is invoked using the IDirect3DDevice8::RunVertexStateShader method.
  • It can read from at most one input register, v0
  • It cannot write to any output register.
  • It can write to constant registers. The values of the constant registers are preserved across invocations of the shader.
  • It runs roughly six times slower than a normal Xbox vertex shader.

The following example illustrates how the xvss instruction might be used.

xvss.1.1  ; Xbox vertex state shader version 1.1.
dp3 r0.w,v0,v0 ;Normalize input vector and store in c[20]
rsq r0.w,r0.w
mul c[20],v0,r0.w

This instruction is required to be at the beginning of all Xbox vertex state shaders.
