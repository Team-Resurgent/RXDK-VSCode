DmSetInitialBreakpoint

Sets a breakpoint in an Xbox title so that a DM_BREAK event occurs before the first thread begins execution.

DMHRAPI DmSetInitialBreakpoint(void);

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

This function is meaningful only if the console was rebooted with the DMBOOT_WAIT flag. When the console is rebooted with DMBOOT_WAIT, it will wait 15 seconds after rebooting before launching a title. During this 15-second period (and only during this period), calling DmSetInitialBreakpoint will set the initial breakpoint.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
