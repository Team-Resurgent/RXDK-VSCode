Required Attributes

stage

Index of the texture stage. This value must be between 0 and 3.

map

Specifies the name of the material type in Deep Exploration that defines the texture. This value is set by Deep Exploration and is used for reimportation of the XDX file. It is ignored by the Xbox Resource Compiler.

texture

Specifies the texture to use for the current stage. This value must correspond to the id attribute of a previously defined <texture>.
