ILiveEngine::SetProperty

Sets the current value of an ILiveEngine property.

HRESULT SetProperty(
    UIX_PROPERTY_TYPE Property,
    DWORD Value
);

Parameters

Property
[in] UIX_PROPERTY_TYPE value specifying the property to set.
Value
[in] DWORD specifying the new property value.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Current property values are retrieved with GetProperty.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
