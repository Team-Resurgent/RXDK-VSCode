Image Builder Warning IM1030

this image may not be accepted for certification

There is a problem in the .xbe file that may prevent it from passing one or more of the Xbox Technical Certification Requirements (TCRs). Your game probably links to a library that is not approved for release. At certification, your game must link only with approved libraries. Approved libraries are defined on a month-by-month basis and are documented in each month's XDK. In general, release versions of the Xbox Title Libraries are approved. Debug versions, special debug libraries, and beta or preview libraries of new functionality are unapproved.

To disable warnings about unapproved libraries, use the /NOLIBWARN property.
