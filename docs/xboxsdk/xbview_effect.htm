Effect (XbView)

The Effect structure encapsulates all of the data needed to render the mesh.

struct Effect 
{
	CHAR  Identifier[EFFECT_IDENTIFIER_SIZE];
	DWORD Flags;	
	DWORD RenderTargetResource;
	DWORD PassCount;
	Pass *rPass;
};

Identifier is a string that identifies the mesh and allows it to be attached to a frame. The frame contains the world matrix for the mesh used to render it. Flags is not used at present; it was designed to allow the mesh to be hidden, instead of rendered. RenderTargetResource is also not used at present. PassCount and rPass specify the number and type of each pass needed to render this effect.
