IDirect3DDevice8::SetDepthClipPlanes

Sets the near and far clip-plane depths used for both the fixed function and programmable pipelines without having to use IDirect3DDevice8::SetTransform.

void SetDepthClipPlanes(
    float Near,
    float Far,
    DWORD Flags
);

Parameters

Near
[in] Value that specifies the near clipping plane.
Far
[in] Value that specifies the far clipping plane.
Flags

[in] Member of the D3DSET_DEPTH_CLIP_PLANES_FLAGS enumerated type. Direct3D keeps track of one set of depth-clipping planes for use when the fixed-function pipeline is active and another set of depth-clipping planes that are used when a vertex shader program is active. Flags is used to determine which set of clipping planes will get set by SetDepthClipPlanes.

Values passed in to Near and Far will be ignored if Flags is either D3DSDCP_USE_DEFAULT_VERTEXPROGRAM_PLANES or D3DSDCP_USE_DEFAULT_FIXEDFUNCTION_PLANES.

Return Values

None.

Remarks

Using code similar to the code below, Direct3D normally sets the values of near and far depth-clipping planes when a projection matrix is set (by calling IDirect3DDevice8::SetTransform).

FLOAT fAspectRatio = 4.0f / 3.0f;
FLOAT fNear = 1.0f;      // Value of near depth-clipping plane.
FLOAT fFar  = 1000.0f;   // Value of far depth-clipping plane.
XGMATRIX matProj;

XGMatrixPerspectiveFovLH( &matProj, XG_PI/4, fAspectRatio, fNear, fFar );
pDevice->SetTransform( D3DTS_PROJECTION, &matProj );

Programmable vertex shaders may want to take advantage of the near and far depth-clipping planes without having to call IDirect3DDevice8::SetTransform. Those titles should call SetDepthClipPlanes.

If this function is not used to set up the default depth-clipping planes, Direct3D will calculate those planes according to the code below:

// If the application is using a 24-bit depth buffer then m_ZScale 
// is 2^24 and is 2^16 for 16-bit depth buffers.
//
// Assuming that pMat points to the projection matrix, then
//    m_WNear = pMat->_44 - pMat->_43 / pMat->_33 * pMat->_34;
//    m_WFar = ( pMat->_44 - pMat->_43 ) / ( pMat->_33 - pMat->_34 ) * pMat->_34 + pMat->_44;
//    m_InverseWFar = 1.0f / m_WFar;

if (bVertexShaderProgramActive)
{
   if ( D3D__RenderState[D3DRS_ZENABLE] == D3DZB_USEW )
   {
      // w-buffering
      pDevice->fNear = pDevice->m_ZScale * pDevice->m_WNear * pDevice->m_InverseWFar;
      pDevice->fFar = pDevice->m_ZScale;
   }
   else
   {
      // z-buffering
      pDevice->fNear = 0.0f;
      pDevice->fFar = pDevice->m_ZScale;
   }
}
else
{
   // Fixed-function-pipeline case:
   if ( D3D__RenderState[D3DRS_ZENABLE] == D3DZB_USEW )
   {
      pDevice->fNear = pDevice->m_ZScale * pDevice->m_WNear * pDevice->m_InverseWFar;
      pDevice->fFar = pDevice->m_ZScale;
   }
   else
   {
      pDevice->fNear = pDevice->m_ZScale * pDevice->m_Viewport.MinZ;
      pDevice->fFar = pDevice->m_ZScale * pDevice->m_Viewport.MaxZ;
   }
}
