Scene::Draw (XbView)

Scene::Draw handles drawing the scene. It takes two parameters. This first is an index into the list of skeletons of which skeleton to draw. If this index is 0xffffffff, the function draws all of the skeletons. Again, the function insures that trying to draw an index outside of the list of skeletons simply does nothing. By default, the first parameter is 0xffffffff. The second parameter is a DWORD containing a number of flags that instruct the application on how to draw the skeletons. The flags are defined as follows in scene.h:

Value Description
SCENE_SET_VERTEXSHADER This flag sets the vertex shader to the vertex shader defined in the pass for this model. If this flag is not set, the vertex shader and vertex shader constants are not set for this model from the pass information in the resources. This enables end users to specify their own shader, if desired.
SCENE_SET_PIXELSHADER This flag sets the pixel shader to the pixel shader defined in the pass for this model. If this flag is not set, the pixel shader and pixel shader constants are not set for this model from the pass information in the resources.
SCENE_SET_TEXTURES This flag sets the textures to those defined in the pass for this model. If this flag is not set, the texture is not set for this model from the pass information in the resources.
SCENE_SET_RENDERSTATES This flag sets the render states to thosee defined in the pass for this model. If this flag is not set, the render states are not set for this model from the pass information in the resources.
SCENE_SET_TEXTURESTATES This flag sets the textureStageStates to those defined in the pass for this model. If this flag is not set, the textureStageStates are not set for this model from the pass information in the resources.
SCENE_SET_ALL This flag provides a simple way to set all of the above to values in the resources to the values set in the pass for this model.

Scene::Draw then calls each of the BaseSkeletonNode::Draw calls based on the first parameter passed in. If this parameter was 0xffffffff, all of the Draw calls for all of the skeletons on the list are called. The flags passed in are passed down to the BaseSkeletonNode::Draw functions.
