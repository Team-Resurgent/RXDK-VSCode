Creating a DSP Effects Image

DSP effects images can be downloaded to the global processor in the Xbox Media Communications Processor (MCP) and used to perform various audio effects processing. There are two tools available for creating DSP effects, DSPBuilder and XGPImage. DSPBuilder is a graphical user interface tool that allows you to build DSP effects image files. XGPImage is a command line tool that allows you to manually build your DSP images. See the following sections for further information:

  • XGP Image
