Statistics Functions

The Xbox Live statistics functions include the following.

Function Description
XClearStatStore Clears the statistics that are stored for the specified player on the local Xbox hard disk.
XOnlineStatLeaderEnumerate Retrieves a contiguous set of players or Xbox Live teams and their stats from a specified leaderboard (scoreboard).
XOnlineStatLeaderEnumerateGetResults Retrieves leaderboard entries requested by the XOnlineStatLeaderEnumerate function.
XOnlineStatRead Initiates a task to retrieve statistics for the specified players or Xbox Live teams.
XOnlineStatReadGetResult Retrieves the statistics data requested by calling XOnlineStatRead.
XOnlineStatReset Resets all statistical information for the specified player on the specified leaderboard.
XOnlineStatUnitEnumerate Not supported.
XOnlineStatUnitEnumerateGetResults Not supported.
XOnlineStatUnitRead Not supported.
XOnlineStatUnitReadGetResult Not supported.
XOnlineStatWrite Sets attributes and ratings for one or more specified players.
XOnlineStatWriteEx Sets statistical information for one or more specified players or Xbox Live teams.
XOnlineStatWriteGetResult Retrieves a handle to a remote file reference. Use this handle to upload file attachments to accompany player statistics written with the XOnlineStatWrite function.
XWriteStatStore Stores the specified statistic for the specified player on the local Xbox hard disk. This statistic is later stored on the specified Xbox Live leaderboard.
