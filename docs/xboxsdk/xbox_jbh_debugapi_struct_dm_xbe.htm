DM_XBE

The DM_XBE structure is used by the DmGetXbeInfo function.

typedef struct _DM_XBE {
    char LaunchPath[MAX_PATH+1];
    DWORD TimeStamp;
    DWORD CheckSum;
    DWORD StackSize;
} DM_XBE, *PDM_XBE;

Members

LaunchPath
Path from which the XBE was launched.
TimeStamp
Time and date that the XBE was generated.
CheckSum
Checksum of the XBE.
StackSize
Size of the stack for the XBE.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
