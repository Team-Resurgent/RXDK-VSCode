XMVVIDEO_DESC

Describes the general attributes of an Xbox Media Video (XMV) file.

typedef struct _XMVVIDEO_DESC{
    DWORD Width;
    DWORD Height;
    DWORD FramesPerSecond;
    DWORD AudioStreamCount;
} XMVVIDEO_DESC;

Members

Width
Width of the video. The surface that each frame is rendered onto must be this width.
Height
Height of the video. The surface that each frame is rendered onto must be this height.
FramesPerSecond
Frame rate of the file.
AudioStreamCount
Number of the audio stream encoded in the file.

Requirements

Header: Declared in Xmv.h.
