IXACTEngine::SetListenerOrientation

Sets the XACT Engine 3D Listener Parameters, used to update the position and orientation of the 3D listener.

HRESULT SetListenerOrientation(
    FLOAT xFront,
    FLOAT yFront,
    FLOAT zFront,
    FLOAT xTop,
    FLOAT yTop,
    FLOAT zTop,
    DWORD dwApply
);

Parameters

xFront
[in] Value that specifies the coordinates of the front orientation vector along the x-axis.
yFront
[in] Value that specifies the coordinates of the front orientation vector along the y-axis.
zFront
[in] Value that specifies the coordinates of the front orientation vector along the z-axis.
xTop
[in] Value that specifies the coordinates of the top orientation vector along the x-axis.
yTop
[in] Value that specifies the coordinates of the top orientation vector along the y-axis.
zTop
[in] Value that specifies the coordinates of the top orientation vector along the z-axis.
dwApply
[in] Value that specifies when the setting should be applied. Can be one of the following values:
Value Description
DS3D_DEFERRED Settings are not applied until the application calls the CommitDeferredSettings method. This allows the application to change several settings and generate a single recalculation.
DS3D_IMMEDIATE Settings are applied immediately, causing the system to recalculate the 3D coordinates for all 3D sound buffers.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

This function is identical to IDirectSound8::SetOrientation. See this section for additional information on setting listener properties.

Requirements
