PFNXMVRELEASEPREVIOUSPACKET

Used in XMVDecoder_CreateDecoderForPackets to supply the decoder with packets.

typedef HRESULT (CALLBACK *PFNXMVRELEASEPREVIOUSPACKET)(
    DWORD Context,
    LONGLONG NextReadByteOffset,
    DWORD NextPacketSize
);

Parameters

Context
[in] Context passed to the callback.
NextReadByteOffset
[in] Byte offset to the next packet.
NextPacketSize
[in] Size of the next packet.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Assuming that Context is a pointer to a decoder object, a function that implements this callback should follow the pseudocode below:

if (NextPacketSize)
{
    Context->LoadingPacket = Context->PendingPacket;
    Context->PendingPacket = NULL;

    LoadNextPacketFromOffset(Context->LoadingPacket, NextReadByteOffset, NextPacketSize);
}
