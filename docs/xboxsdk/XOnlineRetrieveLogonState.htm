XOnlineRetrieveLogonState

Retrieves logon state information from an XONLINE_LOGON_STATE structure obtained with a previous call to XOnlineSaveLogonState. This information can be used to perform a logon with the same players and services that were logged on at the time XOnlineSaveLogonState was called.

HRESULT XOnlineRetrieveLogonState(
    const XONLINE_LOGON_STATE *pLogonState,
    PXONLINE_USER pUsers,
    DWORD *pdwServiceIDs,
    DWORD *pdwServices
);

Parameters

pLogonState
[in] Pointer to an XONLINE_LOGON_STATE structure that specifies the logon state information previously obtained with a call to XOnlineSaveLogonState.
pUsers
[out] Pointer to an array of XONLINE_USER structures that receive the player logon states as extracted from the pLogonState data. The array must be able to hold at least XONLINE_MAX_LOGON_USERS structures of type XONLINE_USER.
pdwServiceIDs
[out] Pointer to an array of DWORD values that receive the service identifiers as extracted from the pLogonState data. This represents the services that were logged onto when the logon state was saved. The array must be able to hold at least XONLINE_MAX_LOGON_STATE_SERVICES values of type DWORD.
pdwServices
[in, out] Pointer to a variable of type DWORD that specifies the number of DWORD values that can be stored in the pdwServiceIDs buffer. If this value is not large enough to hold the required number of service identifiers, the function will fail and return E_INVALIDARG. Upon a successful return, pdwServices contains the actual number of service identifiers written to pdwServiceIDs.

Return Values

If the function is successful, the return value is S_OK.

If the value specified in pdwServices is less than the required number of service identifiers, the return value is E_INVALIDARG.

Remarks

XOnlineRetrieveLogonState extracts the data returned by a previous XOnlineSaveLogonState call. The returned pUsers and pdwServiceIDs arrays can be used in an XOnlineLogon call to attempt to logon onto Xbox Live using the same players and services that were logged on when XOnlineSaveLogonState was originally called.

To persist logon information through a reboot, the game title calls XOnlineSaveLogonState before the reboot. The game title then places the returned XONLINE_LOGON_STATE somewhere in the LAUNCH_DATA before calling XLaunchNewImage. After the reboot, the game title extracts the XONLINE_LOGON_STATE from the launch data, and then calls XOnlineRetrieveLogonState to extract all the parameters needed to call XOnlineLogon again.

Requirements

Import Library: Use Xonline.lib.
