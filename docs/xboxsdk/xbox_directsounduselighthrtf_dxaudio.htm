DirectSoundUseLightHRTF

Enables the azimuth-only version of the 3D filter table.

DirectSoundUseLightHRTF();

Parameters

None.

Return Values

None.

Remarks

All titles using 3D sound must first call either DirectSoundUseFullHRTF or DirectSoundUseLightHRTF before creating a 3D buffer.

The DirectSoundUseLightHRTF version of the 3D filter tables adds 4544 bytes of code and data to your XBE.
