XOnlineTaskClose

Closes an asynchronous task and frees all associated resources.

HRESULT XOnlineTaskClose(
    XONLINETASK_HANDLE hTask
);

Parameters

hTask
[in] Handle of the asynchronous task to be closed.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

After a task handle is closed, the handle becomes invalid and should not be used in subsequent function calls.

Typically, a task should not be closed until it has reached completion. Some tasks, however, never complete and can be safely closed at any time.

If a request needs to be terminated (that is, if the user changes his or her mind and chooses to back out before some information has been retrieved), XOnlineTaskClose can be used to terminate the task.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
