ILivePlayersList

Titles use the methods of the ILivePlayersList interface to add and remove players in the list displayed by the players feature.

The ILivePlayersList interface is obtained by calling the ILiveEngine::GetFeatureInterface method with the UIX_PLAYERS_FEATURE value.

Method Description
ClearDepartedPlayersList Clears the list of departed players.
Refresh Refreshes the players list displayed by the players feature.
RegisterPlayer Adds a player to the list displayed by the players feature.
SetFilterFlags Sets flags that filter which players are shown on the players list.
UnregisterPlayer Removes a player from the current players list.

Requirements

Header: Declared in Uix.h.
