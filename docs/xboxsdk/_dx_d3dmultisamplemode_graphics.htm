D3DMULTISAMPLEMODE

Used for multisampling, it determines sampling size.

typedef enum _D3DMULTISAMPLEMODE
{
    D3DMULTISAMPLEMODE_1X                                   = 0,
    D3DMULTISAMPLEMODE_2X                                   = 1,
    D3DMULTISAMPLEMODE_4X                                   = 2,

    D3DMULTISAMPLEMODE_FORCE_DWORD                          = 0x7fffffff
} D3DMULTISAMPLETYPE;

Constants

D3DMULTISAMPLEMODE_1X
The sampling size is one. A sampling size of one means that supersampling and not multisampling will be used for any antialiasing.
D3DMULTISAMPLEMODE_2X
The sampling size is two. 2×multisampling only make sense when combined with either a linear or quincunx filter. For more information, see Antialiasing Filters.
D3DMULTISAMPLEMODE_4X
The sampling size is four.
D3DMULTISAMPLEMODE_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.

Remarks

The values in this enumerated type are used by the D3DRS_MULTISAMPLEMODE and D3DRS_MULTISAMPLERENDERTARGETMODE render states.
