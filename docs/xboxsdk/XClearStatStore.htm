XClearStatStore

Clears the statistics that are stored for the specified player on the local Xbox hard disk.

DWORD XClearStatStore(
    LPCWSTR lpLocalUserName
);

Parameters

lpLocalUserName
[in] Pointer to a Unicode string that contains the title-defined identifier for the player. This string is not the player's gamertag on the Xbox Live service.

Return Values

Returns ERROR_SUCCESS if successful, or an error code otherwise.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
