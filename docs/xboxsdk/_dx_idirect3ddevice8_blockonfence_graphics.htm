IDirect3DDevice8::BlockOnFence

Blocks the current CPU thread until a specific fence has been crossed.

VOID BlockOnFence(
  DWORD Fence
);

Parameters

Fence
[in] Handle of a valid fence. Use IDirect3DDevice8::InsertFence to insert a fence and get a handle.

Remarks

Event-based blocks are generally done so that other non-rendering threads can run while the current CPU thread is waiting.
