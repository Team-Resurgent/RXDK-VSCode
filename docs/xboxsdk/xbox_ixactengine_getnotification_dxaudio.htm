IXACTEngine::GetNotification

Gets a particular notification from the IXACTEngine.

HRESULT GetNotification(
    PCXACT_NOTIFICATION_DESCRIPTION pNotificationDesc,
    PXACT_NOTIFICATION pNotification
);

Parameters

pNotificationDesc
[in] A pointer to an XACT_NOTIFICATION_DESCRIPTION structure describing the notification to get. If this parameter is NULL, the pNotification parameter returns the next pending notification.

Note    The pSoundCue parameter of the XACT_NOTIFICATION_DESCRIPTION structure will be set to the instance pointer of a sound cue, if the notification was registered on a sound cue index.

pNotification
[out] A pointer to the XACT_NOTIFICATION structure returned.

Return Values

Returns DS_OK if successful; an error code otherwise.

Remarks

Once notifications have been registered using IXACTEngine::RegisterNotification, they can be retrieved using GetNotification.

It is important to note when to expect certain notifications:

  • A start notification is sent at the time stamp of the first play event on a track. It will trigger on the first play event of any track, where multiple tracks are involved.
  • A stop notification is sent at the time stamp of the last stop event, when the sound ends normally, or when IXACTSoundBank::Stop is called—whichever comes first.

If the XACT_FLAG_NOTIFICATION_SOUNDCUE_DESTROYED flag is set in the returned XACT_NOTIFICATION_DESCRIPTION structure, the cue has been freed and no calls can be made using the returned pSoundCue pointer.

The following example illustrates how to get start notifications from a sound bank using a particular cue index:

// This code assumes that g_pXACTEngine is a valid pointer to the
// XACT engine and pSoundBank is a valid pointer to an IXACTSoundBank.

XACT_NOTIFICATION_DESCRIPTION Desc;
XACT_NOTIFICATION Notify;
DWORD dwSoundCueIndex;

memset(&Notify, 0, sizeof(Notify));
pSoundBank->GetSoundCueIndexFromFriendlyName("CUE0", &dwSoundCueIndex);

memset(&Desc, 0, sizeof(Desc));
Desc.wFlags = XACT_FLAG_NOTIFICATION_USE_SOUNDCUE_INDEX;
Desc.dwSoundCueIndex = dwSoundCueIndex;
Desc.u.pSoundBank = pSoundBank;
Desc.wType = eXACTNotification_Start;

if (g_pXACTEngine->GetNotification(&Desc, &Notify) == S_OK)
{
	// Start notification was received
}

To clear pending notifications, call IXACTEngine::FlushNotification.
