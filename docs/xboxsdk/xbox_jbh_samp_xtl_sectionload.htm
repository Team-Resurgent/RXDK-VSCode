SectionLoad Sample

The SectionLoad sample illustrates loading and unloading of sections.

Overview

Section loading and unloading is something that will be used primarily in large projects in which memory restrictions prevent you from having all code and data loaded simultaneously. Furthermore, flagging a section as NOPRELOAD works only if that section cannot be squeezed into a preloaded section.

This sample cannot be written as a couple of small routines that are loaded and unloaded. The routines get squeezed into the end of a preloaded section, and the code always executes (since the routines are always in memory).

Therefore, this sample indicates that section loading and unloading work properly by doing the following:

  • If the data section is loaded, write access to the data area will work fine and the sample displays the time of the last successful access.
  • If the data section is not loaded, accessing the data area will generate an exception.

Note   You should only run this in DEBUG mode so that you can see what is going on.
