XCalculateSignatureGetSize

Determines the size of the signature buffer passed to XCalculateSignatureEnd.

DWORD XCalculateSignatureGetSize(
    DWORD dwFlags
);

Parameters

dwFlags

[in] The signature type. Must match the dwFlags value passed to XCalculateSignatureBegin when the signature context is created. Can be one of the following values.
Value Description
XCALCSIG_FLAG_SAVE_GAME Save game signature.
XCALCSIG_FLAG_NON_ROAMABLE Non-roamable save game signature.
XCALCSIG_FLAG_CONTENT Content signature.
XCALCSIG_FLAG_DIGEST Digest for use in an Xbox Live signature.
XCALCSIG_FLAG_ONLINE Xbox Live signature.

Return Values

The return value is the size, in bytes, of the signature returned by XCalculateSignatureEnd.

Remarks

This function determines the size of the signature calculated by XCalculateSignatureBegin, XCalculateSignatureUpdate, and finally returned by XCalculateSignatureEnd. Allocate an appropriately sized buffer and pass a pointer to that buffer in the pSignature parameter of XCalculateSignatureEnd.

Requirements
