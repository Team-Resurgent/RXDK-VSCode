IDirect3DDevice8::End

Indicates the end of a vertex block that was started by IDirect3DDevice8::Begin.

HRESULT End();

Parameters

None.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

Where vertex throughput is not a critical, a Begin/End block can be employed instead of a vertex buffer. Using a Begin/End block, it is possible to specify a vertex and its attributes and then send it on immediately to the graphics processing unit (GPU) for processing. It is possible to work with as many vertices as is needed. See Xbox Begin/End Blocks for more information.
