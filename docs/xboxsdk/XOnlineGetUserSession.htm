XOnlineGetUserSession

Retrieves the session information for a user.

HRESULT XOnlineGetUserSession(
    DWORD dwUserIndex,
    XUID xuidPeer,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask,
    PXONLINE_PEER_SESSION_RESULTS pResults
);

Parameters

dwUserIndex

[in] The controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.

xuidPeer

[in] XUID of the player for whom to obtain session information.

hWorkEvent

[in, optional] Handle to an event object, created by the title. This event is signaled when there is work to do on the asynchronous task that retrieves the session information. Perform work on the task by calling XOnlineTaskContinue with the handle returned in the phTask parameter. This parameter is optional and may be NULL. If this parameter is NULL, the title should call XOnlineTaskContinue periodically to perform work on the task.

phTask

[out] Pointer to a variable of type XONLINETASK_HANDLE that receives a handle to the task that retrieves the session information. Pump this task to completion using XOnlineTaskContinue. When the task completes, XOnlineTaskContinue returns XONLINETASK_S_SUCCESS. After the task completes, call XOnlineTaskClose to close the task.

pResults

[out] Pointer to an XONLINE_PEER_SESSION_RESULTS structure that receives the session information for the user specified by the xuidPeer parameter.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Use the XOnlineGetUserSession function to obtain the session information for an Xbox Live game session.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
