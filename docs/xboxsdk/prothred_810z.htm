Thread Times

The GetThreadTimes function obtains timing information for a thread. It returns the thread creation time and how much time the thread has been executing in kernel mode. The reported user mode execution time will always be zero, as all Xbox threads run in kernel mode. These times do not include time spent executing system threads or waiting in a suspended or blocked state. If the thread has exited, GetThreadTimes returns the thread exit time.
