IXACTEngine::SetListenerPosition

Sets the listener's position, in distance units. By default, these units are meters (however, the distance units can be changed by calling the SetDistanceFactor method).

HRESULT SetListenerPosition(
    FLOAT x,
    FLOAT y,
    FLOAT z,
    DWORD dwApply
);

Parameters

x
[in] Value that specifies the listener's position vector along the x-axis. Note that this value may be adjusted to prevent floating-point overflow.
y
[in] Value that specifies the listener's position vector along the y-axis. Note that this value may be adjusted to prevent floating-point overflow.
z
[in] Value that specifies the listener's position vector along the z-axis. Note that this value may be adjusted to prevent floating-point overflow.
dwApply
[in] Value that specifies when the setting should be applied. Can be one of the following values:
Value Description
DS3D_DEFERRED Settings are not applied until the application calls the CommitDeferredSettings method. This allows the application to change several settings and generate a single recalculation.
DS3D_IMMEDIATE Settings are applied immediately, causing the system to recalculate the 3D coordinates for all 3D sound buffers.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

This function is identical to IDirectSound8::SetPosition. See this section for additional information on setting listener properties.
