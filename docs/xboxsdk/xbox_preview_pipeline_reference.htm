Xbox Preview Pipeline Reference

The Xbox Preview Pipeline is designed to reduce artist's iteration cycle times and the number of iterations, and to decrease the error rate for all the art assets. In addition, it gives game developers an initial optimized pipeline when starting a new project, among many other benefits.

The Preview Pipeline has the following components:

  • Deep Exploration allows artists to browse art assets and edit material. By pushing a single button, artists can view their assets on the Xbox without developer intervention.
  • Xbox Resource Viewer (XbView) reads Xbox binary resource files (.XBR) and displays the graphical content (including models, textures, and animations) on the Xbox as it would appear in the game engine.
  • Xbox Resource Compiler (XbRC) compiles XDX files (.xdx) into Xbox-optimized binaries (.xbr files.)
  • XDX Reference documents XDX, an XML human-readable file format used for storing graphics data.

The source code for all components of the Preview Pipeline (except Deep Exploration) is available so that developers can incorporate the Preview Pipline into their own art pipeline, tools, or game engine. The preview pipeline is fully customizable, allowing artists and developers to take advantage of new data types, resources, and formats.

To obtain a copy of the source code, please follow the instructions found at https://xds.xbox.com/BPProgInfo.asp?page=content/source.htm. Because this code is on the developer support site, you will be prompted for your user name and password.
