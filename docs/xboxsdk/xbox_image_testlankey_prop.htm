/TESTLANKEY (LAN Key)

/TESTLANKEY:key

Where:

key
Specifies the LAN key for this image.

Remarks

LAN keys are 128-bit numberS entered as 32 hexadecimal characters without the leading "0x".

Example:

123456789A123456789B123456789C12

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the Certificate property page.
  4. Modify the LAN Key property.
