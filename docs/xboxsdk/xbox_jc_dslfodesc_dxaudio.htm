DSLFODESC

Contains parameters for the LFO's (Low Frequency Oscillators).

typedef struct _DSLFODESC{
    DWORD dwLFO;
    DWORD dwDelay;
    DWORD dwDelta;
    LONG lPitchModulation;
    LONG lFilterCutOffRange;
    LONG lAmplitudeModulation;
} DSLFODESC, *LPDSLFODESC;

Members

dwLFO
Low Frequency Oscillator to set data on. Valid values are:
Value Description
DSLFO_PITCH Pitch envelope.
DSLFO_MULTI Multi function envelope.
dwDelay
Delay before the LFO begins, in 32-sample blocks. Can be a value from zero to 65535. (that is, a value of 1 indicates a delay of 32/48000 s)
dwDelta
LFO delta. This delta is added to the LFO value each sample frame. Can be a value from zero to 1023. If the value is 0, the LFO generator is turned off. The frequency of the LFO is calculated by: LFO Frequency = (1500/4) * (dwDelta/2^14)
lPitchModulation
Pitch LFO frequency modulation amount in s0.7 format in octaves. Can be a value from -128 to 127. A value of 0 indicates there should be no frequency modulation; a value of 127 indicates just under one octave of pitch modulation.
lFilterCutOffRange
Frequency cutoff range for multifunction LFO in s3.4 format in octaves. Can be a value from -128 to 127. A value of -128 indicates that the cutoff frequency be modulated by -8 octaves; a value of 127 indicates a modulation of just under +8 octaves
lAmplitudeModulation
Amplitude modulation amount for multifunction LFO in s4.4 format, in dB. Can be a value from -128 to 128. A value of 0 indicates that there should be no amplitude modulation.
