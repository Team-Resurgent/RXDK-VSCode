UIXFriends Sample

The UIXFriends sample shows how to implement Xbox Live friends using the UIX library to handle most of the work.

Overview

Run copytoxb.bat to copy the required fonts and skin files to the Xbox prior to running this sample.

UIXFriends is designed to be a simple demonstration of how to implement Xbox Live friends using the UIX engine. Requesting that somebody be your friend is not supported because this is a feature that is trivial when you have finished a game, but difficult or artifical otherwise. Thus, this sample demonstrates how to manage your friends list, including sending and receiving invites, and receiving friend requests.
