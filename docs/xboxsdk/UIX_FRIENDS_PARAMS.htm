UIX_FRIENDS_PARAMS

Initialization parameters for friends feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_FRIENDS_FEATURE.

typedef struct _UIX_FRIENDS_PARAMS{
    SIZE_T StructSize;
    DWORD UserPort;
    XUID SelectedFriendXUID;
    BOOL SignOutEnabled;
} UIX_FRIENDS_PARAMS, *PUIX_FRIENDS_PARAMS;

Members

StructSize
Size of the structure. Must be set to the structure size before calling ILiveEngine::StartFeature.
UserPort
Controller port index (zero-based) of the player for which a friends list will be displayed.
SelectedFriendXUID
XUID of the player to initially select in the friends list. If a player with the specified XUID is in the friends list, that player will be highlighted when the list is created. Specify zero if no such selection is desired.
SignOutEnabled
Determines whether the friends list screen displays a Sign Out button, allowing the player to log off from the Xbox Live service from within the friends list display. Specify TRUE to enable sign out, FALSE otherwise.

Remarks

This structure must be initialized to zero and StructSize set to the size of the structure before calling ILiveEngine::StartFeature.
