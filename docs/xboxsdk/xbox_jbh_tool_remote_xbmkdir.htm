Remote Make Directory Tool (XbMkDir)

The Remote Make Directory tool (XbMkDir) creates a directory on the Xbox target system's file system.

USAGE: xbmkdir [-x xboxname] directory
OptionDescription
-x Xbox name or Xbox IP. Specify a particular target machine.
-? Display a complete list of available options.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.

Known Issues

The Xbox kernel supports ANSI file names only. As a result, any attempt to create a folder with non-ANSI characters will result in an error saying it's an invalid file name.
