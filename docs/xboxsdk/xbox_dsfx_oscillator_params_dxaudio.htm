DSFX_OSCILLATOR_PARAMS

Contains the parameters used by the sinusoidal oscillator effect.

typedef struct _DSFX_OSCILLATOR_PARAMS{
    DSFX_OSCILLATOR_STATE State;
    DWORD dwOscillator_H;
    DWORD dwOscillator_Y1;
    DWORD dwOscillator_Y2;
} DSFX_OSCILLATOR_PARAMS, *LPDSFX_OSCILLATOR_PARAM;

Members

State
DSFX_OSCILLATOR_STATE structure that contains the state information for this effect.
dwOscillator_H
H parameter used to create the sine wave.
dwOscillator_Y1
Y1 parameter used to create the sine wave.
dwOscillator_Y2
Y2 parameter used to create the sine wave.

Remarks

The following equation is used to calculate the output for this effect.


  y[n] = h*y[n-1] - y[n-2]
