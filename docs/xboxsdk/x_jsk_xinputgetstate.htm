XInputGetState

The XInputGetState function is used to determine the latest known state of the specified input device.

DWORD XInputGetState(
  HANDLE hDevice,       
  PXINPUT_STATE pState  
);

Parameters

hDevice
[in] Handle to the target input device.
pState
[out] Pointer to an XINPUT_STATE structure that receives the latest known state of the input device.

Return Values

If the function succeeds, it will return ERROR_SUCCESS.

If the function fails, it will return a valid Win32 error code.

Remarks

XInputGetState is the primary function that titles use to retrieve the current state of a particular input device. By default, the state returned by XInputGetState is kept current as the device state changes. The pPollingParameters argument to XInputOpen can be used to alter this behavior.

XInputGetState returns a union containing all of the potential packets that could be returned (based on device type). The title must interpret this field based upon the type of the device opened. For more information, see XINPUT_STATE.

Requirements
