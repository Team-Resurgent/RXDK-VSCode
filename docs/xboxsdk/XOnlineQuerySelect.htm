XOnlineQuerySelect

Exceutes a custom operation on a specific entity ID. This is a low-level function. Most developers find it easier to use XOnlineCompetitionManageEntrant.

HRESULT XOnlineQuerySelect(
    DWORD dwUserIndex,
    ULONGLONG qwTeamId,
    DWORD dwDatasetId,
    XENTITY_ID entityId,
    DWORD dwAction,
    DWORD dwNumAttributes,
    const XONLINE_ATTRIBUTE *pAttributes,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE *phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.
qwTeamId
[in] Team ID. Should be specified only if the action is associated with a team. Otherwise, this value should be 0.
dwDatasetId
[in] ID of the dataset to query.
entityId
[in] Entity ID to select.
dwAction
[in] Type of action to perform on the competition's dataset. For more information, see XOnlineCompetitionManageEntrant.
dwNumAttributes
[in] Number of competition attributes in the array specified by the pAttributes parameter.
pAttributes
[in] Pointer to an array of XONLINE_ATTRIBUTE structures, each specifying the parameter ID and value for the Select action being performed.
hWorkEvent
[in, optional] Handle to an event you want to associate with this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by XOnlineQuerySelect.

Return Values

Returns S_OK if successful, or an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
