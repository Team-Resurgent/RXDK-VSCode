ILiveFriendsList::GetFriendByIndex

Retrieves detailed information for the friend at the specified index in the friends list.

HRESULT GetFriendByIndex(
    DWORD FriendIndex,
    XONLINE_FRIEND *pFriend
);

Parameters

FriendIndex
[in] Index of the friend in the friends list. FriendIndex must be between zero and Count.
pFriend
[out] Pointer to an XONLINE_FRIEND structure that receives the friend information.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

GetFriendByIndex, along with Count, is used primarily to enumerate the friends in the list.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
