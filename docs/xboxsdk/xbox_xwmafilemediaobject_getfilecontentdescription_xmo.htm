XWmaFileMediaObject::GetFileContentDescription

Retrieves the content description from the WMA file.

HRESULT GetFileContentDescription(
    WMAXMOFileContDesc *pContentDesc
);

Parameters

pContentDesc
Address of the WMAXMOFileContDesc structure that receives the content description of the WMA file.

Return Values

Returns S_OK if successful, an error code otherwise.
