IXVoiceEncoder::SetVoiceMask

Applies the specified voice mask to the specified input stream of the encoder.

HRESULT SetVoiceMask(
    DWORD dwEncoderInstanceNo,
    LPXVOICE_MASK pVoiceMask
);

Parameters

dwEncoderInstanceNo
[in] Input stream on this encoder. This parameter has a value between zero and the number of input streams minus one. For example, on a four-to-four encoder, this parameter has a value between zero and three.
pVoiceMask
[in] Pointer to an XVOICE_MASK structure that specifies the voice mask to apply to the input stream specified by the dwEncoderInstanceNo parameter. This parameter cannot be NULL; to specify no voice mask, pass an XVOICE_MASK structure in which each member has a value of XVOICE_MASK_PARAM_DISABLED.

Return Values

TBD

Remarks

There are several predefined voice mask types defined in Xvoice.h, including the parameters for the required anonymous voice mask. Games that support voice chat using the Xbox Voice Communicator should use these settings when using the anonymous voice mask.
