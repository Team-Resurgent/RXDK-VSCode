IDirectSoundBuffer8::SetRolloffCurve

Overrides the default rolloff calculations for this source with the specified curve.

HRESULT SetRolloffCurve(
    const FLOAT *pflPoints,
    DWORD dwPointCount,
    DWORD dwApply
);

Parameters

pflPoints
Array of FLOAT values that specify curve points. Values in the array must be between zero and one, where zero is silence (-100 dB) and one is full volume (0 dB). Values greater than one and less than zero are invalid.
dwPointCount
Value that specifies the number of points in the array.
dwApply
Value that specifies when the setting should be applied. This value must be one of the following:
Value Description
DS3D_DEFERRED Settings are not applied until the application calls the CommitDeferredSettings method. This allows the application to change several settings and generate a single recalculation.
DS3D_IMMEDIATE Settings are applied immediately, causing the system to recalculate the 3D coordinates for all 3D sound sources.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

SetRolloffCurve assumes that the point array passed to it is static. As a result, SetRolloffCurve does not make a copy of the point array.

Calling SetRolloffCurve with zero points and a NULL array pointer instructs the buffer to calculate distance attenuation based on the source and listener rolloff factors (instead of being based on the curve).

Calling SetRolloffCurve with one point effectively sets the volume for all distances beyond MaxDistance to a single value.

For distances less than or equal to the minimum distance, the volume is one (full volume) and no distance-based attenuation is applied. For distances greater than or equal to the maximum distance, the volume is attenuated by the value of the last element of the pflPoints array.

Note    Only rolloff factor, minimum distance, and maximum distance contribute to distance attenuation. The distance factor is a separate value used for left/right gains and Doppler calculation.

This function is only valid for buffers created using the DSBCAPS_CTRL3D flag.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
