DmGetDriveList

Retrieves the drive letters that are currently valid on the Xbox console, including mounted memory units, mounted utility drive, and mounted alternate title data.

DMHRAPI DmGetDriveList(
    LPSTR rgchDrives,
    DWORD *pcDrives
);

Parameters

rgchDrives
Pointer to a buffer that receives a string containing all currently valid drive letters.
pcDrives
Pointer to a variable that receives the count of drive letters written to rgchDrives.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If the function succeeds, the list of currently valid drive letters is written to rgchDrives, and pcDrives receives the count of drive letters written to the buffer. One character is written to rgchDrives for each drive letter; so, for example, the returned string might be "DTXUFESVCY".

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
