IDirectMusicPerformance8::CreateStandardAudioPath

Creates an object that represents the stages in data flow from the performance to DirectSound buffers. This method can be used instead of IDirectMusicPerformance8::CreateAudioPath to create a basic predefined audiopath rather than one defined in a source file.

HRESULT  CreateStandardAudioPath(
    DWORD dwType,
    DWORD dwPChannelCount,
    BOOL fActivate,
    IDirectMusicAudioPath **ppNewPath
);

Parameters

dwType
Value that specifies the type of audiopath to create. Can be one of the following values or a combination of up to eight DirectSound Mixbin Constants.
DMUS_APATH_DYNAMIC_3D
Contains a dynamic 3D buffer. Does not send to environmental reverb.
DMUS_APATH_DYNAMIC_MONO
Contains a dynamic mono buffer.
DMUS_APATH_MIXBIN_5DOT1
Sends to 5.1 mixbins on channels 1 through 6.
DMUS_APATH_MIXBIN_5DOT1_ENV
Sends to 5.1 mixbins on channels 1 through 6 and environmental reverb on 7.
DMUS_APATH_MIXBIN_5DOT1_MUSIC
Sends to 5.1 mixbins on channels 1 through 6 and music reverb and chorus on 7, 8.
DMUS_APATH_MIXBIN_QUAD
Sends to quad mixbins on channels 1 through 4.
DMUS_APATH_MIXBIN_QUAD_ENV
Sends to quad mixbins on channels 1 through 4 and environmental reverb on 5.
DMUS_APATH_MIXBIN_QUAD_MUSIC
Sends to quad mixbins on channels 1 through 4 and music reverb and chorus on 5, 6.
DMUS_APATH_SHARED_STEREO
Contains a stereo buffer for a standard music setup with stereo outs and no reverb or chorus send.
DMUS_APATH_SHARED_STEREOPLUSREVERB
Contains a stereo buffer for an ordinary music setup with stereo outs, reverb send, and chorus send.

For more information on these audiopath types, see Standard Audiopaths.

dwPChannelCount
Number of performance channels in the path.
fActivate
Boolean value that specifies whether to activate the path on creation.
ppNewPath
Address of a variable that receives an IDirectMusicAudioPath interface pointer for the audiopath. See IDirectMusicAudioPath8.

Return Values

Returns S_OK if successful, an error code otherwise. Can return one of the following error values:

E_POINTER
DMUS_E_NOT_INIT
E_INVALIDARG
E_OUTOFMEMORY

Remarks

Titles can create custom mixbin audiopaths that specify up to 8 different mixbins. For multi-channel audio content, mixbins will be assigned channels in the order specified in Setting Individual Mixbin Assignments.

The following example demonstrates how to specify a custom mixbin audiopath.

// pPerformance is a valid pointer to an IDirectMusicPerformance8 interface

HRESULT hr;
IDirectMusicAudioPath8 *pMyAudioPath;

// Create an audiopath that uses the back left and back right speakers.
hr = pPerformance->CreateStandardAudioPath(
            ((1<<DSMIXBIN_BACK_LEFT) |
             (1<<DSMIXBIN_BACK_RIGHT)),64,TRUE,&pMyAudioPath);
