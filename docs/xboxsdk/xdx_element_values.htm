values Element

The <values> element groups the data used for animation.

<values/>

Required Attributes

None.

Optional Attributes

None.

Element Information

Parent Elements rotate, scale, translate
Child Elements None.

Remarks

The nature of the data enclosed by <values> is dependent on its parent element.

Parent Element Types of Data
<translate> Groups of three space-delimited floating-point values comprising the translation vector.
<rotate> Groups of four space-delimited floating-point values comprising the rotation quaternion.
<scale> Groups of three space-delimited floating-point values comprising the x, y, and z scaling factors.

Each group of values needs to have a corresponding key time value in the sibling <keyTimes> element.

For examples on how this element is used in actual animations, see Translations With Animation, Rotations With Animation, and Scaling With Animation
