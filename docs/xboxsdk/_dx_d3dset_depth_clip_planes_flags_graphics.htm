D3DSET_DEPTH_CLIP_PLANES_FLAGS

Direct3D keeps track of one set of depth-clipping planes for use when the fixed-function pipeline is active, and keeps track of another set of depth-cliping planes for use when a vertex shader program is active. The D3DSET_DEPTH_CLIP_PLANES_FLAGS enumeration is used to determine which set of clipping planes will be set by IDirect3DDevice8::SetDepthClipPlanes.

typedef enum _D3DSET_DEPTH_CLIP_PLANES_FLAGS{
    D3DSDCP_SET_VERTEXPROGRAM_PLANES = 1,
    D3DSDCP_SET_FIXEDFUNCTION_PLANES = 2,
    D3DSDCP_USE_DEFAULT_VERTEXPROGRAM_PLANES = 3,
    D3DSDCP_USE_DEFAULT_FIXEDFUNCTION_PLANES = 4,
} D3DSET_DEPTH_CLIP_PLANES_FLAGS;

Constants

D3DSDCP_SET_VERTEXPROGRAM_PLANES
Sets the depth-clipping planes used whenever vertex shader programs are active.
D3DSDCP_SET_FIXEDFUNCTION_PLANES
Sets the depth-clipping planes used whenever the fixed-function pipeline is in use.
D3DSDCP_USE_DEFAULT_VERTEXPROGRAM_PLANES
Causes Direct3D to disregard the depth-clipping planes set when using D3DSDCP_SET_VERTEXPROGRAM_PLANE. Direct3D will resume using its own internally calculated clip planes when vertex shader programs are active.
D3DSDCP_USE_DEFAULT_FIXEDFUNCTION_PLANES
Causes Direct3D to disregard the depth-clipping planes set when using D3DSDCP_SET_FIXEDFUNCTION_PLANES. Direct3D will resume using its own internally calculated clip planes when the fixed-function pipeline is active.
