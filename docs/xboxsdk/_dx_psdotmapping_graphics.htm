PSDotMapping

Each texture stage that is set to perform a dot product must convert the 32-bit output of a texture stage to three floating-point numbers that take part in the dot product. This field specifies how this conversion is to take place. The PS_DOTMAPPING(t0,t1,t2,t3) macro allows the dot mapping mode to be set for each texture stage. The macro is defined as follows:

#define PS_DOTMAPPING(t0,t1,t2,t3) (((t3)<<8)|((t2)<<4)|(t1))

Note that even though t0 appears as an argument to the macro, it is ignored in the body of the macro because no dot product texture modes can be specified in stage 0.

The following dot mapping modes may be specified:

PS_DOTMAPPING_ZERO_TO_ONE
Treat the 32-bit output as a concatenation of four unsigned 8-bit numbers, argb. Remap the color components from [0,255] to [0.0,1.0], and use the vector (r,g,b).
PS_DOTMAPPING_MINUS1_TO_1_D3D
Treat the 32-bit output as a concatenation of four unsigned 8-bit numbers, argb. Remap the color components as follows, and use the vector (r,g,b). 0x0 maps to -128/127 (-1.007874016), 0x1 maps to -1.0, 0x80 maps to 0.0, 0xff maps to 1.0.
PS_DOTMAPPING_MINUS1_TO_1_GL
Treat the 32-bit output as a concatenation of four unsigned 8-bit numbers, argb. Remap the color components as follows, and use the vector (r,g,b). 0x80 maps to -1.0, 0x7f maps to 1.0 (2's complement).
PS_DOTMAPPING_MINUS1_TO_1
Treat the 32-bit output as a concatenation of four unsigned 8-bit numbers, argb. Remap the color components as follows and use the vector (r,g,b). 0x80 maps to -128/127, 0x7f maps to 1.0 (2's complement).
PS_DOTMAPPING_HILO_1
Treat the 32-bit output as a concatenation of two unsigned 16-bit numbers, hl. Remap the color components as follows and use the vector (h,l,1.0). 0x0 maps to 0.0, 0xffff maps to 1.0.
PS_DOTMAPPING_HILO_HEMISPHERE
Treat the 32-bit output as a concatenation of two unsigned 16-bit numbers, hl. Remap the color components as follows and use the vector (h,l,sqrt(1-h*h-l*l)). 0x8000 maps to -32768/32767, 0x8001 maps to -1.0, 0x0 maps to 0.0, 0x7fff maps to 1.0.
