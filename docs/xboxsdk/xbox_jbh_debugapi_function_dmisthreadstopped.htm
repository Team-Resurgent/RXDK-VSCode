DmIsThreadStopped

Determines whether the specified thread is stopped and, if it has, returns information about the circumstances that forced the thread to stop.

DMHRAPI DmIsThreadStopped(
  DWORD dwThreadId, 
  PDM_THREADSTOP pdmts
  );

Parameters

dwThreadId
Thread ID.
pdmts
Pointer to a structure containing the thread stop information.

Return Values

Returns XBDM_NOERR if successful, XBDM_NOTSTOPPED if the thread is not stopped, or an error value otherwise.

Remarks

If the return value is XBDM_NOTSTOPPED or an error code, the contents of pdmts are undefined.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
