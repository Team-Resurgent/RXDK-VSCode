DmRemoveUser

Enables users to be removed from the list of those authorized to establish a secure connection.

DMHRAPI DmRemoveUser(
    LPCSTR szUserName
);

Parameters

szUserName
Username to be removed from the list of authorized users.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

For users to establish secure connections, they must be added to the list of authorized users. For more information, see DmAddUser. The DmRemoveUser function provides a way to remove users programmatically from the list of authorized users.
