IDirectMusicSegmentState8::SetPitch

Adjusts the pitch of all sounds playing on this segment.

HRESULT SetVolume(
    long lPitch,
    DWORD dwDuration
);

Parameters

lPitch
Value that specifies the pitch bend, in hundredths of a semitone. Can be a value from -8192 to 8191.
dwDuration
Value that specifies the duration of pitch ramp, in milliseconds. A value of 0 ensures maximum efficiency.

Return Values

Returns S_OK if successful, an error code otherwise.
