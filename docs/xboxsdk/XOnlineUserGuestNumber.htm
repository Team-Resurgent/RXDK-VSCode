XOnlineUserGuestNumber

Macro that, given the dwUserFlags value from an XUID structure, determines the guest number (1–3) of the player.

#define XOnlineUserGuestNumber(dwUserFlags) \
   ((dwUserFlags) & XONLINE_USER_GUEST_MASK)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is the guest number (1–3) of the player. If the player is not a guest, the return value is 0.

The return value is of type DWORD.

Remarks

XOnlineUserGuestNumber accepts the entire dwUserFlags from an XUID structure and returns the guest number of the player. An Xbox Live player can have up to three guests (one for each additional controller) logged on at the same time using the regular player's credentials.

Requirements

Header: Declared in Xonline.h.
