IXHVEngine::UnregisterRemoteTalker

Unregisters a remote talker previously registered with RegisterRemoteTalker.

HRESULT UnregisterRemoteTalker(
    XUID xuidRemoteTalker
);

Parameters

xuidRemoteTalker
[in] XUID of the player to unregister.

Return Values

Returns S_OK if successful, an error code otherwise.
