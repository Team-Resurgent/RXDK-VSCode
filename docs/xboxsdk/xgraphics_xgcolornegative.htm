XGColorNegative

Creates the negative color value of a color value.

XGCOLOR* XGColorNegative(
  XGCOLOR* pOut,
  CONST XGCOLOR* pC
);

Parameters

pOut
[in, out] Pointer to a XGCOLOR structure that is the result of the operation.
pC
[in] Pointer to a source XGCOLOR structure.

Return Values

This function returns a pointer to a XGCOLOR structure that is the negative color value of the color value.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGColorNegative function can be used as a parameter for another function.

This function returns the negative color value by subtracting 1.0 from the color components of the XGCOLOR structure, as shown in the following example.

    pOut->r = 1.0f - pC->r;
