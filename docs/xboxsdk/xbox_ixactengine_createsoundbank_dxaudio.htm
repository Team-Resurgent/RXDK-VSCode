IXACTEngine::CreateSoundBank

Creates an IXACTSoundBank object.

HRESULT CreateSoundBank(
    PVOID pvData,
    DWORD dwSize,
    PXACTSOUNDBANK *ppSoundBank
);

Parameters

pvData
[in] A pointer to the sound bank data.
dwSize
[in] A value indicating the size of the sound bank data.
ppSoundBank
[out] A pointer to a returned IXACTSoundBank object.

Return Values

Returns DS_OK if successful, an error code otherwise.

Remarks

A sound bank file (.xsb) is created by the content authoring tool. The file can then be loaded directly into memory.

In instrumented builds (xactengd.lib and xactengi.lib), a sound bank is registered twice—once in main memory (below 64 MB), and once in upper memory. The latter allows the sound bank to grow as the content creator edits and adjusts content in real time. For more information about upper memory, see Xbox Development System Memory.

Requirements
