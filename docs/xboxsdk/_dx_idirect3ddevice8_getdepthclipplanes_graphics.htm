IDirect3DDevice8::GetDepthClipPlanes

Retrieves the near and far clip-plane depths used for both the fixed function and programmable pipelines.

void GetDepthClipPlanes(
    float *pNear,
    float *pFar,
    DWORD Flags
);

Parameters

pNear
[out] Pointer to a variable of type float that will receive the value of the near depth-clipping plane.
pFar
[out] Pointer to a variable of type float that will receive the value of the far depth-clipping plane.
Flags
[in] Member of the D3DGET_DEPTH_CLIP_PLANES_FLAGS enumerated type. Direct3D keeps track of one set of depth-clipping planes for use when the fixed-function pipeline is active, and another set of depth-clipping planes that are used when a vertex shader program is active. Flags is used to determine which set of clipping planes will be retrieved by GetDepthClipPlanes.

Return Values

None.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
