UIX_SKIN_ICON_INFO

Describes the layout of an icon within a text string. Used with ITitleUIPlugin::SetText.

typedef struct _UIX_SKIN_ICON_INFO{
    DWORD IconResID;
    DWORD InsertPosInText;
    DWORD Flags;
} UIX_SKIN_ICON_INFO, *PUIX_SKIN_ICON_INFO;

Members

IconResID
Resource identifier of the icon.
InsertPosInText
Character position within the text (zero-based) where the icon is positioned. Valid only if Flags is UIX_ICON_INSIDE_TEXT.
Flags

Flags indicating the justification of the icon. Must be one of the following values.
Value Description
UIX_ICON_INSIDE_TEXT The icon is embedded within the text just before character position InsertPosInText.
UIX_ICON_RIGHT_JUSTIFIED The icon is justified to the right of the object.

Remarks

IPluginSupport::GetImage can be used to load the icon from the skin file.

Requirements
