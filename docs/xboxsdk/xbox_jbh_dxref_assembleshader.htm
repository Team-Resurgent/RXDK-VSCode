XGAssembleShader

Assembles an ASCII description of a vertex or pixel shader into binary form.

HRESULT WINAPI XGAssembleShader(
    LPCSTR pSourceFileName,
    LPCVOID pSrcData,
    UINT SrcDataLen,
    DWORD Flags,
    LPXGBUFFER *pConstants,
    LPXGBUFFER *pCompiledShader,
    LPXGBUFFER *pErrorLog,
    LPXGBUFFER *pListing,
    SASM_ResolverCallback pResolver,
    LPVOID pResolverUserData,
    LPDWORD pShaderType
    );

Parameters

pSourceFileName
[in] Source file name—used in error messages.
pSrcData
[in] A pointer to the source data.
SrcDataLen
[in] The source data length.
Flags
[in] SASM_xxx flags. See xgraphics.h for a complete list.
pConstants
[out] If constants are declared in the shader, they are written here. Pass NULL if you don't care.
pCompiledShader
[out] The shader microcode is written here. Pass NULL if you don't care.
pErrorLog
[out] Errors are written here. Pass NULL if you don't care.
pListing
[out] A human-readable listing is written here. Pass NULL if you don't want it.
pResolver
[in] Used by the preprocessor. Can be NULL if you don't use #include in your source file.
pResolverUserData
[in] Passed unmodified to the pResolver function.
pShaderType
[out] Returns the type of shader that was assembled. Pass NULL if you don't care.

Return Values

Returns S_OK if no errors.

Returns a failure code if an error occured. For problems with the assembly program syntax, human-readable errors and warnings are written to the pErrorLog.

Remarks

The original file name and line number of the assembly source code can be specified by adding the #line directive to the source data. The #line directive is recognized only when the normal preprocessor is disabled. The preprocessor can be disabled by passing the SASM_SKIPPREPROCESSOR flag into the Flags parameter of XGAssembleShader.
