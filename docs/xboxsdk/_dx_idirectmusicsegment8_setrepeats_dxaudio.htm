IDirectMusicSegment8::SetRepeats

Sets the number of times the looping portion of the segment is to repeat. By default, the entire segment is looped.

HRESULT SetRepeats(
    DWORD dwRepeats
);

Parameters

dwRepeats
Number of times that the looping portion of the segment is to repeat, or DMUS_SEG_REPEAT_INFINITE to repeat until explicitly stopped. A value of 0 specifies a single play with no repeats.

Return Values

The method returns S_OK.

Remarks

On the Xbox, waves with embedded loop points are unrolled in order to support sample-accurate looping. A segment generated from a wave file with authored loop points translates into a segment of length (number of loops * wave length), with a single looped wave object spanning the length of the segment. The entire segment is treated as a whole and can be looped by SetRepeats, which will generate DMUS_NOTIFICATION_SEGLOOP notifications.

For example, if you authored a wave with a loop that repeats twice, you would hear the loop three times (once, plus the two repeats). If you call SetRepeats(1), you will hear the loop six times and there would be a DMUS_NOTIFICATION_SEGLOOP notification on the fourth time. Depending on the content and timing there could be a tiny gap before the fourth time, because looping with SetRepeats is not sample-accurate.
