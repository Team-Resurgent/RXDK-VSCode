Presence Functions

The Xbox Live presence functions include the following.

Function Description
XOnlinePresenceAdd Adds one or more players to a presence subscription.
XOnlinePresenceClear Clears the presence subscription.
XOnlinePresenceGetLatest Retrieves the most recent presence session information.
XOnlinePresenceGetTitleName Retrieves a title name.
XOnlinePresenceInit Initializes a presence session.
XOnlinePresenceSubmit Creates a presence subscription.
