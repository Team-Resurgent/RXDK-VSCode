D3DDISTANCETYPE

Used by IDirect3DDevice8::GetPushDistance to determine how close the GPU is to idling or waiting.

typedef enum _D3DDISTANCETYPE{
    D3DDISTANCE_FENCES_TOIDLE = 2,
    D3DDISTANCE_FENCES_TOWAIT = 4,
    D3DDISTANCE_FORCE_DWORD = 0x7fffffff,
} D3DDISTANCETYPE;

Constants

D3DDISTANCE_FENCES_TOIDLE
GetPushDistance will return the current number of fences that the GPU must process before it becomes idle.
D3DDISTANCE_FENCES_TOWAIT
GetPushDistance will return the current number of fences that the GPU must process before the resource being waited upon in the IDirect3DDevice8::SetWaitCallback callback is ready to be freed. If the GPU has already passed the waiting point, GetPushDistance would return zero.
D3DDISTANCE_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.
