XONLINE_ARB_REGISTRANT

Contains data about a registrant in an arbitrated session.

typedef struct{
    ULONGLONG qwMachineID;
    XUID xuidUsers[XONLINE_MAX_LOGON_USERS];
    BYTE bReliabilityValue;
} XONLINE_ARB_REGISTRANT, *PXONLINE_ARB_REGISTRANT;

Members

qwMachineID
ID of the Xbox that the player is using. Titles can use the XNetXnAddrToMachineId function to determine the machine ID of an Xbox given its XNADDR structure.
xuidUsers
Array of players logged on by this registrant.
bReliabilityValue
Relative reliability value for this registrant.
