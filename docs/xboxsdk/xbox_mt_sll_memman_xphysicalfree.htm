XPhysicalFree

The XPhysicalFree function frees memory previously allocated with a call to XPhysicalAlloc.

VOID XPhysicalFree(
  LPVOID lpAddress  // address of region
);

Parameters

lpAddress
[in] Pointer to the base address of the region of memory to be freed. This parameter must be the address returned by the XPhysicalAlloc function when the memory was allocated.

Return Values

This function does not return a value.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
