Xbox Game Certification

See the Xbox Guide for details about submitting a game for technical certification or game evaluation. The Xbox Guide is delivered monthly to all participants of the Xbox Development Kit (XDK) program and is available for download from the Xbox developer support Web site (https://xds.xbox.com). The XDK includes a sample game to demonstrate code that passes technical certification requirements. For more information, see Xbox Technical Certification Game.
