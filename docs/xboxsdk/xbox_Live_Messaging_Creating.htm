Creating Messages

Your title creates messages by invoking the XOnlineMessageCreate function. Specify the message type in the bMsgType parameter. The available message types are specified in the documentation XOnlineMessageCreate.

If your message type is XONLINE_MSG_TYPE_TITLE_CUSTOM, you can assign title-specific flags to the message. To do so, call the XOnlineMessageSetFlags function. Your title first clears the most significant byte of the flags with XONLINE_MSG_FLAGS_TITLE_RESERVED. You can then call XOnlineMessageSetFlags again to set custom flag values in the most significant byte.

Your title can also set the properties of a message with the XOnlineMessageSetProperty function. Your title can use both standard properties and custom properties that you define.
