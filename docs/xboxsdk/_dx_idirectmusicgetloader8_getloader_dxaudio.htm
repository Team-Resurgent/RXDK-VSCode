IDirectMusicGetLoader8::GetLoader

Retrieves a pointer to the loader object that created the stream.

HRESULT GetLoader8(
  IDirectMusicLoader ** ppLoader
);

Parameters

ppLoader
Address of a variable that receives the IDirectMusicLoader interface pointer. The reference count of the interface is incremented.

Return Values

If the method succeeds, the return value is S_OK.

If it fails, the method can return E_NOINTERFACE.
