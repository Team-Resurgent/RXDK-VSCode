XOnlineFriendsGetAcceptedGameInvite

The XOnlineFriendsGetAcceptedGameInvite function retrieves invitation data that was stored locally on the Xbox console when a player accepted a cross-title invitation.

Note    It is recommended that titles use XOnlineGameInviteGetLatestAccepted instead of this method.

HRESULT XOnlineFriendsGetAcceptedGameInvite(
    PXONLINE_ACCEPTED_GAMEINVITE pAcceptedGameInvite
);

Parameters

pAcceptedGameInvite
[out] Pointer to an XONLINE_ACCEPTED_GAMEINVITE structure that receives information about the accepted cross-title invitation.

Return Values

XOnlineFriendsGetAcceptedGameInvite always succeeds with one of the following return codes.

Value Description
S_OK There is a game invite on the hard disk.
S_FALSE There is no game invite on the hard disk.

Remarks

If a player receives an invitation to a title he is not currently playing, he can accept the invitation and insert the disc for the new title. The new game (upon boot) calls XOnlineFriendsGetAcceptedGameInvite to retrieve the game invitation.

This function is used in the following sequence of events as follows:

While playing Title A, a player receives a game invitation to Title B. If the player accepts the invitation, Title A calls XOnlineFriendsAnswerGameInvite with XONLINE_GAMEINVITE_YES to respond affirmatively. The XOnlineFriendsAnswerGameInvite function sends the response to the player's friend and writes the invitation to the local Xbox hard disk.

Note    The invitation stored on the Xbox console hard disk expires after 15 minutes.

Title A then instructs the player to insert the new game disc (for Title B). After initializing, Title B calls XOnlineFriendsGetAcceptedGameInvite to retrieve the game invitation from the local hard disk. All Xbox Live titles must call XOnlineFriendsGetAcceptedGameInvite at startup. Titles should call XOnlineFriendsGetAcceptedGameInvite before calling XOnlineLogon or XOnlineFriendsStartup.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
