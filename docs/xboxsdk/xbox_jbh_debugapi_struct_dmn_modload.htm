DMN_MODLOAD

Contains information relevant to a DM_MODLOAD or DM_MODUNLOAD notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_MODLOAD {
    char Name[MAX_PATH];
    PVOID BaseAddress;
    ULONG Size;
    ULONG TimeStamp;
    ULONG CheckSum;
    ULONG Flags;
} DMN_MODLOAD, *PDMN_MODLOAD;

Members

Name
Name of the module that was loaded.
BaseAddress
Address that the module was loaded to.
Size
Size of the module.
TimeStamp
Time stamp of the module.
CheckSum
Checksum of the module.
Flags
Flags indicating additional module information. Can be a combination of the following flags:
FlagDescription
DMN_MODFLAG_XBEThe module is part of an XBE.
DMN_MODFLAG_TLSThe module uses thread local storage.
