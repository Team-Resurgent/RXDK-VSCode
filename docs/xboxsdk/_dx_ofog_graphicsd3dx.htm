oFog

The output fog value register. If D3DRS_FOGTABLEMODE is set to a linear or exponential fog effect, then the value is the fog factor that will be interpolated and routed to the fog table. Otherwise (if the fog effect is D3DFOG_NONE), the shader has complete control over the fog blend factor, which will be a value between 0.0 and 1.0. Only the scalar x-component of the fog is used.
