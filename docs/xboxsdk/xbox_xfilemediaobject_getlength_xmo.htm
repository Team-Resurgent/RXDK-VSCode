XFileMediaObject::GetLength

Determines the total length of the file in bytes.

HRESULT GetLength(
    LPDWORD pdwLength
);

Parameters

pdwLength
[out] Address of a value containing the number of bytes in the file.

Return Values

Returns S_OK if successful, an error code otherwise.
