Voice Chat Structures

This section contains the structures associated with voice communication.

Structure Description
XVOICE_CODEC_HEADER Exposes internal voice API statistics.
XVOICE_MASK Stores configuration values for a voice mask.
XVOICE_QUEUE_XMO_CONFIG Stores configuration information for an voice-queue Xbox media object.
XVOICE_VOICE_PROPERTIES Stores information that the title uses to perform lip synchronization for the graphical representation of a voice-chat user.
