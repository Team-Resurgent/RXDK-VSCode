Disabling Antialiasing

There are two ways to turn off antialiasing after it has been enabled. The first method is to reset the device using IDirect3DDevice8::Reset and specify D3DMULTISAMPLE_NONE for the MultiSampleType member of D3DPRESENT_PARAMETERS. Depending on the situation, you may not want to reset the device. Perhaps antialiasing only needs to be turned off for some selected frames.

The second method of disabling antialiasing is to use the D3DRS_MULTISAMPLEMODE render state and IDirect3DDevice8::SetBackBufferScale.

Disabling antialiasing should be done before any rendering calls, right after the previous call to Swap or Present.

If supersampling is being used, simply disable antialiasing by calling SetBackBufferScale with a scale appropriate to the size of the sampling. Here are some examples:

// Assume that pDevice is a valid pointer to the
// IDirect3DDevice8 interface.

// Example 1. The title is doing 2x horizontal supersampling.
pDevice->SetBackBufferScale( 0.5f, 1.0f );

// Example 2. The title is doing 4x supersampling.
pDevice->SetBackBufferScale( 0.5f, 0.5f );

// Example 3. The title is doing 9x supersampling.
pDevice->SetBackBufferScale( 0.3333333f, 0.3333333f );

If title is using multisampling, then it is necessary to use both D3DRS_MULTISAMPLEMODE and SetBackBufferScale to disable the antialiasing. Here are some more examples:

// Assume that pDevice is a valid pointer to the
// IDirect3DDevice8 interface.

// Example 1. The title is doing 4x multisampling.
// pDevice->SetRenderState( D3DRS_MULTISAMPLEMODE, D3DMULTISAMPLEMODE_1X );
// pDevice->SetBackBufferScale( 0.5f, 0.5f );

// Example 2. The title is doing 2x multisampling.
// pDevice->SetRenderState( D3DRS_MULTISAMPLEMODE, D3DMULTISAMPLEMODE_1X );
// pDevice->SetBackBufferScale( 0.5f, 1.0f );

Changing the multisample mode has the effect of changing the ratio of the sample size to the number of pixels sent to the pixel shader. With a multisample mode of D3DMULTISAMPLEMODE_4X, the sample size will be four and a single pixel will be sent to the pixel shader. With a multisample mode of D3DMULTISAMPLEMODE_1X, the sampling size will be one and a single pixel will be sent on the pixel shader. Setting the mode to D3DMULTISAMPLEMODE_1X turns off multisampling and effectively turns on supersampling, which is why it is necessary to call SetBackBufferScale.
