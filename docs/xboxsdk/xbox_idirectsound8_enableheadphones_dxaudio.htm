IDirectSound8::EnableHeadphones

Determines the headphone settings.

HRESULT EnableHeadphones(
    BOOL fEnabled
);

Parameters

fEnabled
Boolean value that enables or disables the headphone settings. If TRUE, the function enables the headphone settings (crosstalk cancellation is turned off and the stereo HRTF coefficients are used). If FALSE, the function restores the HRTF coeffcients and restores crosstalk cancellation.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

EnableHeadphones only affects the handling of 3D-positioned sounds. When enabled, 3D DirectSound buffers and 3D DirectSound streams will only sound from the front two speakers (presumably through headphones the listener is wearing).

Developers concerned that their game title content will not be properly downmixed on some decoders can do the following:

  1. Shut down DirectSound.
  2. Call DirectSoundOverrrideSpeakerConfig (changing the speaker configuration from surround to stereo, which will force the Xbox console to downmix everything to stereo).
  3. Restart DirectSound.
  4. Call EnableHeadphones.
