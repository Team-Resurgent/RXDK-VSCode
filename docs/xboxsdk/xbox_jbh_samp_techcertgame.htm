Xbox Technical Certification Game

The TechCertGame sample demonstrates code that meets or exceeds Xbox technical certification requirements (TCRs) defined by the Technical Certification Requirements version 3.00, which are available in the latest edition of the Xbox Guide.

Overview

The technical certification game is a playable driving game to be used as a reference for developers. The technical certification game has the following goals:

  • Meet all technical certification requirements.
  • Properly use Xbox reference UI elements.
  • Demonstrate high-performance, optimized XTL code.
  • Show techniques for game localization.

The technical certification game also has certain non-goals:

  • It is not intended to represent an Xbox game in terms of features or playability.
  • It is not designed to be a fun or feature-complete game.
  • Though it passes technical certification, it would score poorly if submitted to the Xbox playtest team for review.

Instead, consider the technical certification game a source for example code that demonstrates exactly what is needed to pass a specific TCR, not as an example of how to make an exciting racing game.

Features and Met Requirements

This version of the title certification game is single-player only, with a single car and landscape available. It has minimal graphics effects and audio support. However, the game meets or exceeds technical certification requirements in the following areas:

  • Xbox Memory Unit support
  • Storage (saved games)
  • Xbox Controller support
  • Xbox system settings
  • Xbox Dashboard integration
  • Soundtracks
  • Utility drive persistence
  • Saved game signing
  • French, German, Italian, and Spanish language support
  • Other stability requirements

Both the TechCertGame and portions of the sample framework code include TCR numbers and names in comments. These comments are placed by the code that meets a given requirement in the format:

  // TCR X-Y Name of Requirement

where X is the TCR section number, Y is the requirement number, and Name of Requirement is the title of the requirement. For example:

  // TCR 3-22 Loss of Controller

Future Features

The version of the technical certification game included with this release of the XDK does not yet contain its full feature set. Certain features are not yet implemented; therefore, certain requirements are not yet met. Future versions of the technical certification game will contain more features and will eventually exercise and explicitly pass every possible technical certification requirement. The following features will be added to the technical certification game in future releases of the XDK:

  • Multiplayer and online support
  • Nicknames

How to Play

To play the technical certification game, use the following controls:

  • To steer left, press the directional pad left or move the left thumbstick left.
  • To steer right, press the directional pad right or move the left thumbstick right.
  • To accelerate, press the right trigger.
  • To brake or accelerate in reverse, press the left trigger.
  • Press the START button to display a menu, from which you can exit the game.
