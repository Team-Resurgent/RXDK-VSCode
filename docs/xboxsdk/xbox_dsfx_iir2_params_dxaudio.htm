DSFX_IIR2_PARAMS

Contains the parameters used by the second-order infinite impulse response (IIR2) effect.

typedef struct _DSFX_IIR2_PARAMS{
    DSFX_IIR2_STATE State;
    DWORD dwFilterB0;
    DWORD dwFilterB1;
    DWORD dwFilterB2;
    DWORD dwFilterA1;
    DWORD dwFilterA2;
} DSFX_IIR2_PARAMS, *LPDSFX_IIR2_PARAMS;

Members

State
DSFX_IIR2_STATE structure that contains the state information for this effect.
dwFilterB0
B0 parameter for this effect.
dwFilterB1
B1 parameter for this effect.
dwFilterB2
B2 parameter for this effect.
dwFilterA1
A1 parameter for this effect.
dwFilterA2
A2 parameter for this effect.

Remarks

The following equation is used to calculate the output for this effect.


y[n] = b0*x[n] + b1*x[n-1] + b2*x[n-2] + a1*y[n-1] + a2*y[n-2]
