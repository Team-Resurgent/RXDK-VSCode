Content Download Functions

The Xbox Live content download functions include the following.

Function Description
XOnlineContentInstall

Begins the download and installation of a specified offering. The function creates an asynchronous task to perform the download and installation.
XOnlineContentInstallGetProgress

Retrieves the current progress of an offering download started with XOnlineContentInstall.
XOnlineContentInstallGetSize

Retrieves the full install size (in blocks) of a piece of downloaded content.
XOnlineContentSetSecurityKey

Sets the security key used to decrypt the package keys of specially signed title content received from PartnerNet.
XOnlineContentVerify

Begins the verification of a specified offering that has been downloaded and installed. The function creates an asynchronous task to perform the verification.
XOnlineOfferingCancel

Creates an asynchronous task to cancel a specified online offering owned by a specified user.
XOnlineOfferingDetails

Creates an asynchronous task to retrieve the details associated with a specified online offering.
XOnlineOfferingDetailsGetResults

Retrieves the details results requested with a call to XOnlineOfferingDetails.
XOnlineOfferingDetailsMaxSize

Determines the details buffer size needed in calls to XOnlineOfferingDetails.
XOnlineOfferingEnumerate

The XOnlineOfferingEnumerate function begins enumeration of the available online content and subscription offerings. The function creates an asynchronous task to perform the enumeration.
XOnlineOfferingEnumerateGetResults

Retrieves the results of an offering enumeration initiated with a call to XOnlineOfferingEnumerate.
XOnlineOfferingEnumerateMaxSize

Determines the buffer size needed to contain a specified number of enumeration records. This buffer size is used in calls to XOnlineOfferingEnumerate.
XOnlineOfferingIsNewContentAvailable

Indicates whether the calling title has new content offerings available. This function can be called without first logging onto the Xbox Live service, and can thus be called by titles that are using the Content Download Manager.
XOnlineOfferingPriceFormat

Converts a price, represented by an XONLINE_PRICE structure, into a display string.
XOnlineOfferingPurchase

Creates an asynchronous task to purchase the specified online offering for the specified user.
