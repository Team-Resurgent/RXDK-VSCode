IXACTWmaPlayList::SetPlaybackBehavior

Sets the way in which an XACT WMA playlist is played.

HRESULT SetPlaybackBehavior(
    DWORD dwFlags
);

Parameters

dwFlags
[in] Flags indicating how the playlist is played. Can be one or more of the following:
Value Description
XACT_FLAG_WMAPLAYLIST_PLAYBACK_RANDOM Indicates that songs should be shuffled before playback.
XACT_FLAG_WMAPLAYLIST_PLAYBACK_LOOP Indicates playlist should loop upon completion of playback.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

When only the XACT_FLAG_WMAPLAYLIST_PLAYBACK_LOOP flag is specified, the entire playlist will played sequentially then start from the beginning of the playlist again.

When only the XACT_FLAG_WMAPLAYLIST_PLAYBACK_RANDOM flag is specified, the entire playlist will be played in random order (each track played once, with no repeats) and then stop.

When both flags are specified, the songs will be played in random order then playback will start from the beginning of the playlist.

To reshuffle playlist for random playback, call SetPlaybackBehavior and specify the XACT_FLAG_WMAPLAYLIST_PLAYBACK_RANDOM flag.

Requirements
