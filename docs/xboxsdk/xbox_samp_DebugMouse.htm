DebugMouse Sample

The DebugMouse sample demonstrates how to use the debug mouse API functions.

Overview

This sample shows how to use the Input API to use a mouse plugged into the Xbox using the Debug Keyboard adapter. You must use a USB mouse. The Xbox supports many different kinds of mice, including wireless mice. The API allows you to poll the mouse position, the mouse wheel, and the button states for the left, right, wheel (or center button), and the two extra buttons on the sides of most new IntelliMouse® mice.

You can plug in up to four mice (one in each port). The sample demonstrates how to handle plugging in and unplugging the mice from all of the ports. It doesn't try to handle moving or clicking four mice at once.
