Competition Functions

The Xbox Live competition functions include the following.

Function Description
XOnlineCompetitionCancel

Cancels a tournament.
XOnlineCompetitionCheckin

Enables a player to perform the required check-in for a competition event. Check-in must succeed prior to starting any round.
XOnlineCompetitionCreate

Creates a competition or tournament.
XOnlineCompetitionCreateGetResults

Retrieves the results of the task started by XOnlineCompetitionCreate or XOnlineCompetitionCreateSingleElimination.
XOnlineCompetitionCreateSingleElimination

Creates a single-elimination competition.
XOnlineCompetitionGetResultsBufferSize

Retrieves the buffer size required for the results of a search initiated by functions such as XOnlineCompetitionSearch and XOnlineCompetitionTopology.
XOnlineCompetitionManageEntrant

Updates the entrant information in a competition.
XOnlineCompetitionSearch

Searches the competitions dataset for result lists using a specific query ID.
XOnlineCompetitionSearchGetResults

Retrieves the results of the task begun by XOnlineCompetitionSearch.
XOnlineCompetitionSessionRegister

Registers a player for a session in a competition.
XOnlineCompetitionSessionRegisterGetResults

Retrieves the results of the task started by the XOnlineCompetitionSessionRegister function.
XOnlineCompetitionSubmitResults

Submits the results of a competition.
XOnlineCompetitionTopology

Searches the topology of a competition.
XOnlineCompetitionTopologyGetResults

Retrieves the results of the topology search started by XOnlineCompetitionTopology.
XOnlineCompetitionTopologySingleElimination

Searches the topology of a single-elimination competition.
XOnlineCompetitionTopologySingleEliminationGetResults

Retrieves the results of the task begun by XOnlineCompetitionTopologySingleElimination.
