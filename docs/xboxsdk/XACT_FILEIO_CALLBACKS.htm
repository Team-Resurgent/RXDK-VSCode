XACT_FILEIO_CALLBACKS

Contains title-defined XACT file I/O callbacks registered with XACTEngineSetFileIOCallbacks.

typedef struct _XACT_FILEIO_CALLBACKS{
    XACT_READFILE_CALLBACK ReadFileCallback;
    XACT_GETOVERLAPPEDRESULT_CALLBACK GetOverlappedResultCallback;
} XACT_FILEIO_CALLBACKS, *PXACT_FILEIO_CALLBACKS;

Members

ReadFileCallback
Title-defined XACT_READFILE_CALLBACK called by XACT to read data from the disk. If this member is NULL, XACT uses the system ReadFile.
GetOverlappedResultCallback
Title-defined XACT_GETOVERLAPPEDRESULT_CALLBACK called by XACT to determine the results of an overlapped read operation begun with a call to the read file callback. If this member is NULL, XACT uses the system GetOverlappedResult.

Remarks

XACT file I/O callbacks are registered using the XACTEngineSetFileIOCallbacks function.

Titles can specify NULL for either (or both) of the callbacks to direct XACT to use the system routine rather than calling the title's function. Specifying NULL for ReadFileCallback is rare, and likely not useful.
