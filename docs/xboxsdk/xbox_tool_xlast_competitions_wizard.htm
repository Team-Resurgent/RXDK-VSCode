The New Competitions Project Wizard

To create a new competitions configuration, click New on the File menu. Choose Competitions Configuration and click OK. This starts the New Competitions Project Wizard.

After displaying a welcome dialog box, the Competitions Project Wizard presents a dialog box asking for the following information.

Competitions Project Name
Specifies the name of the competitions configuration. This name is used as the name of the XML project file for the submission. It cannot contain any folder or path information. The maximum length of the content configuration project name is 38 characters. Valid characters for the content project name are the characters a–z, A–Z, the numerals 0–9, and the characters !, #, $, &, ', (, ), −, @, [, ], ^, _, `, {, }, and ~. You do not need to provide a file name extension.
Project Directory
Specifies the path to the directory containing the project file.
Title ID
Contains the 8-digit hexadecimal title identifier that you receive from Microsoft.

When you are done entering the information, click Next. The New Competitions Project Wizard informs you that you are done creating a project. Click Finish.
