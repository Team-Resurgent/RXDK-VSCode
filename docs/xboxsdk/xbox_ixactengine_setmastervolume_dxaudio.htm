IXACTEngine::SetMasterVolume

Sets the master volume for all audio output from XACT engine.

HRESULT SetMasterVolume(
    WORD wCategory,
    LONG lVolume
);

Parameters

wCategory
[in] Value specifying the category of sounds to set the volume for. XACT allows the sound designer to define a category for each sound. For example, Music, Sound Effects, and Voice may be three categories. This allows the game to have a separate control for each of these game sound elements. To set the volume for all categories, the special category XACT_CATEGORY_INDEX_UNUSED may be used. XACT supports up to 255 categories.
lVolume
[in] Value specifying the attenuation, in 100th of a decibel (millibells) to attenuate the sound. A setting of 0 (default) represents full volume.

Return Values

Returns DS_OK if successful; returns an error code otherwise.
