IDirectSoundBuffer8

Used to manage sound buffers.

DirectSound buffers on the Xbox can be chained together; the output of a buffer does not have to go to the final output. Instead, it can be sent to the input of another buffer. Multiple DirectSound buffers can be mixed together, with the result going to the input of another buffer. This allows better use of 3D hardware resources (because several sounds can be mixed prior to 3D processing). It also provides a way to have additional passes through the LFOs (low frequency oscillators), EGs (envelope generators), and filter blocks prior to final output. See SetOutputBuffer.

The IDirectSoundBuffer8 interface supersedes IDirectSoundBuffer and adds new methods.

To obtain the interface, use the IDirectSound8::CreateSoundBuffer method to retrieve an IDirectSoundBuffer.

The IDirectSoundBuffer8 interface also supports the AddRef and Release methods. The QueryInterface method is not supported on the Xbox.

The LPDIRECTSOUNDBUFFER8 type is defined as a pointer to the IDirectSoundBuffer interface:

typedef struct IDirectSoundBuffer8    *LPDIRECTSOUNDBUFFER8;
