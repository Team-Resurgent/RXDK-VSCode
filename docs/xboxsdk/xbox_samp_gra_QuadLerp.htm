QuadLerp Sample

The QuadLerp sample performs a linear interpolation between four textures based on the specular color.

Overview

This sample demonstrates how to optimize pixel shaders. The pixel shaders used in this program show several implementations of the same pixel shader, each with varying degrees of optimization. The shaders all implement the same effect, which is to linearly interpolate between four textures based on the r, g, b, and a components of the specular color.

This sample is a companion to the Pixel Shader Optimizing white paper.

Path

Source: (XDK Root)\Samples\Xbox\Graphics\QuadLerp

Remarks

Use the left thumbstick to increase the emphasis on a particular texture.

Pixel shaders 2 and 3 have slightly different behavior than the others. Pixel shaders 0 and 1 are nearly identical. The only difference between the two is that pixel shader 1 runs slightly faster. If you do a PIX capture of these two shaders you will see that the ideal fill rate for pixel shader 1 is double the ideal fill rate for pixel shader 0.

If the textures are setup with mip-mapping and texture compression, texture fetch bandwidth doesn't interfere. As a result, the optimized pixel shaders run almost twice as fast as the unoptimized version.
