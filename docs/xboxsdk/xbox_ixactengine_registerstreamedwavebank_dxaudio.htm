IXACTEngine::RegisterStreamedWaveBank

Creates a new streaming IXACTWaveBank.

HRESULT RegisterStreamedWaveBank(
    PCXACT_WAVEBANK_STREAMING_PARAMETERS pParams,
    PXACTWAVEBANK *ppWaveBank
);

Parameters

pParams
[in] Pointer to an XACT_WAVEBANK_STREAMING_PARAMETERS structure.
ppWaveBank
[out] Address of an IXACTWaveBank pointer to be returned.

Return Values

Returns DS_OK if successful, an error code otherwise.

Remarks

A streaming wave bank file (.xwb) is created using XACT. Because the data is streamed off of disk, the file handle to the previously opened file containing the streaming wave bank data is passed.

Wave banks can be specified as being in-memory, or streamed (optimized for DVD or hard disk) in XACT itself.

In instrumented builds (xactengd.lib and xactengi.lib), a wave bank is registered twice—once in main memory (below 64 MB), and once in upper memory. The latter allows the wave bank to grow as the content creator edits and adjusts content in real time. For more information about upper memory, see Xbox Development System Memory.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
