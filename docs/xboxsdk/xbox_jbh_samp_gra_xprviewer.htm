XboxPackedResource Sample

The XboxPackedResource sample (found in the XPRViewer folder) demonstrates how to load textures from an Xbox Packed Resource file. The texture data is read from the file using overlapped I/O and without buffering to get the best performance possible. Also, the application can render a loading screen/animation while the file is being read in the background.

The Xbox Packed Resource format is designed for fast loading of Direct3D resources using the Register() API. The bundler tool processes the textures (format conversion, swizzling, and so on) and then writes out the texture data in the format used by the Xbox GPU. The texture data can then be read into a contiguous block of memory, and then Direct3D textures can be pointed at the data via the Register() API. See the documentation on Register() for more details.

See the Bundler tool to learn how to create Xbox Packed Resource files.
