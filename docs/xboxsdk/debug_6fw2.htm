CONTEXT

A CONTEXT structure contains processor-specific register data. The system uses CONTEXT structures to perform various internal operations. Refer to the header file WinNT.h for a complete definition of this structure.

Requirements

Header: Declared in Winnt.h; include Xtl.h.
