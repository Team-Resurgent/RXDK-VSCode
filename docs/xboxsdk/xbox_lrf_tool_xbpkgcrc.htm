Electronic Submission Package CRC Checker (XbPkgCRC)

The XbPkgCRC tool verifies that an electronic submission package (in the form of a .pkg file) has not been changed or corrupted in any way since it was generated.

Usage

USAGE: xbpkgcrc filename[.pkg]

If the filename argument to XbPkgCRC does not have a .pkg extension, XbPkgCRC appends the .pkg extension to the supplied file name before trying to open the file.

The XbPkgCRC tool relies on the cyclical redundancy check (CRC) generated by the Xbox Premastering Wizard (XbPremaster). The final page of the XbPremaster wizard and the logfile that XbPremaster generates show the CRC value calculated for the final .pkg file. The XbPkgCRC tool recalculates the CRC for a .pkg file, then compares it with the CRC stored in the .pkg file itself. If the two values do not match, XbPkgCRC reports that the .pkg file is corrupt.

Errors may be introduced to a .pkg file during copy operations across an internal network or over the Internet, or simply from disk errors on a local file system. If you suspect that a .pkg file might be corrupt, run XbPkgCRC to verify the validity of the file. As a last step before submitting an electronic submission package for certification, you should run the file through XbPkgCRC to make sure it is intact.

If a .pkg file fails a check by XbPkgCRC, you either need to restore the file from backup or regenerate it by re-running XbPremaster.

Note    XbPremaster has been superseded by the Xbox Game Disc Authoring Tool (XbGameDisc). XbPremaster is still available in the c:\Program Files\Microsoft Xbox SDK\xbox\bin\OldDVDTools folder for use with older projects, but you are strongly encouraged to use XbGameDisc for game disc layout, premastering, and emulation.
