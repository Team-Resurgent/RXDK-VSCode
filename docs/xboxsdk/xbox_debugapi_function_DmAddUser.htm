DmAddUser

Enables users to be added to the list of those authorized to establish a secure connection.

DMHRAPI DmAddUser(
    LPCSTR szUserName,
    DWORD dmpl
);

Parameters

szUserName
Username to be added to the list of authorized users.
dmpl
Privileges for the specified user. This parameter is a combination of one or more of the following constants.
Constant Value Description
DMPL_PRIV_READ 0x0001 The username has read privileges
DMPL_PRIV_WRITE 0x0002 The username has write privileges
DMPL_PRIV_CONTROL 0x0004 The username can control the debug manager.
DMPL_PRIV_CONFIGURE 0x0008 The username can configure the development console.
DMPL_PRIV_MANAGE 0x0010 The username can manage user accounts.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

For users to establish secure connections, they must be added to the list of authorized users. This function provides a way to add them programmatically.

Requirements
