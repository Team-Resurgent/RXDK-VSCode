ntohs

Converts a u_short from TCP/IP network byte order to host byte order (which is little-endian on Intel processors).

u_short ntohs(
  u_short netshort  
);

Parameters

netshort
[in] 16-bit number in TCP/IP network byte order.

Return Values

The ntohs function returns the value in host byte order.

Remarks

The ntohs function takes a 16-bit number in TCP/IP network byte order and returns a 16-bit number in host byte order.

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
