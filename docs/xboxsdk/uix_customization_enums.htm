Drop-in UI Customization Enumerations

The following enumerations are used with Xbox Drop-in UI customization:

Enumeration Description
UIX_INPUT_TYPE Type of controller input passed from the Xbox Drop-in UI engine to an object through ITitleUIPlugin::PassInputToObject.
UIX_OBJECT_TYPE Object type of an object handled by ITitleUIPlugin. Specified in the ObjectType parameter of ITitleUIPlugin::CreateObject when a request is made to create a UI object.
UIX_OBJSTATE_TYPE The specific object state retrieved or set with ITitleUIPlugin::GetObjectState or ITitleUIPlugin::SetObjectState.
