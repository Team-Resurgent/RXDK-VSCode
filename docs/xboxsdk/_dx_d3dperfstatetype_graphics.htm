D3DPERFSTATETYPE

Defines what vertex cache states can be set using D3DPERF_SetState.

typedef enum _D3DPERFSTATETYPE
{
    D3DPERFSTATE_VTX_CACHE,
    D3DPERFSTATE_VTX_FILE,
    D3DPERFSTATE_VTX_ALL
} D3DPERFSTATETYPE;

Constants

D3DPERFSTATE_VTX_CACHE
4 KB pre-transformation and lighting vertex cache.
D3DPERFSTATE_VTX_FILE
24 entry post-transformation and lighting vertex cache.
D3DPERFSTATE_VTX_ALL
Enable or disable all vertex cache states.
