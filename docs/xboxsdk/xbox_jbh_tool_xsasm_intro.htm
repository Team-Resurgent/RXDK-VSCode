Xbox Shader Tools

This section provides information regarding the Xbox Shader Assembler Tool (XSAsm), as well as tools designed to aid in debugging pixel and vertex shaders.

The topics in this section include:

  • Shader Assembler Tool (XSAsm)
  • Shader Assembler Tool Error Codes
  • Pixel Shader Debugger (XbPSCapture)
  • Vertex Shader Debugger (XbVSCapture)
  • Vertex Shader Tune Tool (VSTune)
  • Xbox HLSL Compiler (XFXC)
