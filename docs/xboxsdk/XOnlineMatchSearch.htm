XOnlineMatchSearch

Searches for online-game sessions that match the specified attributes.

HRESULT XOnlineMatchSearch(
    DWORD dwProcedureIndex,
    DWORD dwNumResults,
    DWORD dwNumAttributes,
    const XONLINE_ATTRIBUTE *pAttributes,
    DWORD dwResultsLen,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwProcedureIndex

[in] Stored procedure in the match-making database.

dwNumResults

[in] Number of results to return from the stored procedure. The XOnlineMatchSearch function may return fewer results, but it will never return more than this number.

dwNumAttributes

[in] Number of attributes in the attribute array pointed to by the pAttributes parameter.

pAttributes

[in] Address of an array of XONLINE_ATTRIBUTE structures. The attributes represented by these structures are the parameters to the stored procedure specified by the dwProcedureIndex parameter.

dwResultsLen

[in] Size of the buffer that receives the results from the search. For this parameter, pass the return value from the XOnlineMatchSearchResultsLen function. If this buffer is not large enough for the number of results specified by the dwNumResults parameter, the number of results returned is reduced to fit within this size.

hWorkEvent

[in, optional] Optional. Handle to a caller-created event object. This object will be signalled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.

phTask

[out] Address of an XONLINETASK_HANDLE that receives the session search task. This handle must be pumped to completion by XOnlineTaskContinue. After the task has completed, call XOnlineMatchSearchGetResults to obtain information about the resulting online-game sessions that match the request. After the search results have been obtained, the handle should be closed using XOnlineTaskClose. This task sends the search request to the matchmaking service and receives the response.

Return Values

If XOnlineMatchSearch succeeds, the return value is S_OK.

If XOnlineMatchSearch fails, the return value is one of the following error codes.

Value Description
E_OUTOFMEMORY Insufficient memory. Free some memory.
WSAEADDRNOTAVAIL See Winsock Error Codes for more information.
WSACONNRESET See Winsock Error Codes for more information.
Other See Online Errors for information about other possible errors.

The following table lists the possible values that XOnlineTaskContinue may return when called with the handle for the task created by XOnlineMatchSearch.

Value Description
XONLINE_E_MATCH_INVALID_SEARCH_REQ The request is invalid. The wrong number of attributes was passed to a stored procedure. More detailed error information is returned by XOnlineTaskContinue when the search completes. For debugging purposes, call XOnlineTaskContinue to find out what went wrong.
XONLINE_E_MATCH_STRING_TOO_LONG The attribute string for the create, update, or search operation is longer then 400 characters, or the string wasn't terminated with a null character. Limit string length to 400 wide characters, and ensure that the string is properly terminated.
XONLINE_E_MATCH_BLOB_TOO_LONG The BLOB for the create, update, or search operation is longer then 800 bytes. Limit BLOB length to 800 bytes.
XONLINE_E_MATCH_INVALID_ATTR_TYPE The attribute type for the create, update, or search operation is something other than player or session.
XONLINE_E_MATCH_INVALID_SESSION_ID The specified session ID is not a valid session ID returned by XOnlineMatchSessionGetInfo or XOnlineMatchSessionFindFromID.
XONLINE_E_MATCH_INVALID_OFFSET One of the attribute/parameter offsets in the request was invalid. The attribute ID was OR'd with an invalid data type. Ensure that the attribute ID is OR'd with a valid attribute data type mask (X_ATTRIBUTE_DATATYPE_INTEGER, X_ATTRIBUTE_DATATYPE_STRING, or X_ATTRIBUTE_DATATYPE_BLOB)
XONLINE_E_MATCH_INVALID_PARAM The data type that was passed to the stored procedure is not valid. Ensure that the data type sent to the stored procedure is the expected type.
XONLINE_E_MATCH_INVALID_TITLE_ID The title is not properly registered on the Matchmaking server. The attribute ID is invalid. Check the XMS file generated by MatchSim to ensure that the correct attribute ID is OR'd with the correct data type mask.

Remarks

The XOnlineMatchSearch function is called from an Xbox that wishes to join an online-game session that is hosted by a different Xbox.
