The New Stats Configuration Wizard

To create a new stats configuration submission, select New from the File menu. If your game was compiled with libraries from XDKs released before November 2003, click Legacy Stats Configuration. New titles should select the XLAST Stats version 3 stats configuration by clicking Statsv3 Configuration. After you've selected the appropriate version, click OK to start the New Stats Configuration Wizard.

After displaying the welcome dialog box, the New Stats Configuration Wizard presents a dialog box asking for the following information.

Stats Project Name
Specifies the name of the stats project. This name is used as the name of the XML project file for the submission. It cannot contain any folder or path information. The maximum length of the stats project name is 38 characters. Valid characters for the content package name are the characters a–z, A–Z, the numerals 0–9, and the characters !, #, $, &, ', (, ), -, @, [, ], ^, _, `, {, }, and ~. You do not need to provide a file name extension.
Project Directory
Specifies the path to the directory containing the XML project file (with an extension of .xlast) and a build file (which has an extension of .xsc). Both of these use the stats project name as their file names.
Title ID
Contains the 8-digit hexadecimal title identifier that you receive from Microsoft.
Make stats available on the web
Check this box if you want the statistics in all of the leaderboards in this project to be available on the Web. You can override this for individual leaderboards after you create your stats package.

After you enter the requested information, click Next. The New Stats Configuration wizard displays a dialog box asking you to select the locales you want to support. When you are finished selecting locales, click Next. The New Stats Configuration wizard then displays a dialog box stating that you have finished inputting data for your project. Click Finish to create the XML project file.
