Voice Chat Functions

This section contains the functions associated with voice communication.

Function Description
XVoiceCreateFourToFourFullDecoder Returns an IXVoiceDecoder interface for a four-to-four full decoder.
XVoiceCreateFourToFourFullEncoder Returns an IXVoiceEncoder interface for a four-to-four full encoder.
XVoiceCreateFourToFourRoundRobinEncoder Returns an IXVoiceEncoder interface for a four-to-four round-robin encoder.
XVoiceCreateFourToOneMixingDecoder Returns an IXVoiceDecoder interface for a four-to-one mixing decoder.
XVoiceCreateMediaObject Creates an Xbox Media Object (XMO) for a device on the Xbox Communicator.
XVoiceCreateMediaObjectEx Creates an Xbox Media Object (XMO) for a device on the Xbox Communicator.
XVoiceCreateOneToOneDecoder

Returns an IXVoiceDecoder interface for a one-to-one decoder.
XVoiceCreateOneToOneEncoder

Returns an IXVoiceEncoder interface for a one-to-one encoder.
XVoiceCreateTwoToOneMixingDecoder

Returns an IXVoiceDecoder interface for a two-to-one mixing decoder.
XVoiceCreateTwoToTwoEncoder

Returns an IXVoiceEncoder interface for a two-to-two encoder.
XVoiceDecoderCreateMediaObject Creates an instance of the Xbox Media Object (XMO) that is used to decode voice data.
XVoiceEncoderCreateMediaObject Creates an Xbox Media Object (XMO) that is used to encode the voice data from the Xbox Communicator.
XVoiceGetCodecBufferSize Retrieves the output buffer size for the encoder or decoder Xbox Media Object (XMO) depending on the PCM buffer size to be processed.
XVoiceGetEncodedPacketEnergy Provides an energy measurement of a given encoded media packet.
XVoiceGetEncodedParameters Retrieves the encoded properties for the specified packet of voice data.
XVoiceQueueCreateMediaObject Creates an instance of the Xbox Media Object (XMO) that handles the queuing and decoding of voice data.
XVoiceSynchronizedQueueCreateMediaObject Creates an instance of the Xbox Media Object (XMO) that handles the queuing and decoding of voice data.
