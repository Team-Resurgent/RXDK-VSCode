IDirectSound8::CreateSoundStream

Creates an IDirectSoundStream interface.

HRESULT CreateSoundStream(
	LPCDSSTREAMDESC pdssd,
	LPDIRECTSOUNDSTREAM *ppStream,
	LPUNKNOWN pUnkOuter
);

Parameters

pdssd
Address of a DSSTREAMDESC structure that describes the sound stream to create.
ppStream
Address of a variable that receives the IDirectSoundStream interface pointer of the new stream object.
pUnkOuter
Must be NULL.

Return Values

CreateSoundStream returns DS_OK if successful.

Requirements
