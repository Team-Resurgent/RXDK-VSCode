D3DVSD_SKIPBYTES

Specifies how many DWORDs to skip in the vertex.

D3DVSD_SKIPBYTES( _BYTECount ) \
    (D3DVSD_MAKETOKENTYPE(D3DVSD_TOKEN_STREAMDATA) | 0x18000000 | \
     ((_BYTECount) << D3DVSD_SKIPCOUNTSHIFT))

Parameters

_BYTECount
Number of BYTEs to skip in the vertex.
