XRestoreFloatingPointStateForDpc

Restores a previously saved floating point state. This function must be called from a deferred procedure call (DPC) context.

VOID XRestoreFloatingPointStateForDpc();

Parameters

None.

Return Values

This function does not return a value.

Remarks

The floating point processor state is not automatically saved and restored across a DPC call, so modifying the floating point state while at DPC level will normally cause crashes in a title. However, the floating point state can be explicitly saved with XSaveFloatingPointStateForDpc upon DPC entry, and restored with XRestoreFloatingPointStateForDpc before the DPC is exited, thus allowing the use of floating point operations during the DPC.

XRestoreFloatingPointStateForDpc must be called from within a DPC, and only after previously calling XSaveFloatingPointStateForDpc within that same DPC.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
