XONLINE_STAT_PROC

Contains statistical information about a player, Xbox Live team.

Note    Units, which are dynamic teams, are not supported by Xbox Live.

typedef struct{
    WORD wProcedureID;
    union {
        XONLINE_STAT_UPDATE Update;
        XONLINE_STAT_UPDATE_UNIT UpdateUnit;
        XONLINE_STAT_CONDITIONAL Conditional;
        XONLINE_STAT_CONDITIONAL_UNIT ConditionalUnit;
        XONLINE_STAT_ELO Elo;
        XONLINE_STAT_ELO_UNIT EloUnit;
        XONLINE_STAT_CUSTOM Custom;
    } ;
} XONLINE_STAT_PROC, *PXONLINE_STAT_PROC;

Members

wProcedureID

Action to perform using the statistical information. Can be the following:
Procedure ID Description
XONLINE_STAT_PROCID_UPDATE_REPLACE Replace the current stats for the player with those contained in the Update member of this structure.
XONLINE_STAT_PROCID_UPDATE_REPLACE_UNIT Not supported.
XONLINE_STAT_PROCID_UPDATE_INCREMENT Increment the current stats for the player with those contained in the Update member of this structure.
XONLINE_STAT_PROCID_UPDATE_INCREMENT_UNIT Not supported.
XONLINE_STAT_PROCID_ELO Create an ELO rating for the player using the Elo member of this structure.
XONLINE_STAT_PROCID_ELO_UNIT Not supported.
XONLINE_STAT_PROCID_CONDITIONAL Perform a conditional on the current stats for the player with those contained in the Conditional member of this structure.
XONLINE_STAT_PROCID_CONDITIONAL_UNIT Not supported.

Note    You can use custom procedure IDs in conjunction with the XONLINE_STAT_CUSTOM structure to perform title-specific operations when updating statistics.

Update
XONLINE_STAT_UPDATE structure containing updated player information.
UpdateUnit
Not supported.
Conditional
XONLINE_STAT_CONDITIONAL structure containing conditional player information.
ConditionalUnit
Not supported.
Elo
XONLINE_STAT_ELO structure containing information needed to determine an ELO rating for the player.
EloUnit
Not supported.
Custom
XONLINE_STAT_CUSTOM structure used to perform custom operations.
