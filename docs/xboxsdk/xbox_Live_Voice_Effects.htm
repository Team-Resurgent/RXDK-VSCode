Voice Effects

The Xbox Live libraries provide a set of functionality that titles can use to easily implement voice effects that alter the characteristics of the player's voice. Titles can use these voice effects to greatly enrich a game experience by changing voices to fit a given archetype or game race for the particular game universe. In a fantasy title, for example, a nice pitch-raising effect would make elf players sound like "real" elves. Titles are expected to be creative with this functionality.

Voice Masking

Xbox Live titles must use voice masking on a player's voice by default. One of the standard voice effects is a voice masking effect that removes identifiable characteristics from a player's voice, effectively preserving the player's anonymity. There will be more specific requirements in this area in the future, but at a minimum voice masking should be used for all players who don't explicitly choose to remove the effect.

Voice Recognition

Voice recognition is particularly desirable for genres of games that traditionally require heavy keyboard support, but any style of game could benefit from creative use of this technology. This functionality is included as part of the Xbox High Level Voice (XHV) interface. Information on integrating XHV into a title can be found in the following sections:

  • The SpeechRecognition sample illustrates integration of the various components of the XHV speech recognition API.
  • XHV API support is supplied in the XHV Speech Recognition Reference section.
  • Xbox Speech Recognition (.xsr) binary files may be generated using the Build Speech Recognition Bank command-line tool.

Titles may use voice chat or recognition, but not both simultaneously.
