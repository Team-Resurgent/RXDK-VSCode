CallStackLog Sample

The CallStackLog Sample demonstrates how to capture the current call stack in a running Xbox executable. The sample generates a log file with call stack information. The CallStackLookup sample, run on the development machine, reads the log file with the call stack names and generates a friendly printout.

Overview

DmCaptureStackBackTrace is used to collect an array of addresses for the current call stack. DmWalkLoadedModules is used to iterate over the modules in the XBE and determine their base addresses. This combined information is used to look up the function names in the XBE's debug information (which resides on the development PC).
