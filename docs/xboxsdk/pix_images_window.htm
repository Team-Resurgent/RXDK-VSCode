PIX Images Window

The Images window displays the different surfaces used by the title. Moving through the timeline in either the Timeline window or the Events window updates the images to reflect the current point in the timeline.

Choosing an Image

From the drop-down list in the upper left, the user can choose one of the following surfaces to display:

Render Target
Displays the current render target. While displaying the render target, two sliders appear above the surface: Current Primitives Only and Background Color. By moving the Current Primitives Only slider, you can focus on the current primitive by fading the color of all of the other primitives to the color set by the Background Color slider. By default, all primitives are shown and the background color is black.

Using the drop-down list below the image type selector, the user can choose between displaying color or alpha. Additionally, by clicking Brightness, the user can modify the gamma ramp.

Depth Buffer
Displays the current depth buffer. When the user selects Depth Buffer from the drop-down list in the upper left, an additional drop-down list appears that gives the user the option of Display Depth (the default) or Display Stencil. This menu enables the user to switch between seeing the depth buffer or the stencil buffer.

By clicking Brightness, the user can modify the gamma ramp.

Pipeline
Shows how various primitives work their way through the pipeline as they are processed by the GPU. It provides a useful way to determine if the pipeline is getting flushed between primitives due to a particular render state that is being set. For more details, see PIX Pipeline Image View.
Wireframe
Displays the current render target as though the fill mode was D3DFILL_WIREFRAME.
Texture 0
Displays the texture for stage zero.

Using the drop-down list below the image type selector, the user can choose between displaying color or alpha. Additionally, by clicking Brightness, the user can modify the gamma ramp.

Texture 1
Displays the texture for stage one.

Using the drop-down list below the image type selector, the user can choose between displaying color or alpha. Additionally, by clicking Brightness, the user can modify the gamma ramp.

Texture 2
Displays the texture for stage two.

Using the drop-down list below the image type selector, the user can choose between displaying color or alpha. Additionally, by clicking Brightness, the user can modify the gamma ramp.

Texture 3
Displays the texture for stage three.

Using the drop-down list below the image type selector, the user can choose between displaying color or alpha. Additionally, by clicking Brightness, the user can modify the gamma ramp.

All Textures
Displays all four of the texture stages.
Overdraw
Uses grayscale to show the overdraw of the scene. The lighter the area, the more overdrawn.
Effective Fill Rate
Uses grayscale to show the effective fill rate of the scene. The lighter the area, the lower the fill rate.

Note    Volume and cubemap textures do not display properly in the Images window.

Sizing the Image

The drop-down list in the middle allows the user to select how the image is scaled to the window. The user has the following options:

  • Fit To Window
  • Fill To Window
  • Actual Size
  • Scale By 2
  • Scale By 4
  • Scale By 1/2
  • Scale By 1/4

Using the Images Status Bar

The Images status bar is located below the displayed surfaces. When the mouse is over one of the surfaces, different information gets displayed, depending on the surface. If the surface is a render target, a wireframe, or a texture, the status bar shows the position of the pixel under the cursor, its ARGB color, and the size and format of the surface. If the surface is a depth or stencil buffer, the status bar shows the position of the pixel, a hexadecimal value representing the value of the buffer at that point, and the size and format of the buffer. If the surface is overdraw, or the effective fill rate, then the status bar shows the position of the pixel and either the amount of overdraw (larger number, more overdraw) or the fill rate (larger number, lower fill rate).
