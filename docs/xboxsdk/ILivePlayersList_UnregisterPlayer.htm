ILivePlayersList::UnregisterPlayer

Removes a player from the current players list.

HRESULT UnregisterPlayer(
    CONST ITitlePlayersListItem *pPlayer
);

Parameters

pPlayer
[in] Pointer to an ITitlePlayersListItem interface that represents the player to remove.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method removes a player from the current players list and adds the player to the departed list. The player should have been added to the current list with the ILivePlayersList::RegisterPlayer method before calling UnregisterPlayer.

Also, a maximum of 10 players is allowed on the departed list. If the list is full, the oldest entry is dropped to make room for the player being unregistered. Sort order has no affect on the departed list.

Lastly, the text displayed in the departed list is static (retrieved only once at the time of an unregister operation). The length of the text stored for each entry in the departed list is capped at 127 wide characters.
