XGMatrixLookAtRH

Builds a right-handed, look-at matrix.

XGMATRIX* XGMatrixLookAtRH(
  XGMATRIX* pOut,
  CONST XGVECTOR3* pEye,
  CONST XGVECTOR3* pAt,
  CONST XGVECTOR3* pUp
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
pEye
[in] Pointer to the XGVECTOR3 structure that defines the eye point. This value is used in translation.
pAt
[in] Pointer to the XGVECTOR3 structure that defines the camera look-at target.
pUp
[in] Pointer to the XGVECTOR3 structure that defines the current world's up, usually [0, 1, 0].

Return Values

Pointer to a XGMATRIX structure that is a right-handed, look-at matrix.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixLookAtRH function can be used as a parameter for another function.
