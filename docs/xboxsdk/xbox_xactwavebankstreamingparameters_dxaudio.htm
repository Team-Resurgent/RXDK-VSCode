XACT_WAVEBANK_STREAMING_PARAMETERS

Contains XACT streaming wave bank information.

typedef struct _XACT_WAVEBANK_STREAMING_PARAMETERS{
    HANDLE hFile;
    DWORD dwOffset;
    DWORD dwPacketSizeInMilliSecs;
    DWORD dwPrimePacketSizeInMilliSecs;
} XACT_WAVEBANK_STREAMING_PARAMETERS, *PXACT_WAVEBANK_STREAMING_PARAMETERS;

typedef const XACT_WAVEBANK_STREAMING_PARAMETERS *PCXACT_WAVEBANK_STREAMING_PARAMETERS;

Members

hFile
A file handle associated with the wave bank data.

Note    This handle is obtained by calling CreateFile. The FILE_FLAG_OVERLAPPED and FILE_FLAG_NO_BUFFERING flags must be specified when calling CreateFile for this file handle.

dwOffset
An offset within the file to the wave bank header.
dwPacketSizeInMilliSecs
The packet size in milliseconds used for streaming (two packets are used). This value applies to all stream types. See the XACT Concepts section for example scenarios when setting this value.
dwPrimePacketSizeInMilliSecs
The packet size in milliseconds used for zero-latency streams and loop cache packets. This value applies to zero-latency and looping streams only.

Remarks

The purpose of this structure is to allow a streaming wave bank to be embedded within a larger file containing other data (a "pack" file, for example).

Requirements
