IXVoiceEncoder

Provides access to the one of four available voice chat encoders.

Method Description
GetCodecBufferSize Returns the size of the output buffer required to store the compressed data that results from the specified amount of input data.
ProcessMultiple Compresses the data packets in the specified input array, and outputs the compressed data to the specified output array.
SetVoiceMask Applies the specified voice mask to the specified input stream of the encoder.
XMediaObject Method
Discontinuity
Flush
GetInfo
GetStatus
Process

The IXVoiceEncoder interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
