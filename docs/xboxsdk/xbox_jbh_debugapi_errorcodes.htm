Debugger API Error Codes and Return Values


The following table provides a list of Debugger API error codes and return values.

Code Name Description
0x02DB0000 XBDM_NOERR No error occurred.
0x82DB0000 XBDM_UNDEFINED An undefined error has occurred.
0x82DB0001 XBDM_MAXCONNECT The maximum number of connections has been exceeded.
0x82DB0002 XBDM_NOSUCHFILE No such file exists.
0x82DB0003 XBDM_NOMODULE No such module exists.
0x82DB0004 XBDM_MEMUNMAPPED The referenced memory has been unmapped.
0x82DB0005 XBDM_NOTHREAD No such thread ID exists.
0x82DB0007 XBDM_INVALIDCMD An invalid command was specified.
0x82DB0008 XBDM_NOTSTOPPED Thread not stopped.
0x82DB0009 XBDM_MUSTCOPY File must be copied, not moved.
0x82DB000A XBDM_ALREADYEXISTS A file already exists with the same name.
0x82DB000B XBDM_DIRNOTEMPTY The directory is not empty.
0x82DB000C XBDM_BADFILENAME An invalid file name was specified.
0x82DB000D XBDM_CANNOTCREATE Cannot create the specified file.
0x82DB000E XBDM_CANNOTACCESS Cannot access the specified file.
0x82DB000F XBDM_DEVICEFULL The device is full.
0x82DB0010 XBDM_NOTDEBUGGABLE This title is not debuggable.
0x82DB0011 XBDM_BADCOUNTTYPE The counter type is invalid.
0x82DB0012 XBDM_COUNTUNAVAILABLE Counter data is not available.
0x82DB0100 XBDM_CANNOTCONNECT Cannot connect to the target system.
0x82DB0101 XBDM_CONNECTIONLOST The connection to the target system has been lost.
0x82DB0103 XBDM_FILEERROR An unexpected file error has occurred.
0x82DB0104 XBDM_ENDOFLIST Used by the DmWalkxxx functions to signal the end of a list.
0x82DB0105 XBDM_BUFFER_TOO_SMALL The buffer referenced was too small to receive the requested data.
0x82DB0106 XBDM_NOTXBEFILE The file specified is not a valid XBE.
0x82DB0107 XBDM_MEMSETINCOMPLETE Not all requested memory could be written.
0x82DB0108 XBDM_NOXBOXNAME No target system name has been set.
0x82DB0109 XBDM_NOERRORSTRING There is no string representation of this error code.
0x02DB0001 XBDM_CONNECTED A connection has been successfully established.
0x02DB0002 XBDM_MULTIRESPONSE One of the three types of continued transactions supported by DmRegisterCommandProcessor.
0x02DB0003 XBDM_BINRESPONSE One of the three types of continued transactions supported by DmRegisterCommandProcessor.
0x02DB0004 XBDM_READYFORBIN One of the three types of continued transactions supported by DmRegisterCommandProcessor.
0x02DB0005 XBDM_DEDICATED A connection has been dedicated to a specific threaded command handler.
0x02DB0007 XBDM_FASTCAPENABLED Fast call–attribute profiling is enabled.

0x8007007BERROR_INVALID_NAMEThe file name or directory name syntax is incorrect.
0x800700B6ERROR_INVALID_ORDINALThe system software cannot find the function.

0x80070490ERROR_NOT_FOUNDElement not found.
