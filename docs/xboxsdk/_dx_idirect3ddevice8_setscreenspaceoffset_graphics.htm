IDirect3DDevice8::SetScreenSpaceOffset

Sets a screen-space offset which is applied to all rendering.

HRESULT SetScreenSpaceOffset(
  float x,
  float y
);

Parameters

x
[in] The horizontal offset.
y
[in] The vertical offset.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

An offset of (-0.5f, -0.5f) results in the same pixel fill conventions as OpenGL.

When using field rendering, SetScreenSpaceOffset may be used to apply the even-field offset as shown below.

// Assume that pDevice is a valid pointer 
// to an IDirect3DDevice8 interface.

// Handle the odd field.
pDevice->SetScreenSpaceOffset( 0.0f, 0.0f );

// Render your geometry.
// ...

pDevice->Swap( D3DSWAP_DEFAULT );

// Now handle the even field.
pDevice->SetScreenSpaceOffset( 0.0f, 0.5f );

// Render your geometry.
// ...

pDevice->Swap( D3DSWAP_DEFAULT );

When field rendering, the offset applies to non-back-buffer render targets as well, so the title will need to reset the offset when changing render targets.
