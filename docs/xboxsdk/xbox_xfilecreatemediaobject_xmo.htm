XFileCreateMediaObject

Creates and initializes a file Xbox Media Object (XMO).

HRESULT XFileCreateMediaObject(
    LPCSTR pszFileName,
    DWORD dwDesiredAccess,
    DWORD dwShareMode,
    DWORD dwCreationDisposition,
    DWORD dwFlagsAndAttributes,
    XFileMediaObject **ppMediaObject
);

Parameters

pszFileName
Null-terminated string value that specifies the file path.
dwDesiredAccess
Value that specifies the type of access to the object. Can be set to zero or one of the following values. If dwDesiredAccess is set to zero, applications will be allowed to query the device attributes without accessing the device.
Value
GENERIC_READ
GENERIC_WRITE

In addition, you can specify the following access flags.

Value
GENERIC_EXECUTE
GENERIC_ALL
DELETE
SYNCHRONIZE
STANDARD_RIGHTS_REQUIRED
STANDARD_RIGHTS_ALL

Note    These flags are rarely used or needed

dwShareMode
Value that specifies the share mode of the object. Can be set to zero or one of the values listed below. If dwShareMode is set to zero, the object cannot be shared.
Value
FILE_SHARE_DELETE
FILE_SHARE_READ
FILE_SHARE_WRITE
dwCreationDisposition
Value that specifies the creation distribution.
Value
CREATE_NEW
CREATE_ALWAYS
OPEN_EXISTING
OPEN_ALWAYS
TRUNCATE_EXISTING
dwFlagsAndAttributes
Value that specifies the attributes and flags to use.

Any combination of the following attributes is acceptable for the dwFlagsAndAttributes parameter.

Value
FILE_ATTRIBUTE_ARCHIVE
FILE_ATTRIBUTE_HIDDEN
FILE_ATTRIBUTE_NORMAL
FILE_ATTRIBUTE_READONLY
FILE_ATTRIBUTE_SYSTEM

Note    All other file attributes override FILE_ATTRIBUTE_NORMAL.

Any combination of the following flags is acceptable for the dwFlagsAndAttributes parameter.

Value
FILE_FLAG_OVERLAPPED
FILE_FLAG_NO_BUFFERING
FILE_FLAG_RANDOM_ACCESS
FILE_FLAG_SEQUENTIAL_SCAN
FILE_FLAG_DELETE_ON_CLOSE
FILE_FLAG_BACKUP_SEMANTICS
ppMediaObject
Address of a pointer to the XFileMediaObject that is created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The title is responsible for freeing the XFileMediaObject that is created by using the Release method.

For descriptions of the flags, see the CreateFile documentation.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
