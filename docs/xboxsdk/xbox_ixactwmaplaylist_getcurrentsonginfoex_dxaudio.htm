IXACTWmaPlayList::GetCurrentSongInfoEx

Retrieves information about the current song in an XACT WMA playlist.

HRESULT GetCurrentSongInfoEx(
    PXACT_WMASONG_DESCRIPTION pDesc,
    PXACTWMASONG *ppSong
);

Parameters

pDesc
[in] Pointer to an XACT_WMASONG_DESCRIPTION that describes the song to be returned.
ppSong
[out] Pointer to an IXACTWmaSong to be returned.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

GetCurrentSongInfoEx can be called at any time, even when the playlist is not playing. However, calling this function while the song is stopped may cause a block. Subsequent calls to this function will not block as long as the current song does not change.

Note    Of the calls in the XACT API, GetCurrentSongInfoEx and GetCurrentSongInfo are the only calls that may block. All other calls return immediately or are asynchronous.

At least one song must be added to the playlist for this call to succeed.

Requirements
