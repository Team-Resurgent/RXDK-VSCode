XOnlineTeamSetFamilyTitleID

Sets the family title ID for teams.

HRESULT XOnlineTeamSetFamilyTitleID(
    DWORD dwTitleID
);

Parameters

dwTitleID
[in] Family title ID to use for teams.

Return Values

Returns S_OK if successful, an error code otherwise.

XONLINE_E_INVALID_TITLE_ID is returned when title identifier supplied is neither a valid title identifier or one of the family title identifier.

Remarks

A title may call this function only once, and must call it before attempting to create a team, enumerate teams, write team stats, and so on. Using teams functionality before calling this function will result in the real title ID being used rather than the team family title ID.

Titles should also pass the team family title ID to XOnlineMessageEnableReceivingFamilyTitleIDs in order to receive notifications of incoming XONLINE_MSG_TYPE_TEAM_RECRUIT messages via XOnlineGetNotificationEx and to have those messages returned by XOnlineMessageEnumerate.

Attempting to use this function to enable a family title ID without first including that ID in the title's certificate will result in an error message displayed in the debug output as well as the function not working correctly. A family title ID can be included in a title's certificate by opening the project properties dialog box in Visual Studio, and adding the family title ID to the alternate title IDs.

Setting the family title ID will generate debug output for informational purposes.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
