XOnlineQuerySearchGetResults

Retrieves the results of the task begun by XOnlineQuerySearch.

HRESULT XOnlineQuerySearchGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwTotalResults,
    DWORD *pdwReturnedResults,
    DWORD *pdwResultsSize,
    PBYTE pbResults
);

Parameters

hTask
[in] Task handle received from XOnlineQuerySearch.
pdwTotalResults
[out] Address of a DWORD. Upon return, the DWORD holds the total number of items in search result.
pdwReturnedResults
[out] Address of a DWORD. Upon return, the DWORD holds the number of items returned by the search..
pdwResultsSize
[in, out] Address of a DWORD that holds the size, in bytes, of the buffer pointed to by pbResultsBuffer. If XOnlineQuerySearchGetResults returns E_INSUFFICIENT_BUFFER, the DWORD contains the number of bytes necssary to hold the results list. The appropriate value for this parameter can also be determined by calling XOnlineQueryGetResultsBufferSize before calling this function.
pbResults
[out] Pointer to a buffer that receives the results list. This buffer must be at least the number of bytes returned by XOnlineCompetitionGetResultsBufferSize.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
