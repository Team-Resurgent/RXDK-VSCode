XOnlineFriendsGameInvite

Sends a game invitation to a specified list of friends.

Note    It is recommended that titles support voice message attachments to game invitations by using the XOnlineGameInviteSend function.

HRESULT XOnlineFriendsGameInvite(
    DWORD dwUserIndex,
    XNKID SessionID,
    DWORD dwFriendListCount,
    const XONLINE_FRIEND *pToFriendList
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player sending the invite. The player must have been successfully logged on with a previous call to XOnlineLogon.
SessionID
[in] Game session identifier of the game for which the invitation is being sent. Obtain the session identifier by calling XOnlineMatchSessionGetInfo.
dwFriendListCount
[in] Number of friends in the pToFriendList array.
pToFriendList
[in] Pointer to an array of XONLINE_FRIEND structures specifying the friends to invite.

Return Values

If XOnlineFriendsGameInvite succeeds, the return value is S_OK.

If XOnlineFriendsGameInvite fails, the return value is one of the following error codes.

Value Description
XONLINE_E_NOTIFICATION_NO_TASK The friends startup task is not active. You must first call XOnlineFriendsStartup to create the startup task.
Other See Online Errors for information about other possible errors.

Remarks

The task returned by the XOnlineFriendsEnumerate function must be active at the time the title calls the XOnlineFriendsGameInvite function.

Pump the task returned by the XOnlineFriendsStartup function to send the invitation to the friends specified by the pToFriendList parameter.

Pump the task returned by XOnlineFriendsEnumerate to enumerate the friends list and determine the status of the sent invitations.

The friends receiving the invite indicate their responses to the receiving titles, which then calls XOnlineFriendsAnswerGameInvite to respond to the invitations.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
