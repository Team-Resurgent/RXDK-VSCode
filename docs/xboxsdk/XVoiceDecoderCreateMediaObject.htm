XVoiceDecoderCreateMediaObject

Creates an instance of the Xbox Media Object (XMO) that is used to decode voice data.

HRESULT XVoiceDecoderCreateMediaObject(
    DWORD dwCodecTag,
    LPWAVEFORMATEX pwfxDecoded,
    LPXMEDIAOBJECT *ppMediaObject
);

Parameters

dwCodecTag

Codec that the decoder should use. Can be set to one of the following values:
Value Description
WAVE_FORMAT_VOXWARE_SC03 Specifies the Voxware SC03 codec.
WAVE_FORMAT_VOXWARE_SC06 Specifies the Voxware SC06 codec.

pwfxDecoded
Address of the WAVEFORMATEX structure that specifies the wave format that the data from this XMO should be returned in.
ppMediaObject
Address of a pointer to the XMO created.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in XVoice.h.

Import Library: Use XVoice.lib.
