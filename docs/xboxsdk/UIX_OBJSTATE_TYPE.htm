UIX_OBJSTATE_TYPE

The specific object state retrieved or set with ITitleUIPlugin::GetObjectState or ITitleUIPlugin::SetObjectState.

typedef enum _UIX_OBJSTATE_TYPE{
    UIX_OBJSTATE_LIST_SELECTION_INDEX = 0,
    UIX_OBJSTATE_LIST_ITEM_GREYED = 1,
    UIX_OBJSTATE_LIST_ITEM_HIGHLIGHTED = 2,
    UIX_OBJSTATE_WORD_WRAP = 3,
    UIX_OBJSTATE_FORCE_DWORD = 0x7fffffff,
} UIX_OBJSTATE_TYPE;

Constants

UIX_OBJSTATE_LIST_SELECTION_INDEX
Selection index of a list box object.
UIX_OBJSTATE_LIST_ITEM_GREYED
Greyed state of a list item.
UIX_OBJSTATE_LIST_ITEM_HIGHLIGHTED
Highlighted state of a list item.
UIX_OBJSTATE_WORD_WRAP
Word wrapping on a text object.
UIX_OBJSTATE_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.

Remarks

The meaning of the Value (or pValue) and ItemIndex parameters of ITitleUIPlugin::GetObjectState and ITitleUIPlugin::SetObjectState vary depending on the state being referenced, as follows:

State Value ItemIndex
UIX_OBJSTATE_LIST_SELECTION_INDEX Index of the currently selected item. (ignored)
UIX_OBJSTATE_LIST_ITEM_GREYED TRUE if the item is greyed, FALSE otherwise. List item index.
UIX_OBJSTATE_LIST_ITEM_HIGHLIGHTED TRUE if the item is highlighted, FALSE otherwise. List item index.
UIX_OBJSTATE_WORD_WRAP TRUE if word wrap is enabled, FALSE otherwise. (ignored)
