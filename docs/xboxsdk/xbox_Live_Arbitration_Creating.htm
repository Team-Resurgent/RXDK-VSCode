Creating an Arbitrated Game Session

The process of creating arbitrated game sessions differs, depending on whether the session is part of a competition.

  • Creating an Arbitrated Game
  • Creating an Arbitrated Competition Round
  • After the Game is Created

Creating an Arbitrated Game

A title creates a session with matchmaking by calling the XOnlineMatchSessionCreate function, becoming the match host. Your title provides other players with the ability to discover the session and obtain a session ID using the matchmaking or presence functions.

Creating an Arbitrated Competition Round

To use arbitration in a competition, go through the normal process of configuring, creating, and scheduling the competition. Let the players check in and establish the connection between the players.

After the Game is Created

Whether a game session is a round in a competition or not, the next step in getting the game session going is the same—the title must establish peer-to-peer connections with the match host for all players. Players get together in the pregame lobby and agree on game settings. Eventually, they reach a "commit point" and the game session is about to begin. The registration process begins. At this point, players must be notified that an arbitrated session is about to begin and that disconnects will be penalized.
