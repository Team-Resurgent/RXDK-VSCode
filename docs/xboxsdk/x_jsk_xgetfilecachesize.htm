XGetFileCacheSize

The XGetFileCacheSize function returns the number of bytes currently reserved for the file system cache.

SIZE_T XGetFileCacheSize(VOID);

Parameters

This function has no parameters.

Return Value

The return value is the number of bytes currently reserved for the file system cache.

Remarks

The default cache size is 64 KB. A title can use the XSetFileCacheSize function to change the number of bytes reserved for the cache.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
