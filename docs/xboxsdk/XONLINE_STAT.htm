XONLINE_STAT

Represents a statistic that can be stored or retrieved using the statistics service.

typedef struct{
    WORD wID;
    XONLINE_STAT_TYPE type;
    union {
        LONG lValue;
        LONGLONG llValue;
        double dValue;
        LPCWSTR lpString;
    } ;
} XONLINE_STAT, *PXONLINE_STAT;

Members

wID

Title-defined identifier for this statistic, or an identifier for a service-defined statistic. The following table lists the service-defined statistics.
Identifier Description Type
XONLINE_STAT_RANK Rank statistic. Ranks have a type of XONLINE_STAT_LONG (32-bits).
XONLINE_STAT_RATING Rating statistic. Ratings have a type of XONLINE_STAT_LONGLONG (64-bits).
XONLINE_STAT_LEADERBOARD_SIZE Number of entries on the scoreboard. The scoreboard size has a type of XONLINE_STAT_LONG (32-bits).
XONLINE_STAT_ATTACHMENT_PATH Path to a statistics attachment. Attachment paths have a type of XONLINE_STAT_LPCWSTR (variable-length string).
XONLINE_STAT_ATTACHMENT_SIZE Size of a statistics attachment. Attachment sizes have a type of XONLINE_STAT_LONG (variable-length string).
XONLINE_STAT_UNIT_ACTIVITY_COUNTER Not supported. Not supported.
XONLINE_STAT_UNIT_LAST_ACTIVITY_DATE Not supported. Not supported.

Note    Leaderboards are also referred to as scoreboards.

type
Value of type XONLINE_STAT_TYPE. This value specifies the type of the statistics.
lValue
Stores a integer (32-bit) value for this statistic.
llValue
Stores a big integer (64-bit) value for this statistic.
dValue
Stores a real number (64-bit) value for this statistic.
lpString
Stores a variable-length string. This type should be used only for storing player nicknames and storing paths to statistics attachments.
