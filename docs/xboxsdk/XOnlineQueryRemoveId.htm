XOnlineQueryRemoveId

Enables a title to invoke a query that removes an entity (row) from the specified data set. 

HRESULT XOnlineQueryRemoveId(
    DWORD dwUserIndex,
    ULONGLONG qwTeamId,
    DWORD dwDatasetId,
    XENTITY_ID entityId,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE *phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.
qwTeamId
[in] Team ID. Should be specified only if the entries to be removed are associated with a team. Otherwise, this value should be 0.
dwDatasetId
[in] ID of the dataset to query.
entityId
[in] Entity ID that selects the row to be removed.
hWorkEvent
[in, optional] Handle to an event you want to associate with this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by XOnlineQueryRemoveId.

Return Values

Returns S_OK if successful, or an error code otherwise. The error codes can include the following.

Error Code Description
XONLINE_E_QUERY_ENTITY_NOT_FOUND The specified entity ID was not found in the dataset.
XONLINE_E_QUERY_DATASET_NOT_FOUND The specified dataset ID was not found.
XONLINE_E_QUERY_PROCEDURE_NOT_FOUND The specified procedure index was not found.
XONLINE_E_QUERY_PERMISSION_DENIED The entity was not added to the dataset by the specified player or team.

Remarks

Entities can be removed only by the same player or team that added them.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
