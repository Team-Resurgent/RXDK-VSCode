IXACTSoundSource::SetMixBins

Sets the mixbins of an IXACTSoundSource object.

HRESULT SetMixBins(
    LPCDSMIXBINS pMixBins
);

Parameters

pMixBins
[in] Address of a DSMIXBINS structure that specifies the mixbins to assign to this buffer. This parameter can be NULL, in which case the default mixbins will be used.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
