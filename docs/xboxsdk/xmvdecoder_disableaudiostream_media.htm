XMVDecoder::DisableAudioStream

Disables an audio stream in an Xbox Media Video (XMV) file.

void DisableAudioStream(
    DWORD AudioStream
);

Parameters

AudioStream
[in] Number of the audio stream to disable.

Return Values

None.

Remarks

If the audio stream getting disabled is the current sychronization stream, then the video will play back unsynchronized until either of the following occur: 1) another audio stream is enabled, or 2) the sychronization is explicitly set to a different stream.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.
