XONLINE_NOTIFICATION_EX_INFO

typedef struct{
    BYTE bMessageType;
    DWORD dwMessageID;
    DWORD dwNotifyFlags;
} XONLINE_NOTIFICATION_EX_INFO, *PXONLINE_NOTIFICATION_EX_INFO;

Members

bMessageType
Value indicating the message type:
Message Type
XONLINE_MSG_TYPE_TITLE_CUSTOM
XONLINE_MSG_TYPE_FRIEND_REQUEST
XONLINE_MSG_TYPE_GAME_INVITE
XONLINE_MSG_TYPE_TEAM_RECRUIT
XONLINE_MSG_TYPE_COMP_REMINDER
XONLINE_MSG_TYPE_COMP_REQUEST
XONLINE_MSG_TYPE_LIVE_MESSAGE
dwMessageID
ID of the message. When automatic summary refresh is not disabled, this message ID can passed to XOnlineMessageSummary to retrieve the complete message summary.
dwNotifyFlags
Additional flags for the message:
Flag Description
XONLINE_MSG_FLAG_HAS_VOICE Message contains a voice attachment.
XONLINE_MSG_FLAG_HAS_TEXT Message contains a text body.
XONLINE_MSG_FLAG_ALTERNATE_TITLE Message is from an alternate title.
XONLINE_MSG_FLAGS_TITLE_RESERVED Flags reserved for title custom messages (all bits returned as-is).
