IDirect3DPushBuffer8::BeginFixup

Begins a fix-up bracket.

HRESULT BeginFixup(
  LPDIRECT3DFIXUP8 pFixup,
  BOOL NoWait
);

Parameters

pFixup
[in, out] If NULL, any subsequent fix-ups are applied immediately to the push-buffer. If pFixup is non-NULL, any subsequent fix-ups are appended to the specified fix-up object.
NoWait
[in] If pFixup is non-NULL, this is ignored; otherwise, if NoWait is FALSE, then IDirect3DPushBuffer8::BeginFixup will block until the push-buffer is no longer referenced by the GPU, before allowing any fix-up modifications to begin. If pFixup is NULL and NoWait is TRUE then no blocking will occur.

Return Values

If this method succeeds, it returns S_OK.

Remarks

IDirect3DPushBuffer8::EndFixup should be called when done creating the fix-ups.
