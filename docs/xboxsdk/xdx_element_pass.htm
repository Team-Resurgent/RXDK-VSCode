pass Element

The <pass> element is a container for all of the state needed to render a single pass of an object.

<pass/>

Required Attributes

None.

Optional Attributes

None.

Element Information

Parent Elements model
Child Elements draw, material, matrixMapping, pixelShaderPass, renderState, vertexShaderPass

Remarks

Each <model> can have multiple <pass> tags.
