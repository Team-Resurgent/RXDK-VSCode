IDirectSound8::GetEffectData

Retreives the data for the specified effect.

HRESULT GetEffectData(
    DWORD dwEffectIndex,
    DWORD dwOffset,
    LPVOID pvData,
    DWORD dwDataSize
);

Parameters

dwEffectIndex
Value that specifies the index of the effect.
dwOffset
Value that specifies the offset, in bytes, of the start of the update region.
pvData
Address of a buffer that receives the data.
dwDataSize
Value that specifies the buffer size, in bytes.

Return Values

Returns S_OK if successful, an error code otherwise.
