IXHVEngine::SetMixBinMapping

Sets the mixbins to use when sending voice data from a specified source remote talker to a specified destination (local talker or speaker output).

HRESULT SetMixBinMapping(
    XUID xuidRemoteTalker,
    DWORD dwLocalPort,
    const DSMIXBINS *pMixBins
);

Parameters

xuidRemoteTalker
[in] The XUID of the registered remote talker.
dwLocalPort
[in] Controller port of the local talker. To set speaker playback mixbins, specify XHV_PLAYBACK_TO_SPEAKERS.
pMixBins
[in] Pointer to a DSMIXBINS structure that specifies the mixbins to use when sending voice data from the remote talker specified in xuidRemoteTalker to the destination specified in dwLocalPort.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

For local talkers, the DSMIXBINS structure can contain only XHV_LOCAL_TALKER_MAX_MIXBINS mixbin/volume pairs (the dwMixBinCount member must be less than or equal to XHV_LOCAL_TALKER_MAX_MIXBINS). For speaker output, the maximum number of mixbin/volume pairs is XHV_SPEAKERS_MAX_MIXBINS.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
