XOpenSoundtrackSong

The XOpenSoundtrackSong function opens a Microsoft® Windows Media® Audio (WMA) song from the Dashboard Music directory.

HANDLE WINAPI XOpenSoundtrackSong(
  DWORD dwSongId,
  BOOL fAsyncMode
);

Parameters

dwSongId
[in] Identifier of the song to open. This identifier must have been returned by a previous call to XGetSoundtrackSongInfo.
fAsyncMode
[in] Flag indicating whether the song should be opened in synchronous or asynchronous mode.

fAsyncMode should be TRUE if XWmaDecoderCreateMediaObject is used to create an Xbox Media Object to decode the file, or if the title implements its own in-memory decoder and reads the song file asynchronously.

If WmaCreateDecoder is used to create an Xbox Media Object to decode the file, fAsyncMode must be FALSE.

Return Values

If successful, the function returns a valid file handle to the WMA song represented by dwSongId; otherwise, it returns INVALID_HANDLE_VALUE.

Remarks

When the handle to the song is no longer needed, close it by calling CloseHandle.

Requirements

Header: Declared in xbox.h; for include use xtl.h.
Import Library: Use xapilib.lib.
