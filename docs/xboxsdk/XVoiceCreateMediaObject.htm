XVoiceCreateMediaObject

Creates an Xbox Media Object (XMO) for a device on the Xbox Communicator.

HRESULT XVoiceCreateMediaObject(
    PXPP_DEVICE_TYPE XppDeviceType,
    DWORD dwPort,
    DWORD dwMaxAttachedPackets,
    LPCWAVEFORMATEX pwfxFormat,
    LPXMEDIAOBJECT *ppMediaObject
);

Parameters

XppDeviceType

[in] Device to connect to on the Xbox Communicator.

Can be one of the following values:

Value Description
XDEVICE_TYPE_VOICE_HEADPHONE Specifies the headphone on the Xbox Communicator.
XDEVICE_TYPE_VOICE_MICROPHONE Specifies the microphone on the Xbox Communicator.

dwPort

[in] Port that the device is plugged into. Can be a value ranging from zero to three, or one of the following values:
Value Description
XDEVICE_PORT0 The leftmost controller port (port 0).
XDEVICE_PORT1 The second controller port from the left (port 1).
XDEVICE_PORT2 The third controller port from the left (port 2).
XDEVICE_PORT3 The rightmost controller port (port 3).

dwMaxAttachedPackets
[in] Maximum size of the object's internal queue.
pwfxFormat

[in, optional] Address of a WAVEFORMATEX structure that specifies the wave format that should be used by this device.

Note    The only format that should be used for input and output is mono, 8Khz with 16-bit samples.

ppMediaObject
[out] Address of a pointer to the XMO created for the device.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in XVoice.h.

Import Library: Use XVoice.lib.
