vregVB Element

The <vregVB> element specifies how vertex data is mapped to a particular vertex register.

<vregVB
   name = string
   format = string
   inputFormat = string
/>

Required Attributes

name

Describes what the element represents in the vertex. Although the name can be any string, the Xbox Preview Pipeline defines the following names:

Value Description
POSITION Position of the vertex.
BONEWEIGHTS Bone weights used to influence the vertex.
BONEMATRICES Indices used to reference the bones that influence the vertex.
NORMAL Normal vector for the vertex.
TEXCOORD0 Texture coordinates for the vertex.
TANGENT Tangent vector for the vertex.
BINORMAL Binormal vector for the vertex.
SxT Result of S×T for the vertex.

The name attribute is used when assigning a vertex buffer to a vertex shader input stream.

format

Describes the format of the data getting mapped to the register given in name. This is the output format used on the Xbox, and must be one of the following values: FLOAT1, FLOAT2, FLOAT3, FLOAT4, D3DCOLOR, NORMSHORT1, NORMSHORT2, NORMSHORT3, NORMSHORT4, NORMPACKED3, SHORT1, SHORT2, SHORT3, SHORT4, PBYTE1, PBYTE2, PBYTE3, PBYTE4, FLOAT2H. These correspond to the D3DVSDT_xxxx format types used for D3DVSD_REG. See D3DVSD_REG for more details.

Optional Attributes

inputFormat

Describes the format of the input XDX file and allows for remapping of the input data to packed output data. For example, a normal in FLOAT3 format can be packed as NORMPACKED3 in the compiled output file. If the inputFormat attribute is not specified, it is assumed to be the same as the output format given in the format attribute. The inputFormat attribute uses the same values as format.

Element Information

Parent Elements declVB
Child Elements None.

Remarks

The order in which <vregVB> elements appear within their parent <declVB> element is very important. The Xbox Resource Compiler assumes that the vertex data will appear in the same order as the tags. For an example showing how this element may be used, see Setting Up a Vertex Buffer.
