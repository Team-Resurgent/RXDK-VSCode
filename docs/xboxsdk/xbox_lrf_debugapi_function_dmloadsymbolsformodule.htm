DmLoadSymbolsForModule

Loads debugging symbols from a .pdb file for a specified module.

DMHRAPI DmLoadSymbolsForModule(
    CONST CHAR *ModuleName,
    ULONG_PTR BaseAddress,
    SIZE_T Size,
    ULONG TimeStamp
);

Parameters

ModuleName
[in] Pointer to a null-terminated string that specifies the name of the module for which symbols will be loaded.
BaseAddress
[in] Base address of the module.
Size
[in] Size, in bytes, of the module.
TimeStamp
[in] Time stamp of the module.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
