IDirectMusicPerformance8::CloseDown

Closes down the performance object. An application that created the performance object and called IDirectMusicPerformance8::InitAudioX on it must call CloseDown before the performance is released.

HRESULT CloseDown();

Parameters

None.

Return Values

The method returns S_OK.

Remarks

Failure to call CloseDown can cause memory leaks or program failures.

If your application has obtained any interfaces to DirectSound buffers, it should release them before calling CloseDown.

The method releases any downloaded instruments that have not been unloaded.
