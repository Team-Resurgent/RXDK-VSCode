XOnlineCompetitionSubmitResults

Submits the results of a competition.

HRESULT XOnlineCompetitionSubmitResults(
    DWORD dwTemplate,
    ULONGLONG qwCompetitionID,
    const XONLINE_ARB_ID *pArbitrationID,
    DWORD dwArbitrationReportFlags,
    const XONLINE_ARB_REPORT_DATA *pArbitrationReportData,
    DWORD dwNumStatProcs,
    const XONLINE_STAT_PROC *pStatProcs,
    DWORD dwNumAttributes,
    const XONLINE_ATTRIBUTE *pAttributes,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwTemplate
[in] ID of the template for the competition. Obtain this ID from XLAST.
qwCompetitionID
[in] Unique ID assigned to the competition.
pArbitrationID
[in] Pointer to an XONLINE_ARB_ID containing the same session and round IDs used in the call to XOnlineCompetitionSessionRegister.
dwArbitrationReportFlags
[in] Arbitration report flags. The flags must include XONLINE_ARB_REGISTER_FLAG_USER_COMPETITION or XONLINE_ARB_REGISTER_FLAG_PUBLISHER_COMPETITION, depending on whether the competition is created by a user or a publisher. For more information, see the dwFlags parameter of the XOnlineArbitrationReport function.
pArbitrationReportData
[in] Array of XONLINE_ARB_REPORT_DATA structures that are used to report the competition event results for each player. This parameter is optional and can be set to NULL.
dwNumStatProcs
[in] Number of structures in the array specified by the pStatProcs parameter.
pStatProcs
[in] Array of XONLINE_STAT_PROC structures that are used to report the statistics for each player.
dwNumAttributes
[in] Number of attribute entries in the array specified by the pAttributes parameter.
pAttributes

[in] Array of XONLINE_ATTRIBUTE structures that specify the additional attributes to report on. This array MUST contain at least the following three integer attributes, in this order:
Attribute ID Description
XONLINE_COMP_ATTR_EVENT_ENTITY_ID The ID of the event for which results are to be submitted.
XONLINE_COMP_ATTR_EVENT_WINNER The PUID of the winner of the event.
XONLINE_COMP_ATTR_EVENT_LOSER The PUID of the loser of the event

In addition to these attributes, any additional custom attributes configured on the competition's Events table via XLAST can be included here.

hWorkEvent
[in, optional] Handle of an event that you want to associate with this task. If you do not want to associate a handle with this event, set this parameter to NULL.
phTask
[out] Address of an XONLINETASK_HANDLE variable. Upon return, the XONLINETASK_HANDLE variable contains the handle of the task initiated by this function.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements
