ILiveFriendsList

Titles use the methods of the ILiveFriendsList interface to access a player's friends list when the friends feature is enabled but not actively displayed.

The ILiveFriendsList interface is obtained by calling ILiveEngine::GetFriendsForUser.

Method Description
Count Returns the count of friends in the friends list.
GetFriendByIndex Retrieves detailed information for the friend at the specified index in the friends list.
GetFriendByXUID Retrieves detailed information for the friend with the specified user identifier.
HasChanged Indicates whether the friends list has changed since Refresh was last called.
IsReady Indicates whether the friends list has initialized and is ready to service requests.
Refresh Retrieves the latest, up-to-date friends list.
Remove Removes a friend from the player's friends list.
Request Sends a request to another player asking that the two players be placed on each other's friends list.

The ILiveFriendsList interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
