D3DPERF_QueryRepeatFrame

The D3DPERF_QueryRepeatFrame is used to determine if a title using PIX needs to re-render an exact duplicate of a frame. If D3DPERF_QueryRepeatFrame determines that re-rendering is needed, PIX can combine timing and pushbuffer data for a frame into a single comprehensive view.

BOOL D3DPERF_QueryRepeatFrame();

Parameters

None.

Return Values

Returns TRUE right after a call to Present/Swap (up until Present/Swap is called again) to denote that PIX is requesting that the current frame be rendered as an exact duplicate of the previous frame.

Remarks

See Synchronizing Pushbuffer and Timing Data for more detail on using this function.

Requirements

Header: Declared in D3d8perf.h.

Import Library: Use Use d3d8i.lib or D3d8d.lib.
