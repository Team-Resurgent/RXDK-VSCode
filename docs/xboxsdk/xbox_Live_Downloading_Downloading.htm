Downloading, Installing, and Removing Content Packages

If your title uses the Content Download Manager, it does not need to take independent action to download content packages. The Content Download Manager handes the download automatically. However, if you need the ability to customize the download process, you can do so by having your title call the XOnlineContentInstall function. XOnlineContentInstall downloads and installs a specified offering. While the download and installation is in progress, your game can call XOnlineContentInstallGetProgress to determine the progress of the content download. Typically, your game uses this information to display a progress bar, the percentage downloaded, or the time remaining.

When XOnlineTaskContinue returns XONLINETASK_S_SUCCESS, the task has completed successfully. The content in the specified offering has been downloaded from the online server and installed onto the user's local Xbox hard disk. The title should then verify that the content was correctly downloaded and installed, and that it is valid, signed content. Use XLoadContentSignatures to obtain access to signature data for installed content. Additionally, the title should always perform verification before using installed content in actual gameplay (including in subsequent game sessions) to verify that the content has not been altered.

Your title can call the XGetContentInstallLocationFromIDs to determine the directory where the content was installed, enabling your title to find the content for use. Additionally, installed content can be enumerated with XFindFirstContent.

To remove installed content, call XRemoveContent, which removes all data associated with the offering.
