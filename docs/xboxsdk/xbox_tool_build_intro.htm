Build, Submission, and Certification Tools

This section provides information regarding the build and certification tools in this version of the Xbox Development Kit (XDK). The tools described in this section are subject to change in later versions of the XDK.

The tools in this section include:

  • Build Speech Recognition Bank (BuildSpeechRecBank)
  • BuildOffer Tool (BuildOffer)
  • Bundler Tool (Bundler)
  • Content/Demo for Xbox (CDX)
  • Downloader Enum BLOB Tool (DlEnum)
  • Electronic Submission Package CRC Checker (XbPkgCRC)
  • Image Certification Tool (ImageCert)
  • VerifierX Analyzer (VXA)
  • XBE Certificate Tool (XBECert)
  • XBECopy Tool
  • XBE Image Builder Tool (ImageBld)
  • XBldSigs Tool
  • Xbox Game Disc Authoring Tool (XbGameDisc)
  • Xbox Game Disc Format Utility (GDFUtil)
  • Xbox Install Content Tool (XbInstCont)
  • Xbox Live Authoring and Submission Tool (XLAST)
  • Xbox Save Game Corruption Tool (XbSGCorrupt)
  • Xbox Tape CRC Tool (XboxCRCTape)
  • XMVEncoder Tool (XMVEncoder)

Note You can also find tools that help you prepare for certification in the XDK Launcher. For more information, see Certification Tools.
