DmStartProfile

Registers profiler interrupt handlers.

DMHRAPI DmStartProfile(
  PHANDLE pHandle    , 
  ULONG   nFreq , 
  PDMPROFILE_HANDLER pfnHandler
  );

Parameters

pHandle
Pointer to memory that will receive the handle for this profiler interrupt handler.
nFreq
Frequency of the interrupt in 100-nanosecond intervals (so 10,000 would be one millisecond). DmStartProfile doesn't support arbitrary profile intervals, it just picks the closest available supported interval. There is no mechanism for advising how often the interrupt actually fires.
pfnHandler
Pointer to the profiler interrupt handler. Profiler interrupt handlers must take the following form: 
void __stdcall my_handler(PDMN_PROFINT pdata);

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements
