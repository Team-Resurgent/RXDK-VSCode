D3DTILE_ZENDTAG

Returns an index one past the last tag bit used by the specified tile. This macro has been provided as a convenience to help set the ZStartTag member of D3DTILE. Tiles are GPU views that significantly improve rendering speed for render targets and depth buffers by reorganizing memory to improve bandwidth, and by allowing depth buffers to be compressed.

#define D3DTILE_ZENDTAG(pTile) ((((pTile)->ZStartTag + ((pTile)->Size / D3DTILE_TAGSIZE)) + 255) & ~255)

Parameters

pTile.
Pointer to an initialized D3DTILE structure.

Return Values

Returns an index one past the last tag bit used by the specified tile.
