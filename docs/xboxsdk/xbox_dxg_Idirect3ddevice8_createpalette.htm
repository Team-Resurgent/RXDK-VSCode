IDirect3DDevice8::CreatePalette

Creates a palette resource.

HRESULT CreatePalette(
D3DPALETTESIZE Size,
  D3DPalette** ppPalette
);

Parameters

Size
[in] Specifies the number of entries to be placed in the palette. Any of the following constants are valid values:
D3DPALETTE_256
D3DPALETTE_128
D3DPALETTE_64
D3DPALETTE_32
ppPalette
[out, retval] Address of a pointer to the returned IDirect3DPalette8 interface, representing the created palette.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value is E_OUTOFMEMORY.
