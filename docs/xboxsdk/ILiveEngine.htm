ILiveEngine

Titles use the methods of the ILiveEngine interface to access the Xbox Drop-in UI engine and perform tasks such as enabling features, retrieving notifications, and rendering the UI.

The ILiveEngine interface is obtained by calling the UIXCreateLiveEngine function.

The ILiveEngine interface should not be released while a feature is active. The ILiveEngine::EndFeature method should be called before calling the engine's Release method.

Method Description
ClearInput Clears the current input from the Xbox controller or controllers to the ILiveEngine interface.
DoWork Performs work within the Xbox Drop-in UI engine, and informs the title of further steps to perform.
EnableFeature Enables an Xbox Drop-in UI feature.
EndFeature Ends the currently active feature and frees associated resources.
GetExitInfo Returns exit information for the most recently exited feature.
GetFeatureInterface Retrieves a feature interface for a specified feature.
GetFriendsForUser Retrieves an ILiveFriendsList interface representing the friends list for a specified player, and begins friends list enumeration for that player.
GetNotifications Determines pending notifications for a specified player.
GetProperty Retrieves the current value of an ILiveEngine property.
GetSelectionInfo Retrieves information about the current selection from the ILiveEngine interface.
LogOff Logs off from the Xbox Live service, including logging off all currently logged on players.
NotificationSetState Sets the current online state information for the specified player.
Reboot Reboots the Xbox console.
Render Renders active Xbox Drop-in UI screens to the specified surface.
SetAudioPlugin Sets the ITitleAudioPlugin used by the Xbox Drop-in UI engine. This can be the default audio plug-in obtained with UIXCreateAudioPlugin, or a custom plug-in implemented by the title.
SetInput Forwards controller input received by the title on to the Xbox Drop-in UI engine.
SetProperty Sets the current value of an ILiveEngine property.
SetUIPlugin Sets the ITitleUIPlugin used by the Xbox Drop-in UI engine. This can be the default UI plug-in obtained with UIXCreateUIPlugin, or a custom plug-in implemented by the title.
StartFeature Starts an Xbox Drop-in UI feature.
UseVoiceMail Enables voice message options for game invites and friend requests.

The ILiveEngine interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
