DirectSoundCreate

Creates and initializes an object that supports the IDirectSound8 interface.

HRESULT DirectSoundCreate(
    LPGUID pguidDeviceId,
    LPDIRECTSOUND8 *ppDirectSound,
    LPUNKNOWN pUnkOuter
);

Parameters

pguidDeviceId
Reserved. Must be set to NULL.
ppDirectSound
Address of a variable to receive an IDirectSound8 interface pointer.
pUnkOuter
Reserved. Must be set to NULL.

Return Values

Returns DS_OK if successful, an error code otherwise. Can be one of the following values:

DSERR_NOAGGREGATION
DSERR_NODRIVER
DSERR_OUTOFMEMORY

Remarks

DirectSoundCreate cannot be called prior to main() (i.e. from a constructor on a global object) as internal state information has not been fully initialized yet.
