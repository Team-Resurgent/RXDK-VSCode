XONLINE_NAT_TYPE

Defines the type of network address translation (NAT).

typedef enum {
    XONLINE_NAT_OPEN = 1,
    XONLINE_NAT_MODERATE,
    XONLINE_NAT_STRICT,
} XONLINE_NAT_TYPE;

Constants

XONLINE_NAT_OPEN
NAT type is open. Players behind these NAT types can usually connect to any other player.
XONLINE_NAT_MODERATE
NAT type is moderate. Players behind these NAT types can usually connect to others behind XONLINE_NAT_OPEN or XONLINE_NAT_MODERATE NAT types.
XONLINE_NAT_STRICT
NAT type is strict. Players behind this NAT type frequently experience connectivity problems with others unless those players are behind XONLINE_NAT_OPEN type devices.
