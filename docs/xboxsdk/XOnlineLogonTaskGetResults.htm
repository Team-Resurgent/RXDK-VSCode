XOnlineLogonTaskGetResults

Returns the result of a completed XOnlineLogon or XOnlineSilentLogon call.

HRESULT XOnlineLogonTaskGetResults(
    XONLINETASK_HANDLE hLogonTask
);

Parameters

hLogonTask

[in] Online task handle returned by the call to XOnlineLogon.

Return Values

TheXOnlineLogonTaskGetResults function returns one of the following values. The return value is the result status of the completed XOnlineLogon call.

Value Description
XONLINE_S_LOGON_CONNECTION_ESTABLISHED The logon was successful.
XONLINE_E_LOGON_CANNOT_ACCESS_SERVICE One or more problems were encountered trying to log onto the Xbox Live service. The player should be given the option to reboot into the network troubleshooting portion of the Xbox Dashboard.
XONLINE_E_LOGON_CONNECTION_LOST The connection to the service was lost. Assume all socket connections are lost, and close the logon task. A new call may be made to XOnlineLogon if desired.
XONLINE_E_LOGON_INVALID_USER A logon could not be performed because of problems with one or more of the specified player accounts. The player should be given the option to reboot into the accounts management section of the Xbox Dashboard to resolve the problem.
XONLINE_E_LOGON_KICKED_BY_DUPLICATE_LOGON The player accounts currently logged on from this console has logged onto the service from another console. The player logged onto this console has been logged off.
XONLINE_E_LOGON_SERVERS_TOO_BUSY The logon servers are currently too busy to perform the logon request. The player should be instructed to try to log on again later.
XONLINE_E_LOGON_UPDATE_REQUIRED The title should call XOnlineTitleUpdate to update the title.

Remarks

Once the XOnlineLogon task has completed (XOnlineTaskContinue returns a value other than XONLINETASK_S_RUNNING for that task), the results of the logon can be obtained by calling XOnlineLogonTaskGetResults. This function may return more complete status information than XOnlineTaskContinue did.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
