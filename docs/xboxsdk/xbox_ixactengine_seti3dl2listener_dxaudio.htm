IXACTEngine::SetI3dl2Listener

Sets the XACT Engine Interactive 3D Audio Level 2 (I3DL2) listener properties.

HRESULT SetI3dl2Listener(
    LPCDSI3DL2LISTENER pds3dl,
    DWORD dwApply
);

Parameters

pds3dl
[in] Address of a DSI3DL2LISTENER that specifies the properties of the listener.
dwApply
[in] Value that specifies when the setting should be applied. Can be one of the following values:
Value Description
DS3D_DEFERRED Settings are not applied until the application calls the CommitDeferredSettings method. This allows the application to change several settings and generate a single recalculation.
DS3D_IMMEDIATE Settings are applied immediately, causing the system to recalculate the 3D coordinates for all 3D sound buffers.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

This function is identical to IDirectSound8::SetI3DL2Listener. See this section for additional information on setting listener properties.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
