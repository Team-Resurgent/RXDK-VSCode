XFONT_OpenDefaultFont

Loads the default bitmap font that is compiled directly into the library.

HRESULT __stdcall XFONT_OpenDefaultFont (
  XFONT **ppFont
);

Parameters

ppFont
[out] Address of a pointer to the returned XFONT interface, representing the opened font.

Return Values

On success, the function returns S_OK.

Remarks

The default font is an arial font, 24 pixels high, and only contains the western character set. The default font is intended as a convenience for titles that just need a quick way of displaying information.

Each time XFONT_OpenDefaultFont is called, it allocates approximately 112 bytes fto store the font's current state. No additional memory will be allocated throughout the lifetime of the font.
