XInputGetDeviceDescription

The XInputGetDeviceDescription function retrieves the vendor identifier, product identifier, and version number of the specified device. Currently, this function supports only lightgun devices.

DWORD XInputGetDeviceDescription(
    HANDLE hDevice,
    PXINPUT_DEVICE_DESCRIPTION pDescription
);

Parameters

hDevice
[in] Handle to a lightgun device previously opened with XInputOpen.
pDescription
[out] Pointer to an XINPUT_DEVICE_DESCRIPTION structure that receives the vendor, product, and version information.

Return Values

If the function succeeds, the return value is ERROR_SUCCESS.

If the device is not connected, the return value is ERROR_DEVICE_NOT_CONNECTED.

If the device does not support description information, the return value is ERROR_NOT_SUPPORTED.

Remarks

This function can be used to store, along with lightgun calibration data, the vendor, product, and version of that lightgun. Calibration data stored for a particular vendor/product/version combination can be used for other lightguns that return that same combination, as the lightguns should be sufficiently similar that the calibration sets are roughly equivalent.

Note    Calling XInputGetDeviceDescription blocks for about two or three milliseconds.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
