DmOpenSecureConnection

Opens a secure connection to an Xbox development kit so that commands may be sent.

DMHRAPI DmOpenSecureConnection(
  PDM_CONNECTION *ppConnection
  LPCSTR szPassword
  );

Parameters

ppConnection
Pointer to a PDM_CONNECTION that will receive the connection.
szPassword
String that contains the password used to log on.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
