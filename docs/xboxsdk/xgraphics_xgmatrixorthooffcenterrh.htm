XGMatrixOrthoOffCenterRH

Builds a customized, right-handed orthogonal projection matrix.

XGMATRIX* XGMatrixOrthoOffCenterRH(
  XGMATRIX* pOut,
  FLOAT l,
  FLOAT r,
  FLOAT b,
  FLOAT t,
  FLOAT zn,
  FLOAT zf
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
l
[in] Minimum x-value of view-volume.
r
[in] Maximum x-value of view-volume.
b
[in] Minimum y-value of view-volume.
t
[in] Maximum y-value of view-volume.
zn
[in] Minimum z-value of the view volume.
zf
[in] Maximum z-value of the view volume.

Return Values

Pointer to a XGMATRIX structure that is a customized, right-handed orthogonal projection matrix.

Remarks

An orthogonal matrix is an invertible matrix for which the inverse of the matrix is equal to the transpose of the matrix.

The XGMatrixOrthoRH function is a special case of the XGMatrixOrthoOffCenterRH function. To create the same projection using XGMatrixOrthoOffCenterRH, use the following values: l = -w/2, r = w/2, b = -h/2, and t = h/2.

All the parameters of the XGMatrixOrthoOffCenterRH function are distances in camera-space. The parameters describe the dimensions of the view-volume.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixOrthoOffCenterRH function can be used as a parameter for another function.
