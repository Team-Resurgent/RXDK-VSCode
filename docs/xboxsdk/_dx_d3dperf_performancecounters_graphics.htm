D3DPERF_PerformanceCounters

Enumeration of all of the non-Direct3D events that the driver counts.

typedef enum _D3DPERF_PerformanceCounters
{
    PERF_VERTICES,

    PERF_SETTRANSFORM_WORLD,
    PERF_SETTRANSFORM_VIEW,
    PERF_SETTRANSFORM_PROJECTION,
    PERF_SETTRANSFORM_TEXTURE,

    PERF_PUSHBUFFER_SEGMENTS,
    PERF_PUSHBUFFER_WAITS,
    PERF_OBJECTLOCK_WAITS,
    PERF_PRESENT_WAITS,
    PERF_D3DDEVICE_BLOCKUNTILIDLE_WAITS,
    PERF_D3DDEVICE_BLOCKUNTILVERTICALBLANK_WAITS,
    PERF_D3DDEVICE_BLOCKONFENCE_WAITS,
    PERF_CPUSPINDURINGWAIT_TIME,
    PERF_UNSWIZZLING_TEXTURE_LOCKS,
    PERF_PUSHBUFFER_JUMPTOBEGINNING,
    PERF_RUNPUSHBUFFER_BYTES,

    PERF_REDUNDANT_SETRENDERSTATE,
    PERF_REDUNDANT_SETTEXTURESTATE,
    PERF_REDUNDANT_SETVERTEXSHADER,
    PERF_REDUNDANT_SETTRANSFORM,
    PERF_REDUNDANT_SETTEXTURE,
    PERF_REDUNDANT_SETPALETTE,
    PERF_REDUNDANT_SETSTREAMSOURCE,

    PERF_SETSTATE_POINTPARAMS,
    PERF_SETSTATE_COMBINERS,
    PERF_SETSTATE_TEXTURESTATE,
    PERF_SETSTATE_SPECFOGCOMBINER,
    PERF_SETSTATE_TEXTRANSFORM,
    PERF_SETSTATE_LIGHTS,
    PERF_SETSTATE_TRANSFORM,

    D3DPERF_MAX,
    D3DPERF_FORCE_LONG  =   0xFFFFFFFF

} D3DPERF_PerformanceCounters;

Constants

PERF_VERTICES
Number of vertices that have been processed.
PERF_SETTRANSFORM_WORLD
Number of times the world transformation has been set.
PERF_SETTRANSFORM_VIEW
Number of times the view transformation has been set.
PERF_SETTRANSFORM_PROJECTION
Number of times the projection transformation has been set.
PERF_SETTRANSFORM_TEXTURE
Number of times a transformation matrix has been set for a texture stage.
PERF_PUSHBUFFER_SEGMENTS
Number of push-buffer segments.
PERF_PUSHBUFFER_WAITS
Number of push-buffer waits.
PERF_OBJECTLOCK_WAITS
Number of object-lock waits.
PERF_PRESENT_WAITS
Number of present waits.
PERF_D3DDEVICE_BLOCKUNTILIDLE_WAITS
Number of IDirect3DDevice8::BlockUntilIdle waits.
PERF_D3DDEVICE_BLOCKUNTILVERTICALBLANK_WAITS
Number of IDirect3DDevice8::BlockUntilVerticalBlank waits.
PERF_D3DDEVICE_BLOCKONFENCE_WAITS
Number of IDirect3DDevice8::BlockOnFence waits.
PERF_CPUSPINDURINGWAIT_TIME
Number of cycles spent spinning, waiting for the GPU.
PERF_UNSWIZZLING_TEXTURE_LOCKS
Number of unswizzling texture locks.
PERF_PUSHBUFFER_JUMPTOBEGINNING
Number of times jumped to the beginning of the push-buffer.
PERF_RUNPUSHBUFFER_BYTES
Number of bytes traversed in user-created push-buffers during calls to IDirect3DDevice8::RunPushBuffer. (Only accounts for top-level calls to RunPushBuffer; bytes traversed by calling RunPushBuffer from within a user-created push-buffer are not included.)
PERF_REDUNDANT_SETRENDERSTATE
Number of non-redundant calls to IDirect3DDevice8::SetRenderState. For more information about redundant calls, see the Remarks section.
PERF_REDUNDANT_SETTEXTURESTATE
Number of non-redundant calls to IDirect3DDevice8::SetTextureStageState. For more information about redundant calls, see the Remarks section.
PERF_REDUNDANT_SETVERTEXSHADER
Number of non-redundant calls to IDirect3DDevice8::SetVertexShader. For more information about redundant calls, see the Remarks section.
PERF_REDUNDANT_SETTRANSFORM
Number of non-redundant calls to IDirect3DDevice8::SetTransform. For more information about redundant calls, see the Remarks section.
PERF_REDUNDANT_SETTEXTURE
Number of non-redundant calls to IDirect3DDevice8::SetTexture. For more information about redundant calls, see the Remarks section.
PERF_REDUNDANT_SETPALETTE
Number of non-redundant calls to IDirect3D8::SetPalette. For more information about redundant calls, see the Remarks section.
PERF_REDUNDANT_SETSTREAMSOURCE
Number of non-redundant calls to IDirect3DDevice8::SetStreamSource. For more information about redundant calls, see the Remarks section.
PERF_SETSTATE_POINTPARAMS
Number of times this state is modified on the hardware.
PERF_SETSTATE_COMBINERS
Number of times this state is modified on the hardware.
PERF_SETSTATE_TEXTURESTATE
Number of times this state is modified on the hardware.
PERF_SETSTATE_SPECFOGCOMBINER
Number of times this state is modified on the hardware.
PERF_SETSTATE_TEXTRANSFORM
Number of times this state is modified on the hardware.
PERF_SETSTATE_LIGHTS
Number of times this state is modified on the hardware.
PERF_SETSTATE_TRANSFORM
Number of times this state is modified on the hardware.
D3DPERF_FORCE_LONG
Forces this enumeration to compile to 32 bits in size. This value is not used.

Remarks

A function call is considered redundant if it sets the state to the current value. If the title's code is not compiled using D3DCOMPILE_NOTINLINE, Direct3D will use inline versions of functions that do not check for redundant calls and do no performance counting.
