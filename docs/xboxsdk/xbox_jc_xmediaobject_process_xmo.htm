XMediaObject::Process

Submits buffer to the stream.

HRESULT Process(
    LPCXMEDIAPACKET  pInputBuffer, 
    LPCXMEDIAPACKET  pOutputBuffer
);

Parameters

pInputBuffer
Address of the input XMEDIAPACKET structure.
pOutputBuffer
Address of the output XMEDIAPACKET structure. Data processed by the Xbox Media Object will be placed in this packet.

Return Values

Returns DS_OK if successful, or an error specific to the Xbox Media Object otherwise.

Remarks

Process creates an output packet for a given input packet. In the case of asynchronous Xbox Media Objects, Process will return immediately; synchronous Xbox Media Objects will return when the Xbox Media Object has finished its processing and written data to the output packet.

Upon return, the following items are modified:

Asynchronous Xbox Media Objects:

  • pOutputBuffer->pdwCompletedSize is set reflecting the amount of data processed by the Xbox Media Object.
  • pOutputBuffer->pdwStatus is set to XMEDIAPACKET_STATUS_PENDING.
  • pInputBuffer->pdwCompletedSize is set reflecting the amount of data processed by the Xbox Media Object.
  • pInputBuffer->pdwStatus is set to XMEDIAPACKET_STATUS_PENDING.

Synchronous Xbox Media Objects:

  • pOutputBuffer->pdwCompletedSize is set reflecting the amount of data processed by the Xbox Media Object.
  • pInputBuffer->pdwCompletedSize is set reflecting the amount of data processed by the Xbox Media Object.
  • pOutputBuffer->pvBuffer contains newly processed data.
