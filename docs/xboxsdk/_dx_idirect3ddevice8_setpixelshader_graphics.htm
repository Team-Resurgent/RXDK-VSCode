IDirect3DDevice8::SetPixelShader

Sets the current pixel shader to a previously created pixel shader.

HRESULT SetPixelShader(
  DWORD Handle
);

Parameters

Handle
[in] Handle to the pixel shader, specifying the pixel shader to set. The value for this parameter can be the handle returned by IDirect3DDevice8::CreatePixelShader.

If the handle is NULL, the legacy pixel pipeline is used.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

Calling this function will reset any shader constants set by SetPixelShaderConstant to their values as defined in the pixel shader program.
