Xbox Live Overviews

This section contains overviews related to Xbox Live. It is organized into the following topics:

Introducing Xbox Live
Describes the Xbox Live service, including Q & A and overviews of Live from a developer's view and from a subscriber's view.
Initializing the Xbox Live Library
Discusses what your title must do before it can use any of the services provided by Xbox Live.
Xbox Live Drop-in User Interface (UIX)
Introduces the Xbox Live Drop-in User Interface (UIX) and describes how to use it and customize it.
Connecting to Xbox Live with Authentication
Presents an overview of authenticated connections to Xbox Live.
Downloading Content from Xbox Live
Describes how to enable your titles to download content and add it to the game.
Updating Your Game Using Autoupdate
Discusses how to update players' copies of your title after it has shipped. Updates can fix security problems, bugs, and so forth.
Playing Games with Matchmaking
Describes the matchmaking service, which enables game titles to exchange information about currently running, network-enabled game sessions.
Notifications and State Information
Describes how to use the notifications service to notify titles of real-time events.
Presence Information
Describes how to retrieve presence information for a given set of players. Essentially, titles use the presence service to help players detect the presence of their friends, teammates, and competition opponents on Xbox Live.
Competitions
Discusses how game developers can offer structured competitions, called tournaments, in their games. The competitions service is responsible for scheduling competitions, sending reminders to players, and progressing players.
Arbitration
Describes the arbitration service, which supervises registered game sessions in an automated way. Arbitration enables accurate score reporting for all ranked games, including tournament matches.
Teams
Describes the teams services available on Xbox Live and explains how to use them.
Messaging
Discusses how game developers use messaging to help build player community within their title. Publishers can create messages such as a Message of the Day to be delivered directly to the players of their titles.
Voice Chats
Provides an introduction to the voice chat capabilities that games can use on Xbox Live.
Friends Lists
Describes how to manage friends requests, removals, and game invitations.
Families of Titles
Introduces publishers to a way to perform certain tasks across multiple titles. Publishers can group their games into one or more families of titles by assigning them the same family title ID.
Statistics
Discusses the stats service and how to use it to store and retrieve statistical information about players of your game title.
Game Clips
Describes how publishers and players can share data (game clips) that relates to statistically measured performance within a game.
Storage
Every title on Xbox Live has a data store associated with it. This overview discusses how your title can use this storage to distribute data to subscribers and enable players to exchange data.
Debugging and Testing Games for Xbox Live
Discusses several means games developers can use to debug and test their titles.
Best Practices and Lessons Learned
Presents best practices, tips, and suggestions that you can use to help improve your title's Xbox Live implementation.

For more information about specific application programming interface (API) methods, see the Xbox Live References.
