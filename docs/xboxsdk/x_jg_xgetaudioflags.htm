XGetAudioFlags

The XGetAudioFlags function returns the global audio settings set in the Dashboard.

DWORD XGetAudioFlags();

Parameters

None.

Return Values

The return value is a 32-bit value. The lower 16 bits contain exactly one of the following values:

VALUEMEANING
XC_AUDIO_FLAGS_STEREOStereo
XC_AUDIO_FLAGS_MONOMono
XC_AUDIO_FLAGS_SURROUNDDolby Surround Sound

The upper 16 bits of the return value contain a combination of zero or more of the following values.

VALUEMEANING
XC_AUDIO_FLAGS_ENABLE_AC3Dolby Digital Enabled
XC_AUDIO_FLAGS_ENABLE_DTSDTS Enabled

Use the following macros from Xbox.h to test for these values:

  • XC_AUDIO_FLAGS_BASIC
  • XC_AUDIO_FLAGS_ENCODED

Examples:

if ( XC_AUDIO_FLAGS_BASIC( dwSpeakerConfig ) == XC_AUDIO_FLAGS_STEREO )
                                        
if ( XC_AUDIO_FLAGS_ENCODED( dwSpeakerConfig ) & XC_AUDIO_FLAGS_ENABLE_AC3 )

Notes:

The XC_AUDIO_FLAGS_ENABLE_AC3 flag occurs only in combination with the XC_AUDIO_FLAGS_SURROUND flag.

The XC_AUDIO_FLAGS_ENABLE_DTS flag may occur in combination with any of the other flags.

Remarks

The XGetAudioFlags function retrieves the current Dashboard audio settings, whereas the GetSpeakerConfig function retrieves the current active settings. Typically, these are the same. However, they will differ if the application has called DirectSoundOverrideSpeakerConfig.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
