IDirect3DDevice8::BeginVisibilityTest

Begins a visibility test.

HRESULT BeginVisibilityTest(
);

Parameters

None.

Return Values

If this method succeeds, it returns D3D_OK.

Remarks

When this method is invoked, the internal pixel counter is cleared in preparation for this visibility test.
