Compiler Error C2993

'identifier' : illegal type for non-type template parameter 'parameter'

You cannot declare a template with a structure or union argument. Use pointers to pass structures and unions as template parameters.

The following sample generates C2993:

// C2993.cpp
// compile with: /LD
struct MyStruct 
{
   int a;char b;
};

template <class T, struct MyStruct S>   // C2993
// try
// template <class T, struct MyStruct * S>
class CMyClass 
{
};

This error will also be generated when an attempt is made to use floating-point non-type template parameters.

If it is a function template, use a function argument to pass in the floating-point non-type template parameter. The following example is valid in both the Visual Studio .NET 2003 and Visual Studio .NET versions of Visual C++. If it is a class template, there is no easy workaround.

// C2993b.cpp
// compile with: /LD
template<class T, float f>   // C2993
void func(T)
{
}

template<class T>   // OK
void func2(T, float)
{
}

}
