XNetDnsRelease

Releases the resources used for a DNS lookup.

INT XNetDnsRelease(
    XNDNS *pxndns
);

Parameters

pxndns
Address of the XNDNS structure returned by XNetDnsLookup.

Return Values

Returns zero if successful, an error code otherwise.

Remarks

Pending DNS lookups can be cancelled by calling XNetDnsRelease before the lookup is completed.

The following example shows how to handle a DNS lookup when the title provides an event handle.


WSAEVENT hEvent = WSACreateEvent();
XNDNS * pxndns = NULL;
INT err = XNetDnsLookup("sample.microsoft.com", hEvent, &pxndns);
WaitForSingleObject(hEvent, INFINITE);
if (pxndns->iStatus == 0)
{
	// Look at pxndns->cina for the number of addresses returned
	// Look at pxndns->aina for the vector of addresses returned
}
else
{
	// An error occurred.  One of the following:
	// 	pxndns->iStatus == WSAHOST_NOT_FOUND - No such host
	//    pxndns->iStatus == WSAETIMEDOUT - No response from DNS server(s)
}
XNetDnsRelease(pxndns);
// Don't dereference pxndns after this point

If the title does not provide an event, the title must poll iStatus of the XNDNS structure as follows.


XNDNS * pxndns = NULL;
INT err = XNetDnsLookup("sample.redmond.corp.microsoft.com", NULL, &pxndns);
while (pxndns->iStatus == WSAEINPROGRESS)
{
	// Do something else while lookup is in progress
}
if (pxndns->iStatus == 0)
{
	// Look at pxndns->cina for the number of addresses returned
	// Look at pxndns->aina for the vector of addresses returned
}
else
{
	// An error occurred.  One of the following:
	// 	pxndns->iStatus == WSAHOST_NOT_FOUND - No such host
	//    pxndns->iStatus == WSAETIMEDOUT - No response from DNS server(s)
}
XNetDnsRelease(pxndns);
// Don't dereference pxndns after this point

Header: Declared in Winsockx.h. for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. For online titles, use the secure version of the library file (XOnlines.lib) for final testing (not available for system link play).
