IDirect3DDevice8::GetDebugMarker

In debug builds, gets the last debug marker set on the device. In retail builds, this function compiles away to nothing.

DWORD GetDebugMarker();

Parameters

None.

Return Values

Returns the last debug marker set on the device by IDirect3DDevice8::SetDebugMarker.
